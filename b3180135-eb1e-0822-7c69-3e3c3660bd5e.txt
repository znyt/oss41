        因为，夏阳枭决定赌一把。
<br/>
<br/>        他不信连华生和傅夜沉之间会没联系，只是他没有找到他们之间的联系方法而已。
<br/>
<br/>        墓园里，他已经派人去蹲守了。
<br/>
<br/>        蹲了一天，临近黄昏，他们终于等来了一个西装革履戴着墨镜的男人。
<br/>
<br/>        男人手里拿着一束金合欢花，刚弯腰将花束放在墓碑前，身后就有十几个人从四面八方冒出来，纷纷掏出手枪对准他，当中甚至有人在呵斥：“傅夜沉，束手就擒吧！”
<br/>
<br/>        “我想，你们认错人了。”傅昼景摘掉了脸上的墨镜，转过身去，面对着这群特警。
<br/>
<br/>        大家面面相觑后，才缓慢收起手枪。
<br/>
<br/>        为了不认错，为首的队长对傅昼景进行了搜身，从他的皮夹里翻出了他的身份证，重新核对身份后，才放他离开。
<br/>
<br/>        在自己办公室里坐立不安的夏阳枭，得知消息，去墓园的人不是傅夜沉，而是傅夜沉的弟弟傅昼景时，心里更加慌了起来。
<br/>
<br/>        他独自一个人在办公室的落地窗前徘徊，一包烟吸了一根又一根，直到烟灰缸里满是烟头，盒子里没有烟可吸后，他才关了灯，离开了办公室。
<br/>
<br/>        夏阳枭刚上车，就接听到实验室里连华生的父亲连中凯打来的电话。
<br/>
<br/>        “夏局，实验成功了，她醒了。”听筒里传来连中凯高兴不已的声音。
<br/>
<br/>        夏阳枭悬着的一颗心，也稍稍放了下来：“她一切正常吗？”
<br/>
<br/>        “嗯，身体各项都健康也达标了。只不过……”连中凯欲言又止。
<br/>
<br/>        夏阳枭下意识地追问：“只不过什么？”
<br/>
<br/>        “只不过，我们得让她进入沉睡状态，才能进行下一步的解药提炼。毕竟，她是母体，解药无法另行培养，只能从她血液里采集。她若是清醒的，我们这样做的话，就相当于在慢性杀人……”
<br/>
<br/>        “二十二年前，你们把她造出来的意义是什么？那么现在，就是她的使命。”夏阳枭直接打断了连中凯的话。
<br/>
<br/>        “但是，让她进入沉睡状态，得使用安眠药剂，长期用这种药剂，对母体有损，也是有风险的。”
<br/>
<br/>        “那就把她变成植物人！”
<br/>
<br/>        “她可是你女儿！”
<br/>
<br/>        “在我眼里，她和裴金澈一样，都是异类。”夏阳枭不假思索地回答。
<br/>
<br/>        “好。”连中凯不再反驳，而是听从指令似的应道。
<br/>
<br/>        夏阳枭挂了电话后，心里的大石头总算是落下了。
<br/>
<br/>        一个月后的这天傍晚，连华生在自己制药厂里的实验室里和自己的团队加班商讨药剂的配合比时，接到了夏玄灵打来的电话。
<br/>
<br/>        “连华生，你赶紧带着你的药箱过来，阿澈晕过去了。”夏玄灵急得连说话的声音都带着粗喘。
<br/>
<br/>        连华生忍俊不禁道：“你倒是有本事了，能让金澈在你身上做晕过去。”
<br/>
<br/>        “你别废话了行吗？他是真的晕过去了！”夏玄灵急得声音哽咽。
<br/>
<br/>        连华生淡定反问：“什么症状？”
<br/>
<br/>        “他、他脸色惨白，呼吸频率不对。还有……还有我看不出来啊！我又不是医生！”夏玄灵忍不住破口大骂，“你TMD的不是医生吗？赶紧给本大小姐过来救人！”
<br/>
<br/>        “好，我马上过来，地址。”连华生向自己的团队打了个手势，便一边接听电话，一边离开了实验室。
<br/>
<br/>        夏玄灵报了地址。
<br/>
<br/>        连华生在自己的制药厂里拿了几盒治疗贫血的药剂和口服的胶囊，随后赶到了裴金澈的公寓里。
<br/>
<br/>        夏玄灵给他开的门，急忙叙述裴金澈当时的情况：“他在客厅的沙发上。我给他做完晚饭后，才发现他晕倒了。”
<br/>
<br/>        连华生疾步走到客厅里，将医药箱放到茶几上后，给裴金澈做了一番检查。
<br/>
<br/>        就在他准备给他的手臂做肌肉注射的时候，发现他臂弯上有针孔，心里顿时纳闷地皱了下眉头。
<br/>
<br/>        一针下去后，过了一会儿，裴金澈终于苏醒了过来。
<br/>
<br/>        连华生又让夏玄灵去倒杯温水，让裴金澈吃口服药。
<br/>
<br/>        趁着夏玄灵去倒水的空隙，连华生瞥了裴金澈一眼，问道：“你不是有快速自愈的能力吗？为什么臂弯上会留下针孔的血痂？”
<br/>
<br/>        裴金澈收了手，默不吭声。
<br/>
<br/>        “还有，你这面色，像是长期贫血造成的。你去献血了吗？”连华生接着问道。
<br/>
<br/>        裴金澈回过头去，下意识地看了一眼夏玄灵那边。
<br/>
<br/>        连华生立即明白了是什么意思，于是扯大了嗓音，吆喝道：“玄灵，你可以下去买包红糖上来吗？你家这位的药，需要配着红糖一起吃。”
<br/>
<br/>        “好！我马上去！”夏玄灵将温开水端过来后，又急急忙忙地出了门。
