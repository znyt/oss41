　　“大仁，你小师弟最近在忙什么，还是在看书？”田不易问到。
<br/>　　他已经有很长一段时间没见过苏重了，如果不是饭菜越发可口美味，证明做饭的还是自家小徒弟，他都以为苏重下山消失了。
<br/>　　“师弟最近在练功。”宋大仁经常看望苏重，了解的比较详细。
<br/>　　“练功？”田不易眉头微微皱起：“他还在尝试修炼玄清道？”
<br/>　　还是忍不住修仙诱惑在尝试修炼吗？
<br/>　　“不是，师弟在修炼他自创的武功，好像叫聚元功。”宋大仁挠挠头有些不确定道。
<br/>　　他对世俗武功是不怎么看的上的。如果这不是自家小师弟修炼的功法，他连名字都不会记住。
<br/>　　“胡闹，武功是那么好创造的吗！”田不易呵斥了一句。不过也没怎么在意。
<br/>　　世俗武功而已，即便练出了叉子，也不过是走火入魔损伤经脉。他一颗黄芽丹下去，什么伤都好了。
<br/>　　田不易他也不太看得上世俗武功。不过，一个十多岁的小孩子，看了一段时间的书，竟然敢扬言自创武功？
<br/>　　“走，跟我去看看。”田不易突然收起脸上怒容笑呵呵道。
<br/>　　两人出了门，御剑腾空而起，直奔后山竹林而去。
<br/>　　苏重搬到后山竹林的事情，田不易是知道的。他还暗中去看过，就怕自己这个小徒弟想不开。
<br/>　　后山不远，两人修为高深，几乎眨眼就到了地方。
<br/>　　看着竹屋周围井然有序的种满各种植物，田不易欣慰的笑了笑。
<br/>　　他上次来偷看，就发现苏重在种花种树，他还是挺赞成的，有种爱好打发时间，就不会胡思乱想，能转移注意力。
<br/>　　“走，咱们去看看你这位小师弟。”田不易对自己这个徒弟还是挺好奇的。
<br/>　　小小年纪，自己在山上造了个竹屋隐居，和个小老头似的，他不觉得怪异只觉得好玩。就和小孩子过家家一样。
<br/>　　刚走进竹屋周围，田不易脸色就一变。
<br/>　　“大仁，你有没有觉得，这周围田地灵气极其稠密。”他修炼多年，对这个最为熟悉。
<br/>　　“师傅，确实如此。”宋大仁仔细感知片刻也不由满脸惊讶。
<br/>　　“小师弟是故意选的这里？”宋大仁惊讶。
<br/>　　“问问他就知道了。”田不易觉得自己这个小徒弟不简单。
<br/>　　能找到这个地方，起码感知上就非常敏锐。
<br/>　　他们说话声音不小，苏重却始终房门紧闭，透过打开的竹窗，两人看到盘坐在蒲团上修炼的苏重。
<br/>　　“看着还到是那么一会儿事。”田不易摇头笑道，却没出声打扰。虽然看不上世俗武功，但修炼的时候贸然打扰，说不定就会让自家小徒弟受伤。等一会儿，他也不怎么在意。
<br/>　　刚想玩他脸色不由一变，苏重身上骤然散发出一股独特气息，接着周围小院突然狂风大作。
<br/>　　在田不易的感知中，大片天地灵气骤然汇聚，竟正争先恐后的钻进苏重体内。
<br/>　　“小师弟的病好啦？”宋大仁惊喜到，他也感觉到异常。
<br/>　　“等会问问他。”田不易同样惊讶。世俗武功能有这种效果？
<br/>　　没等多久苏重就醒过来。他早就感知到田不易到来。但刚才正处于关键时刻，他也不好停下来修炼。现在修炼完毕，立刻起身走出小竹屋。
<br/>　　“徒儿，刚才是怎么回事？”田不易已经冷静下来。他和掌门道玄都不知道如何改换资质。苏重一个十多岁小童就更不可能。
<br/>　　但刚才苏重确实吸收了灵气，他疑惑不解。
<br/>　　“师傅，我用武功突破先天，以此沟通天地吸收灵气入体。”苏重也不隐瞒。
<br/>　　“也就是说，你能修炼啦？”宋大仁惊喜。
<br/>　　苏重遗憾摇头：“出了些叉子，能吸收天地灵气，但不能修炼玄清道。”
<br/>　　苏重当初设想的很好，突破先天后引起入体，然后借此修炼玄清道。
<br/>　　可惜玄清道自成一体，他用内力牵引来的灵气，即便按照玄清道运转，也没有丝毫作用。他怀疑可能是经过聚元功运转，灵气已经失去最初面貌，无法用来修炼太极玄清道。
<br/>　　“那你打算怎么办？”田不易上下打量自家徒弟。
<br/>　　即便再看不上世俗武功，他也知道用武功突破先天的难度。
<br/>　　青云山立派千年，不乏世俗武功高手收入门中。他们就是因为修炼到先天，能够利用天地灵气，才被门派看重。
<br/>　　没有修仙法门，却用内力沟通天地，这些都是天资高绝之辈。
<br/>　　青云收徒弟，往往会考虑这些武功天才。只要心性品德良好，收入山门很快就能崭露头角。
<br/>　　苏重自创武功，一月突破先天。田不易认为苏重就是那种修炼武功的天才。
<br/>　　可惜苏重还是不能修炼玄清道。但之前把苏重送下山的打算就要往后拖一拖了。他想看看苏重是不是还能创造奇迹。
<br/>　　“只能继续修炼聚元功。怎么说也踏上了修仙路。”苏重无奈。
<br/>　　“聚元功到底有什么效果，给我说说。”田不易想看看自家小徒弟的本事。
<br/>　　“要说效果，力气大算不算？”苏重挠头。
<br/>　　聚元功就是用来突破先天，所以聚元内力温润中正，没有一点儿攻击力。但正因为温润，所以最擅长滋养身体。他修炼一个半月，全身细胞都被聚元内力浸泡滋养。
<br/>　　本来他应该只用一个月就能突破先天，但大量聚元内力被用来滋养身体，苏重不得不继续修炼。
<br/>　　“力气大，力气有多大。”田不易不太在意。心里有些好笑，苏重再天才也不过是个十多岁孩童。能创造出一个半月突破先天的武功，就已经非常了不起，还能指望有什么变化。
<br/>　　苏重转头打量四周，走到一个儿臂粗黑竹面前。两手抓住黑竹猛然发力，咔嚓一声，黑竹应声而断。
<br/>　　田不易顿时脸色大变。
<br/>　　黑竹到底有多坚韧他非常清楚。徒弟们锻炼身体，砍伐的可都是拇指粗细黑竹。
<br/>　　这个儿臂粗的黑竹，除了宋大仁，其他几个徒弟可别想容易砍断。怎么也得折腾一天。苏重竟然徒手折断？
<br/>　　“徒弟，能把聚元功给我看看嘛。”田不易觉得脸皮发烫。师傅索要徒弟功法，这可不太地道。不过为了其他徒弟，他也顾不得了。
<br/>　　正因为苏重表现出的强劲体魄，让他意识到，如果徒弟们都能练成聚元功，实力定然能实现飞跃。想施展青云神剑御雷真诀，可是需要强劲体魄的！
<br/>　　“当然没问题，还望师傅指正。”苏重丝毫没有敝帚自珍的意思。田不易修炼数百年见多个时光，苏重还希望他能帮自己改造一下功法呢。
<br/>　　他以后只能在聚元功的路子上走，能有人帮忙再好不过。
