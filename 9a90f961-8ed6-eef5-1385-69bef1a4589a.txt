　　第662章
<br/>　　程大雷一日一日坐大，已经是宋伯康的眼中钉肉中刺，自己必须解决掉他，否则他迟早会掉过头来解决自己。
<br/>　　这件事宋伯康明白，程大雷也同样明白。
<br/>　　双方之间迟早要两军对垒，只是那一刻还未发生而已。
<br/>　　借着这次朔方城扩张，宋伯康已经派了十几名斥候混进城中，只等一个月黑风高的好天气，便用一场大火焚毁城外农田。
<br/>　　先用断粮计，之后便要着手对付程大雷。
<br/>　　城外良田千顷，虽然有关鱼的鱼字军守卫，但想要一丝空子都没有，也几乎是不可能的事情。
<br/>　　这一夜，便是一个月黑风高的好天气。程大雷窝在屋子里，听着外面风声呼呼刮过。他面前既没有苏樱，也无有樊梨花。只有一个徐神机不时品一口小酒，吃一口花生米。
<br/>　　“大当家，你这样躲着也不是个办法呐？”徐神机忽然冒出一句。
<br/>　　程大雷猛地激灵一下：“什么叫躲着，本当家需要躲谁。你没看到现在朔方城多忙，我只是抽不开身。”
<br/>　　“我没说大当家在躲什么事，大当家何必如此大惊小怪，有些此地无银三百两，也有些不打自招。”
<br/>　　徐神机猥琐的眼神，竟露出一抹狡黠。这是令程大雷吃惊的，他另外吃惊的一件事是，徐神机竟然知道『不打自招』四个字。
<br/>　　而程大雷又不得不承认，徐神机说对了。他如被扎破的气球一样松懈下来，也饮了一口酒，无可奈何的叹口气。
<br/>　　“多情自被无情恼，有时候我也蛮羡慕军师你的，长成你们这般模样，也就不会有这种苦恼了。”
<br/>　　徐神机摸摸下巴，觉得程大雷这话说得不对，如果程大雷算是魁梧，那么自己就应该是儒雅，毕竟也没差多少么。
<br/>　　“话说回来，军师为何不再成个亲？”
<br/>　　如今徐神机处于半归隐状态，平日程大雷也不需要他出谋划策，当然他从来也没有出谋划策过。如此，倒是确定了他在山寨第一军师的地位，毕竟是高手么，高手都是不轻易出手的。现在就连和珅都觉得，自己和徐军师有很大的差距。
<br/>　　而以他现在的地位，成一门亲事还是不费什么事的，平素徐神机就好花街柳巷乱转，可他似乎也一直没有动过这方面心思。
<br/>　　“唉，人已经老了，有些念头也不该有了，屋里添个人，也是麻烦不是。”
<br/>　　程大雷不得不对其竖起大拇指：“军师是个明白人。”
<br/>　　“一般啦，也就一般啦。”
<br/>　　正此时，外面敲锣打鼓的动静，有人高声喊着：“走水啦，走水啦。”
<br/>　　徐神机激灵一下，放心酒杯道：“大当家，哪处走水了，我们要不要去看看。”
<br/>　　其实程大雷有些懒得去看，毕竟他若事必躬亲，那要和珅也就无用了。不过既然身在朔方城，左右也无事，程大雷便索性出去看看。
<br/>　　二人披上袍子，刚推开屋门，便有夜风呼呼吹进怀里。
<br/>　　“嚯，好大风。”
<br/>　　程大雷下意识道了一声，同时也有些担忧，风借火势，火助风威，这如果着起大火，怕是一时半会不太好扑灭。
<br/>　　他与徐神机快步离开城主府，循着火光找过去，见人群围在一起，和珅见程大雷进来，连忙拱手：“大当家。”
<br/>　　程大雷点点头：“什么状况？”
<br/>　　“倒是不太严重，放火的人被抓住了。”
<br/>　　“咦，还有纵火犯？”
<br/>　　“这人还有几个同伙，张将军的人正在追捕，刚才已经审问清楚，是凉州派过来的。”
<br/>　　和珅简单说了一下目前的情况。这纵火犯来自凉州城，本来主意打得特别好，一面在城内纵火，吸引守军的注意力。另外埋伏在城外的人动手，放火焚毁麦田。
<br/>　　但他们低估了和珅的能力，在和珅的整治下，整座城宛若铁桶相仿，城中的巡逻队络绎不绝。
<br/>　　这人刚放起火便被巡逻队抓个现行，三拳两脚下去，便把所有都吐了。
<br/>　　程大雷听罢皱起眉头：“倒是不能大意，有问题没有？”
<br/>　　和珅的脸色很难看，他本来还想趁着程大雷在朔方城，做些事给程大雷看看，谁料到竟然有人来放火烧粮。假若大火烧起来，自己的脸面往那里搁。
<br/>　　“大当家放心，一切包在属下身上，绝对不让敌人的奸计得逞。”
<br/>　　“不如，我与你一起去城外看看？”
<br/>　　“大当家且去休息，这次事情如果出了岔子，和某提头见大当家。”
<br/>　　和珅说得如此斩钉截铁，程大雷也不太好勉强。毕竟，自己也要学会放权下来，给和珅一定的信任。
<br/>　　“好，我等你的消息，明天晚上请你吃酒。”
<br/>　　“是。”
<br/>　　程大雷和徐神机又回到城主府，但其实仍旧有些放心不下，毕竟这关乎一城的未来。
<br/>　　而今天的事，同样告诉程大雷另一道讯息：宋伯康决定要对自己动手了。
<br/>　　另外，这也说明，相府已经解决长安的麻烦，却也不知，李乐天现在是怎样的状况。
<br/>　　长夜漫漫，无心睡眠，程大雷与徐神机抱着火炉取暖，烤一些花生苹果什么的，熬着漫长的时间。
<br/>　　在程大雷走后，和珅一张脸黑得瘆人，便是张肥看到也有些怕。
<br/>　　“张将军，我刚才对大当家说得话你已听见，如果今日的事情办不好，我这颗脑袋可就要搬家了。”
<br/>　　“好说好说。”
<br/>　　“如此，也就莫怪和某人发号施令。”和珅轻咳一声：“点三千人出城，今日若放走一个贼人，便用他人头与和某陪葬。”
<br/>　　张肥一激灵，他平时不是很看得起和珅这种读书人，但读书人狠起心来，实在令人不寒而栗。
<br/>　　三千肥字军出城，扫荡农田周围。关鱼已得到消息，保持全体警戒状态。
<br/>　　凉州城这次大概派出一百余人，只等城内火起便展开行动。可城中的治安委实太好了些，刚放起一点小火就被抓个现形，他们等的大火一直没烧起来。
<br/>　　之后，便被张肥的肥字军扫荡到，然后慌忙做鸟兽散，而张肥得到的命令是抓活的。
<br/>　　当然，这是和珅的命令。
<br/>(本章完)
