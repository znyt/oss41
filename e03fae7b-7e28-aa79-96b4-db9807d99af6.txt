
<br/>　　<br/>　　    屋子里的人神色都很严峻，一时间话头都不知道从哪起，几乎都在努力的消化着，自己今天所见的一幕。<br/>　　<br/>　　    李菲儿坐在角落里，两手捧着一杯热水，眼神呆滞的看着地面，她觉得自己都不敢闭上眼睛了，眼皮一合脑袋里就会出现李良胸膛被那把画戟穿透的画面，今天死了的那个考古队员是她的同学，在上船之前对方还曾经跟她说过几句话，没想到下潜到湖底，人就再也没有回来了，还是被抚仙湖尸库中的尸体给杀的。<br/>　　<br/>　　    “啪，啪”王惊蛰低着脑袋，还是站在人群外面靠着门框子，手里随意的把玩着个打火机，一不小心就打着了火后，他抬起头发现屋里人的视线都落在了自己身上。<br/>　　<br/>　　    “不好意思……”王惊蛰尴尬的说道。<br/>　　<br/>　　    文峰忽然站起来，瞪着通红的眼睛来到他身前，一把抓着他的领子就吼道：“你不是会看么，你不是知道要出事么，啊？这回你怎么不行了，你怎么不告诉李良一声，他要死了啊”<br/>　　<br/>　　    王惊蛰皱了皱眉，停顿片刻后说道：“我不是万能的，不是上帝也不是耶稣更不是阎王爷，我告诉他什么啊？告诉他，那些士兵活了，会杀人？”<br/>　　<br/>　　    “你，你哪怕是提醒他一声也行啊，你，你不是先游出去的么，你肯定知道会有事，你为什么不告诉他，啊？”文峰崩溃的说道。<br/>　　<br/>　　    王惊蛰伸手抵着他的胸膛，推开文峰歪着脑袋望着黄詹说道：“这就是我要你们在水下听我领导的原因，但可惜你他们做的并不够好，是吧，黄教授？”<br/>　　<br/>　　    黄詹叹了口气，跟身边的人交代把文峰拉回来，他又抬头点了点脑袋，说道：“是我们疏忽了，也是我想的少了，也实在没想到水下那些两千多年前的尸体会突然活了”<br/>　　<br/>　　    王惊蛰摇头说道：“不是活了，死了两千多年，不可能在起死回生，从哪个角度来说都说不通”<br/>　　<br/>　　    黄詹诧异的问道：“那是怎么回事，视频里明明能看清，那些尸体都睁开了眼睛，也……”<br/>　　<br/>　　    王惊蛰摆了下手，打断他说道：“不是只有活人才能动的，死人也同样可以，比如尸变了，或者是这些尸体被操控了，都有可能”<br/>　　<br/>　　    “啪，啪”黄九郎拍了拍桌子，站起来说道：“这个问题先不用说了，说了也没答案，先研究下这抚仙湖古城，接下来到底是探还是不探了，我刚才已经把这件事上报到了xx办公室，那边正在研究，结论还没有下，但无非就是两个可能，不勘察抚仙湖了，所有人员马上返回离开，如果继续勘察的话该怎么办？”<br/>　　<br/>　　    张生利张了张嘴，摊着手说道：“黄主任，你觉得他们谁还会敢在下去么？”<br/>　　<br/>　　    别说敢不敢下潜了，你现在就是让这帮人在湖边走走，碰着水可能都得麻爪了，这就是挥之不去的心理阴影，甚至搞不好都得整出心里疾病来。<br/>　　<br/>　　    但是别人敢不敢黄九郎不知道，王惊蛰肯定没问题，所以他把这个锅就甩了过来。<br/>　　<br/>　　    “惊蛰，如果上面要求继续探滇国古城，恐怕还得你下去了……”<br/>　　<br/>　　    “啪”王惊蛰按着打火机，面无表情的说道：“你是吃准了我会再下，行，这没问题，但就我一个人你觉得我下去了能有啥用？”<br/>　　<br/>　　    黄詹一听，顿时一滞。<br/>　　<br/>　　    王惊蛰说道：“你们这个什么特别事务处理办公室，就你一个光杆司令么？调几个人过来帮忙总归可以吧”<br/>　　<br/>　　    黄詹摇头说道：“我最多能调两三个人来，但来的可能也都不是管用的，他们都各司其职在外面，自己有自己的一摊”<br/>　　<br/>　　    “那我可搞不定了”王惊蛰说道。<br/>　　<br/>　　    黄九郎指了指他身后的陈三岁，说道：“你应该有不少的朋友么，能不能请来几个帮帮忙？待遇什么的好说，有什么要求尽管提，我能力范围的，都能给你解决了”<br/>　　<br/>　　    王惊蛰眯了眯眼睛，说道：“人么倒是真有，不过有一个可能来的话，会有点麻烦”<br/>　　<br/>　　    “什么麻烦？”黄九郎诧异的问道。<br/>　　<br/>　　    “他在里面蹲着呢”王惊蛰叹了口气，说道：“我这个朋友是倒斗的大行家，最顶尖的摸金校尉，上下几千年的古墓，他想走一趟都没问题，这抚仙湖水下的古城，说白了也不过就是一座水下坟墓，特别是中间那座金字塔，我估计更有可能是当时滇国过往的坟冢，要想彻底一探古城的话，我真挺需要一个摸金校尉的，有他在能给我帮不少的忙，只不过可惜了，前段时间他犯了点事，人在岭南监狱里蹲着呢，判了六年半的徒刑，这才刚进去没几个月”<br/>　　<br/>　　    黄九郎“哦”了一声，问道：“犯的什么事？”<br/>　　<br/>　　    “走私古董吧，不过赃物已经追回来，上缴了”<br/>　　<br/>　　    黄九郎寻思了下，顿时摆手说道：“问题不大，我想办法批一下把人借过来，之后真要是能把古城探明白了他这也算是戴罪立功，我可以向岭南方面提出为他减刑，当然了前提是，他人被借过来之后，不能逃了”<br/>　　<br/>　　    王惊蛰笑道：“开什么玩笑呢，六年多的徒刑他坚持下就过去了，犯不上为这几年给自己背个越狱的罪名，不划算，还有……除了他以外，我还能找个人过来，你要探古城，我肯定得把人手准备全了，准备的还是自己能信得到的，这要是来不托底的人，你让我下去我也怕啊，后背交给的肯定都是能生死与共的战友”<br/>　　<br/>　　    “可以，你安排人，我安排条件！”<br/>　　<br/>　　    当天晚上，黄九郎这边就得到了上面的批示，结论就一个，那就是无论如何都得把抚仙湖湖底的古城探明白了，从上面考虑的角度是，水下尸库的尸体有那么多，现在还诈尸了，这要是不捅咕明白了的话，万一这些尸体真集体全活了然后跑上来的话，那恐怕就是一场空前的灾难了。<br/>　　<br/>　　    黄九郎将必探抚仙湖的计划告诉王惊蛰后，他就开始着手准备了，以最快的速度搭配出一队抚仙湖水下探古城的顶配天团出来。
