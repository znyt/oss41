　　身穿黑袍穿梭于黑洞中，配合着他如闪电一般的速度，根本就无迹可寻。
<br/>　　哪怕是每一层的守塔长老，都丝毫没有察觉到，在他们的严密防守下，有人能抵达最下层。
<br/>　　说来也是，谁敢选择这种不要命的方式前往最底层，虽然这些守塔长老都有这种实力，却不敢这样尝试。
<br/>　　因为，陨落心炎，可不是他们能够对付的东西，何况这一枚异火，已经诞生出了灵智。
<br/>　　...
<br/>　　...
<br/>　　片刻后，慕清霄身形一顿，停留在了距离塔底二十多米的上空，完全融入黑暗，气息也没有一丝的泄露。
<br/>　　这里的空间不像上面，四周空荡荡一片，没有任何的修炼室，空间中充斥着恐怖的高温。
<br/>　　若是实力没有到达斗灵级别，根本不可能长时间待在此处。
<br/>　　他能清晰的察觉到，就在左侧不远处，有一道极强的气息，隐藏在那里，斗皇巅峰，看来是守塔长老没错。
<br/>　　而在他的脚下，是一口直径二十米左右的坑洞，坑洞中赤红色岩浆翻滚，唯有那岩浆气泡爆裂时发出的细微声音回荡。
<br/>　　坑洞中的岩浆颇为粘稠，难以泛起浪花，因此给人一种压抑而死寂的感觉。
<br/>　　而在岩浆的上空两米处，明显有一条条微不可查的空间皱褶，正是塔底的最后一层封印。
<br/>　　斗皇巅峰，美杜莎绝对能离开练气塔，只要封印一破，外面的长老也无暇顾及她吧？
<br/>　　想到此处，慕清霄也就没有任何顾虑，暗道：“准备好了吗？”
<br/>　　话语刚落，套着一袭黑袍的美杜莎就出现在了他的身后，微微点头，道：“动手吧。”
<br/>　　伴随着清脆声音落下，黑色自体内席卷而出，黑色羽翼伸展而出，两丈多高的黑色神凤完全融入黑暗之中。
<br/>　　出现在上空的同时，恐怖的威压，使得整个练气塔都为之一颤！
<br/>　　就在守塔长老失神的瞬间，慕清霄俯冲而下，利爪直接劈在了薄弱的封印上......
<br/>　　“不好！”
<br/>　　守塔长老惊叫一声，想要阻拦，却有一道强大的黑袍神秘人拦在了他的身前。
<br/>　　在恐怖的一爪下，塔底的封印上，顿时出现密密麻麻的裂痕，宛如蜘蛛网一般，朝着四周蔓延而去。
<br/>　　“住手！”
<br/>　　守塔长老爆喝一声，想要阻拦，疏忽大意间，却被黑袍人一掌印在胸膛上，身躯倒飞而出，眼睁睁的看着封印破碎。
<br/>　　伴随着一阵清脆的响声，封印彻底爆碎开来。
<br/>　　趁着守塔长老被截住，慕清霄没有丝毫犹豫，化作黑影，直接冲进下方的岩浆中，溅起一阵岩浆浪花，如鱼得水，很快就消失了踪影。
<br/>　　见慕清霄消失，美杜莎周身空间一阵扭曲，便从原地消失。
<br/>　　望着破碎的封印，守塔长老眼中闪过一丝惊惧，身形爆射而出，向着塔顶冲去，几十秒后，震耳欲聋的声音在整个内院上空回荡。
<br/>　　“封印被破！”
<br/>　　在声音响起的一瞬间，内院东边，三道恐怖的气息骤然升起。
<br/>　　周围空间一阵扭曲，眨眼间，两名老者已经出现在了天焚炼气塔的上空，另外一道恐怖的气息，则向着美杜莎离开的方向追去。
<br/>　　苏长老望着嘴角含有一丝血迹的守塔长老，神色凝重，道：“究竟怎么回事？”
<br/>　　“有斗宗强者，通过中央黑洞潜进塔底，趁机破开了封印......”
<br/>　　“什么？封印被破开了？”
<br/>　　守塔长老话语还未落下，便被庆长老惊叫一声给打断。
<br/>　　回过神来，庆长老气的胡子都歪了，厉喝道：“那还愣着干什么？快让塔内的学员全速撤离，我们先进去看看情况，做好最坏的打算！”
<br/>　　说罢，他与苏长老的身影就消失不见。
<br/>　　数十秒后，两位长老就出现在了塔底上空，望着坑洞中翻滚的岩浆，皆是露出无与伦比的凝重之色。
<br/>　　“短短几秒，封印真的破了，出手的最起码也是七星斗宗，甚至更高，学院里什么时候混进了这种强者？”
<br/>　　苏长老凝重的点了点头，道：“出手的人，对内院很熟悉，而且不知用什么方法，成功避开了我们的视线。”
<br/>　　此时，坑洞中除了岩浆气泡爆裂发出的细微声音，并没有任何的异动，使得两位长老心中满是困惑。
<br/>　　“怎么没有动静？”等待了几分钟，庆长老神色古怪的问道。
<br/>　　苏长老盯着脚下的岩浆，道：“你问我，我问谁去？”
<br/>　　就在这时，空间一阵扭曲，另一名老者出现在了上空。
<br/>　　“一星斗宗，对方空间奥妙的领悟比我高深，没能拦的住，但他硬生生接了我三掌，估计不会好受。”
<br/>　　“一星斗宗绝不可能破开封印，肯定有同伙，有没有见到其他的可疑人物？”苏长老转过头来，神色凝重的问道。
<br/>　　老者一愣，看着脚下消失的封印，摇了摇头，道：“倒是没有见到其他可疑人物......”
<br/>　　苏长老皱了皱眉，道：“这里由我们两人来镇守，你安排人手，去将今晚在练气塔中修炼的学员都留下来，一个个排查，后者可能就藏在学员当中。”
<br/>　　“记住，查到可疑人物，切记不要动手，先稳住对方在说，暂时不知道对方的数量，也不知道他们的目的，其中隐藏着七星斗宗，或者更强，小心行事！”
<br/>　　“好，交给我吧！”
<br/>　　老者满口答应，其实他也不想待在这里，毕竟下面可是封印着一枚可怕的异火，现在封印被破，谁也不知道会发生什么事情。
<br/>　　“下面可能有变故，陨落心炎没有爆发，对于学院来说，是不幸中的万幸，我们暂时镇守这里，学院的事情，交给其他长老和执事去做。”
<br/>　　微微点头，庆长老浑浊的双眼中闪过一抹思索。
<br/>　　“前几天黑角域的韩枫被神秘强者击杀，连凶手是谁都不知道，现在学院中也出现这种莫名其妙的事情，多事之秋啊！”
<br/>　　“他们冒险破开封印，却没有动手，看来有人隐藏在暗中，想要针对学院啊，院长不再，这里只能由我们来抵御外敌了。”
<br/>　　此时，塔底！
<br/>　　赤红的岩浆世界，充斥着死寂，看上去似乎没有一丝生命迹象，望不见尽头，宛如一滩死水，不起一丝波澜。
