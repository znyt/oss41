        “……”此刻，傅昼景沉默了。
<br/>
<br/>        苏漫雪红唇微扬，白了傅昼景一眼，胸有成竹地接着说道：“你现在可是季氏集团的总裁，若是不想自己身败名裂，就最好与童筱颖保持点距离。免得，被那些狗仔乱写，说你窥觊自己的大嫂，那就糟糕了。”
<br/>
<br/>        这女人把话说到这个份上，傅昼景忽然想起了刚刚的那件事，立即抓住苏漫雪的手，去抢她的手提包。
<br/>
<br/>        “把手机拿出来！”傅昼景冷斥。
<br/>
<br/>        刚刚苏漫雪是拿着手机对着他和筱颍的，那么她一定拍到了他抱着筱颍的画面。
<br/>
<br/>        苏漫雪也不藏着掩着，十分坦白地说道：“我已经上传到我的邮箱里面了。冷夫人今日带童筱颖这么一亮相，现在娱乐圈里的人都知道，她是傅家的大少奶奶。你说，我拿着这张相片，是不是可以当护身符？嫂子和小叔子搞到一块，这种花边乱伦的新闻，一定会让季氏集团和傅氏集团股市下跌吧？”
<br/>
<br/>        “苏漫雪，你顶着一张天使的脸，干得却是肮脏的事情。亏你还是个大明星，若是让你的粉丝知道，你是如此龌龊的女人，到时候，身败名裂的人，会是谁？”傅昼景不甘示弱地反问，“别忘了，你和筱颍长得一模一样。我若是一口咬定这个女人是你，你会怎么做？”
<br/>
<br/>        “那你试试看啊！照片里，童筱颖身上那件晚礼服可是上百万。我可穿不起。”苏漫雪得意地笑了笑。
<br/>
<br/>        傅昼景顿时无话可说。
<br/>
<br/>        苏漫雪懒得跟傅昼景废话，而是自顾自地接着说道：“接下来的这段日子，你最好不要惹恼我。不然，我这暴脾气，说不定就把照片给卖了出去。”
<br/>
<br/>        “你……”
<br/>
<br/>        未等傅昼景把话说完，苏漫雪傲慢地仰起头来，潇洒地转身离去。
<br/>
<br/>        要说自己不能怀孕的原因是因为当了别人的小三，这件事情，倒是令童筱颖上了心。
<br/>
<br/>        可是，这事，是那两个陌生人说的，应该也不能全信吧！
<br/>
<br/>        童筱颖觉得，自己有必要等傅夜沉回来，再问问傅夜沉。
<br/>
<br/>        等到她返回会场的时候，明星演唱的环节已经结束了，台上是杂技表演，台下的宾客可以开始正式享用晚餐。
<br/>
<br/>        这次晚宴吃的是西餐，主食被服务员端上桌的时候，童筱颖就试了下味道，便没再吃第二口了。
<br/>
<br/>        宴席末了，服务员上来收拾了餐桌，又给在座的每位端上了不同口味的果茶。
<br/>
<br/>        先前的商演结束后，现在开始的是拍卖会，进行拍卖的东西，全是各位明星的私人珍品。而拍卖所得的钱，是否全部或部分用来做慈善，完全由明星们自己做决定。
<br/>
<br/>        而压轴的那条“蓝色生死恋”的13.14克拉的蓝色钻石项链，是国际影后的外国前夫送给她的，她此次拿出这条蓝色钻石项链出来拍卖，一是为了做慈善，二是为了告别过去。
<br/>
<br/>        童筱颖陪着冷夫人看了一眼拍卖清单，里面大多都是明星们的限量款衣服和香包，也有拍卖古玩珍品的。
<br/>
<br/>        冷夫人似乎对这些东西都不感兴趣，虽说她未出过国门，但她的衣服和手提包都是国际顶尖级的设计师为她量身定制的，所以，她又岂会对明星们用过的二手服饰香包感兴趣。
<br/>
<br/>        至于古玩珍品，傅家大庄园里的那些东西，已经够古董了，再买回去，显得有些小巫见大巫。
<br/>
<br/>        唯独让冷夫人看上眼的是那条“蓝色生死恋”项链。
<br/>
<br/>        前面那些拍卖的东西一轮又一轮地过场走完后，拍卖师终于请出了众人翘首以盼的“蓝色生死恋”。
<br/>
<br/>        那条蓝色钻石项链的做工确实很精致，钻石在鉴定师的仪器鉴定下，得出来是无杂质的纯色蓝钻，已有百年的历史。
<br/>
<br/>        鉴定师当众鉴定完后，拍卖师报出了起步价：“壹亿壹仟万！”
<br/>
<br/>        顿时，在场一片唏嘘。
<br/>
<br/>        冷夫人写上自己的竞争价后，落落大方地举起了牌子：“壹亿壹仟壹佰万。”
<br/>
<br/>        “壹亿壹仟贰佰万。”
<br/>
<br/>        “壹亿壹仟叁佰万。”
<br/>
<br/>        “壹亿壹仟肆佰万。”
<br/>
<br/>        在场的豪门，争先恐后地竞价，但都是一百万一百万地往上加。
<br/>
<br/>        冷夫人最后一次亮牌是“壹亿伍仟万”。
<br/>
<br/>        结果，还有人继续高于冷夫人的亮牌。
<br/>
<br/>        竞价最后被炒到了贰亿伍仟万。
<br/>
<br/>        就在大家惊赞的时候，拍卖师忽然铿锵有力地惊呼：“现在有位先生直接亮出了叁亿的竞价！”
<br/>
<br/>        会场上，众人议论纷纷起来。
<br/>
<br/>        冷夫人一脸不悦地丢了自己手中的牌子，像是做好了放弃的准备。
