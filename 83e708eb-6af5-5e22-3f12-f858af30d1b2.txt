　　不远处，擎苍宗阵营之中，神色已经恢复正常的绝无敌，看到大屏幕上的成绩，也不由得直摇头，
<br/>　　“亏我之前，居然还想把他当做我那心魔的替代品，没想到竟是一个草包货！”
<br/>　　绝无敌瞥了一眼叶尘的成绩之后，便收回目光，继续闭上双眼，微微调息起来。
<br/>　　龙辰、风华舞、乐无忌，甚至连洛冰玄，也都在心里暗暗摇头，
<br/>　　“看来是我想太多了！”
<br/>　　所有经历过千劫神塔的人都知道，闯关的速度越慢，被神雷劈中的次数就会越多！
<br/>　　以叶尘这种慢如蜗牛的速度，即便是龙辰和绝无敌，也绝对难以取得什么好成绩。
<br/>　　所以，没有人会认为，叶尘是在扮猪吃虎，如果他真的想要这种方式，来吸引众人的注意力，那简直就太蠢了！
<br/>　　叶尘并不知道，外面这些人对他看法，即便是知道了，他也不会在乎。
<br/>　　此时的他，正美滋滋的享受着法则神雷的细润，别提有多开心了。
<br/>　　别人都是生怕被法则神雷劈中，唯独叶尘反而嫌雷劈的太少！
<br/>　　终于，
<br/>　　用了大约一个多小时，叶尘来到第二十一层石阶，已经被法则神雷足足劈了180次！
<br/>　　咔嚓！
<br/>　　咔嚓！
<br/>　　咔嚓！
<br/>　　又是九道神雷过后，
<br/>　　嗡~~
<br/>　　围绕在叶尘周围的造化法则，忽然开始发生了变化，渐渐汇聚在一处，形成了一道无比华美，且透着神秘气息的符文！
<br/>　　“哈哈！造化法则符文！成了！！”
<br/>　　叶尘大喜过望，
<br/>　　原本，从领悟出法则，到将法则凝聚成符文，即便是渡劫帝君级的强者，也需要耗费数百年乃至上千年的时间！
<br/>　　而如今，叶尘在法则神雷的帮助之下，竟然只用了短短一个多小时！这让他怎能不喜？
<br/>　　“继续！”
<br/>　　叶尘欣喜过后，将造化法则符文，收回体内，然后将第二条毁灭法则缓缓祭出，继续朝上方迈进。
<br/>　　……
<br/>　　一转眼，
<br/>　　又过去了一个多小时，
<br/>　　截止到此时，
<br/>　　第二十一组的十六个人，十四人已经被淘汰，唯独只剩下叶尘和凌嚣，依旧还在千劫神塔之中。
<br/>　　只不过，凌嚣此时已经突破了500层，马上就要朝600层冲击，而叶尘却才只堪堪突破50层！
<br/>　　足足十倍的差距！
<br/>　　众人已经再次纷纷嘲讽起来，
<br/>　　“这小子，还真是特么的能熬啊！居然还没有被淘汰！”
<br/>　　“看来这小子，也不是那么废物嘛！”
<br/>　　“不过是死撑罢了！我敢打赌，最多60层，他肯定会被轰出来！”
<br/>　　……
<br/>　　然而，
<br/>　　又过去了将近一个小时，
<br/>　　叶尘已经稳步迈入了80层，依旧还在缓慢上升。
<br/>　　而那凌嚣，虽然成功达到了600层，可此时的速度，已经完全慢了下来。
<br/>　　终于，
<br/>　　在进入612层长达十几分钟之后，
<br/>　　轰！
<br/>　　一道人影从千劫神塔中飞出，
<br/>　　“咳咳咳！”
<br/>　　凌嚣落地之后，第一时间望向大屏幕，发现叶尘的名字还在亮着，顿时两眼一瞪，
<br/>　　“怎么可能？那小子居然还没有……噗！”
<br/>　　凌嚣惊讶过后，等看到叶尘的成绩，顿时捂着肚子，哈哈大笑了起来，
<br/>　　“居然才88层，废物啊废物！你是来搞笑了吗？真是浪费本公子的感情！哈哈哈！”
<br/>　　然而很快，
<br/>　　又一连过去了一个多小时，
<br/>　　叶尘已经稳步迈入100层，却似乎还没有任何要出来的征兆。
<br/>　　三个小时之后，
<br/>　　叶尘迈入130层，继续保持着匀速，稳步上升。
<br/>　　六个小时后，
<br/>　　叶尘来到了160层，依旧还是没有要出来的意思……
<br/>　　下方的众人，都快要疯了！
<br/>　　尤其是第二十二组的考核者，更是忍不住开始破口大骂起来，
<br/>　　“这小子，简直就是占着茅坑不拉屎啊！”
<br/>　　“这准备让我们等到什么时候？”
<br/>　　“磨磨叽叽的，还有完没完了！”
<br/>　　“长老，您就不管管吗？那小子若是一直赖在里面不出来，难道我们就只能在这里等着不成吗？”
<br/>　　有人实在忍受不了了，开始向元始门的长老抗议起来。
<br/>　　那老者，显然也没有遇到过这种情况，脸色同样有些难看。
<br/>　　他负责主持千劫神塔的考核，已经上百年了，一般正常情况下，最多也就八九个小时，肯定也就出来了。
<br/>　　像叶尘这么墨迹的，却还是头一次遇到，之前简直闻所未闻！
<br/>　　“诸位稍等，我去和门主大人商议一下！”
<br/>　　很快，
<br/>　　那老者和元始门门主以及一众高层，经过一番商议之后，最终决定：
<br/>　　叶尘进入的那个入口，先暂时封闭，剩下的考核者，全部化为十五人一组，继续进行考核。
<br/>　　不一会，
<br/>　　剩下的考核者，重新分组完毕，并且开始有条不紊的进行起来。
<br/>　　接下来的几组之中，出现了一些成绩相对比较亮眼的考核者，
<br/>　　譬如，
<br/>　　无极宗的易白，一举拿下了652层的惊人成绩，引得众人一阵惊呼。
<br/>　　太玄宗的徐良山，也取得了647层的好成绩。
<br/>　　落霞宗的余苍梧，也突破了600层。
<br/>　　随着考核的继续进行，几乎没有人再去关注叶尘的成绩。
<br/>　　毕竟，在众人看来，叶尘完全就是在死撑，根本不会取得什么好的成绩，早晚自己受不了了，自然就会主动退出。
<br/>　　一直到第二十七组，曦月拿下了660层的惊人成绩，这才又有人想起了叶尘，
<br/>　　“快看！那姓叶的小子，居然突破到400层啦！”
<br/>　　有人忽然惊呼了一声，众人纷纷朝大屏幕的最下角望去，
<br/>　　412层！
<br/>　　众人顿时一片哗然，
<br/>　　叶尘居然在不知不觉中，已经突破到了400层！
<br/>　　这已经远远超过了，所有人对他的预期！
<br/>　　这一刻，
<br/>　　已经没有人再出言嘲讽，尽管很多人，依旧不相信，叶尘的最终成绩，可以超过十大天骄，可是就目前的情况来看，他的这份成绩，已经相当不错！
<br/>　　至少，已经超越了很多普通天才！
<br/>　　“莫非，闯关的速度越慢，反而越有利？”
<br/>　　有人开始怀疑，是叶尘掌握了某种更有利的闯关技巧。
<br/>　　然而，
<br/>　　接下来的几组，有人尝试着用这种方式闯关，成绩却一塌糊涂……
<br/>　　一转眼，
<br/>　　随着最后一组的登场，从叶尘进入千劫神塔到此时，已经足足过去五天时间！
<br/>　　而叶尘，也已经冲破了600层，而且依旧在稳步上升！
<br/>　　到了这个时候，所有人的脸色，终于开始发生细微的变化……
