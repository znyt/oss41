　　声音没有掩饰，木叶和木天同时动作一顿，抬头望向天空。
<br/>　　那里已经乌云密布，厚重的黑色云层笼罩了整片大地，并且还在持续聚集，白允儿的周身无端刮起狂风，吹起衣袍猎猎作响。
<br/>　　部分重伤者开始东倒西歪，无法在这狂风中站稳，待在木屋近处的顾凉晨和尚九娣，却感受不到丝毫风起。
<br/>　　不等他们反应过来这是怎么回事，丧尸王木天就再次开口：“真没想到你已经强大到了这种地步，不过很可惜，再强也只能拖住一人。”
<br/>　　随着木天的话音落下，在场的狼牙成员全体沉默起来，气氛异常的压抑。
<br/>　　他们太弱了！
<br/>　　无论哪个都挡不住！
<br/>　　而白允儿也只能挡住一个，所以无论如何，木屋都会被人破入！
<br/>　　木天玩味的看向白允儿，指了指木屋后方：“你是打算拦我，还是拦后面那个异种？”
<br/>　　白允儿沉默，低头一言不发。
<br/>　　木天面上的笑容绽放，双脚轻微移动调整到了进攻姿态：“那我就不客气了。”
<br/>　　同时，木叶也再次迈开脚步，嘴角挂着淡笑朝着木屋走去，他的步伐不快不慢，带着悠然自得的自信。
<br/>　　白允儿面对着丧尸王，那么楚涵就交给异种王，反之亦然，所以无论白允儿怎么选择，都改变不了注定的结局！
<br/>　　顾凉晨的面上浮现出一抹绝望，他很理智，明白再多的人也拦不住那名异种。
<br/>　　能与丧尸王一同出现，并且又如此行驶诡异的，必然就是那个将很多人都耍的团团转，神秘莫测的异种王木叶！
<br/>　　尚九娣缓慢的将手中枪支放下，依靠在木屋旁泣不成声，她知道她拦不住，甚至一点用都没有……
<br/>　　绝望浮现在众人的心头，只能被动等待的时候，丧尸王木天则开始发动了攻击！
<br/>　　他的右脚朝前横跨，左脚一个弹力起跳，顿时整个身躯便半腾空的朝着白允儿射去，于此同时他的右手成拳，猛然从身侧用力挥出。
<br/>　　轰！
<br/>　　一道恐怖的挥动声炸响，这一拳竟然直接将空气都砸出断层！
<br/>　　仅仅是一刹那，拳头便带着极致的威能来到白允儿身前！
<br/>　　木天做好了一拳轰在那道雷电屏障上的准备，那种防御他遭受过一次，清楚它的威力，所以这一拳的威力已到达十成，是木天近身攻击的最强一击。
<br/>　　此时的白允儿才抬起头，双眼的瞳孔已经完全蜕变为绚丽的蓝色，也将她整个人映照的尤其冰冷。
<br/>　　她面无表情之中，却毫无预兆的一条腿往斜前方一跨，而后——
<br/>　　刷！
<br/>　　猛然一步来到木天身侧，同时左右手前后挥出，缠绕着木天轰出的那一拳，噼里啪啦的一阵劈砍。
<br/>　　这是借力打力，白允儿直接利用了反复多次的速度，将这一拳的力量卸掉一大半！
<br/>　　木天愣了一瞬间，没想到白允儿根本没有设立任何防御屏障，而是与他展开了近身较量！
<br/>　　这时候的木屋后方，则毫无预兆的发出一阵电闪雷鸣声，只见一道闪电忽然从天空竖直劈下，狠狠的对准了与木屋只有一米之隔的木叶！
<br/>　　轰！
<br/>　　一声巨响，在木叶身旁爆开！
<br/>　　在那闪电劈来的瞬间，木叶便立即躲过，并且一下子退后到老远，他的表情瞬变，不再是悠闲自信，而是满面的警惕和不解。
<br/>　　上方的天空依旧黑云密布，但却集中在木屋上方，并且在那木屋的周围，一道道细小的雷电不断闪烁，将整个木屋笼罩，形成了一个随时可能触发恐怖雷电的保护罩。
<br/>　　尚九娣和顾凉晨双双愣住，没料到在千钧一发之际，竟然会有这种转折。
<br/>　　木叶被完全挡在了木屋之外，一旦靠近便是来自天空的电闪雷鸣，这些闪电与白允儿空手瞬发的屏障完全不同，威力更不是一个等级。
<br/>　　哪怕是异种王，也经不起这种冒险！
<br/>　　与白允儿展开了近身缠斗的木天也在此刻反应过来，他震惊的难以组织当下的语言，只能看着白允儿冒出一个字：“你？！”
<br/>　　白允儿却依然面无表情，戴着银色手套的双手不断朝前进攻，每一次都抓向木天的脖颈处，招招犀利异常。
<br/>　　只是在这等快速的进攻之下，她左肩处的伤口也因此裂开，好不容易愈合的伤势开始复发，并且更严重的流血不止。
<br/>　　所有人都明白了过来，白允儿的确发了飙，但却用她最厉害的能力用来守护楚涵！
<br/>　　丧尸王和异种王，她要硬抗！
<br/>　　所有在场的狼牙成员全部激发了凶性，他们不再被动的站在一边不知所措，小队模式再次形成，分工有序的朝着木天小心靠近。
<br/>　　他们很清楚自己没有能力击杀这两名不善者中的任何一人，甚至连挡都挡不住，但他们在狼牙训练这么久学会的，可不仅仅是杀与被杀。
<br/>　　“分散！骚扰！”
<br/>　　一声令下，现场所剩不多的防守队成员便忽然散开，形成了古怪又毫无章法的样子，他们绕着木天不断游走，时不时的一道攻击逼近其耳后，同时又快速退开。
<br/>　　哪怕是丧尸王也是丧尸，耳部是致命部位，木天下意识的便会躲开，无法将其当做不存在。
<br/>　　但这些人在木天眼里也弱小如蝼蚁，当他被骚扰的烦不胜烦，想要回头解决的时候，却又发现这些蝼蚁早已跑开老远。
<br/>　　就像是成群结队的一列蚊虫，在耳边嗡嗡嗡的吵闹，时不时的跑来叮一口，让人烦躁异常。
<br/>　　分心让丧尸王步步后退，在白允儿更加猛烈的进攻下无法招架，那些防守队成员就在四周游荡，看准时机随时准备上前。
<br/>　　狼牙众人看到了机会，白允儿也在这种配合下多次伤到木天，让他的脖颈处多了好几道血淋淋的抓痕！
<br/>　　但就在这样后不久，忽的一个身影出现在白允儿的余光之中！
<br/>　　她瞳孔一缩，已经伸出的右手快速收回，同时身体一下子朝旁边闪躲！
<br/>　　刷！
<br/>　　噗！
<br/>　　只听得两道声音接连响起，前者是白允儿速度爆发的移动声，后者则是白允儿小臂处一道极深伤口的喷血声！
