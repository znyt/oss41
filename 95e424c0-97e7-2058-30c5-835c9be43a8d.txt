
    王惊蛰当天离开曹王庄以后，就去了县城，按照老头说的是有一户人家失联了，还有一户搬到了县城里他们也有联系，于是他就给对方打了个电话然后说明来意，没想到这次却扑空了，对方告诉他自己已经搬过几次家了，以前家里的老东西全都给扔了，他要的那张老羊皮他也有印象但真不知道弄哪去了。<br/>　　<br/>　　    王惊蛰挺失望的，四张老羊皮看来他也就能弄到手里两张了，而且还是不关联的，以他现在的水平来讲，还是挺难把余下的阵图给推演出来的，所以这时候王惊蛰就觉得自己有必要走一趟唐山了。<br/>　　<br/>　　    在县城里休息了一天，隔天王惊蛰启程就往唐山走了，两地离的并不远，他是当天下午就到的，然后直奔陈家大宅而去。<br/>　　<br/>　　    刚进陈家大宅，捋着小路观望，王惊蛰对他向叔的感官就觉得又有点摸不透了，这大宅里的风水有一个词来形容就是简直绝了。<br/>　　<br/>　　    上一次在骊山秦村跟向缺见面，他觉得对方就是个挺随和的父亲然后才是个很内敛的中年，不显山不显水地但往那一坐就能给人一种八方风雨来他都巍然不动的感觉。<br/>　　<br/>　　    王惊蛰到了陈家大宅后就直接去书房见他了，向缺此时正坐在书桌后面看着一本中文版的荷马史诗，桌上还放着一副龙飞凤舞的字画，人的身上充满了浓浓的书香气，虽然向缺小本都没有毕业，可这种意境一拿出来的话，还是挺有味道的。<br/>　　<br/>　　    书看不看得懂不重要，字写得好不好看也无所谓。<br/>　　<br/>　　    但是气质必须到位！<br/>　　<br/>　　    “向叔，你这生活可以啊……”王惊蛰笑着说道。<br/>　　<br/>　　    “唉，来了，小蛰？”向缺见他进来，就拿起毛巾擦了擦手把书放在了一旁，说道：“我现在的日子就是，四十岁之前跨马扬刀一往无前，四十岁之后凡事都退一步海阔天空，生死已经全看淡了，什么都无所谓了，主要是享受生活”<br/>　　<br/>　　    “我得学学您了，我发现你举手投足之间全是知识啊”<br/>　　<br/>　　    向缺顿时哈哈一笑，矜持地说道：“学吧，学到手里全是活”<br/>　　<br/>　　    四十来岁的向缺，如今站在大街上，除去认识他的人剩下的肯定都得把他当成是一个平淡无奇的油腻中年男子，就是普罗大众中最普普通通的一员，但谁人也无法想到，往前十五年的话，此人是叱咤风水阴阳界最牛逼闪闪的一号人物，他现在用一个词来形容就是，返璞归真活出真正的风范了。两人刚聊了几句，书房外面有人敲门，随后一个显得极其雍容华贵，看着就让人想到母仪天下这个词的中年女子走了进来，她端着茶盘进来后就冲着王惊蛰笑了笑，向缺抬头说道：“叫陈婶吧，这是惊蛰，我上次去骊山见到了王仙芝，这是他儿子”<br/>　　<br/>　　    女王范十足的陈婶放下茶盘主动给两人倒了杯茶，王惊蛰连忙伸出双手接了过来，对这个女人他可得妥妥地保持着礼敬的心态，因为据他所知这是向缺身后站着的唯一的一个女人，这要是放在百年前的话，她肯定是后宫里最当之无愧的头面人物。<br/>　　<br/>　　    “你们喝茶，先聊着”陈婶笑着跟王惊蛰说道：“我让厨房做上饭菜，再收拾一间客房，晚上你就住这里吧”<br/>　　<br/>　　    “唉，好，麻烦陈婶了”<br/>　　<br/>　　    陈夏笑道：“家里多少年了，都难得能来客人，晚上陪着你向叔喝一点聊聊天，你们坐着吧我先出去了”<br/>　　<br/>　　    陈夏走了后，王惊蛰调侃着跟向缺说道：“叔，人生赢家啊”<br/>　　<br/>　　    向缺傲然说道：“我年轻那时候还可以的……”<br/>　　<br/>　　    他俩闲聊了几句，向缺就问他是不是有什么事来，王惊蛰就从陈放撞邪的事上讲起，最后一直说到了南汇大团陵园下虬髯客的那个墓。<br/>　　<br/>　　    听完王惊蛰说的，难得淡定的向缺靠着椅子也皱起了眉头：“风尘三侠中关于李靖和红拂女的事迹还是挺多的，毕竟两人随着李世民征战到最后入朝为官都有史书可以查询，但这个虬髯客世人却知之甚少了”<br/>　　<br/>　　    王惊蛰点头说道：“是啊，向叔，现在的人最多只知道他是风尘三侠之一，手上很有点本事，但具体什么细节却无人得知了，他显得有点太神秘了”<br/>　　<br/>　　    “八字借命……”向缺喃喃地嘀咕了一声，良久后才说道：“古井观道藏三千，其中也没有关于八字借命的介绍，我原以为这一术已经彻底失传没有可能再面世了，没想到你还捋出一条线来，逆天改命的前身就是八字借命，要是真能把这个理清楚了，这可是天大的机缘了”<br/>　　<br/>　　    王惊蛰当即正色说道：“所以我才来见您一面，这方面的理论除了您以外，天下间我也想不出谁能给我分析出点干货了”<br/>　　<br/>　　    向缺瞥了他一眼说道：“你把我捧的有点高了吧？”<br/>　　<br/>　　    王惊蛰顿时呲牙笑了：“叔，换个人我也懒得去捧啊，您这道行可是在这摆着呢”<br/>　　<br/>　　    “哈哈，你这嘴上的活可以啊……”向缺摆了摆手，沉思着说道：“关于八字借命我不熟悉也不能妄下断言，你先留在这里两天，我稍微琢磨一下”<br/>　　<br/>　　    “那倒是不急，还是稳妥点好！”<br/>　　<br/>　　    这天下午，王惊蛰就留在了陈家大宅，两人在书房里商讨了小半天头绪也就理出了一点，晚间的时候陈夏叫他们过来吃饭，菜虽不算太丰盛，但荤素都有，却也很精致。<br/>　　<br/>　　    吃饭的时候，向征就坐在王惊蛰的对面，这次再见他要比上一次的状况好了不少，可能是因为天道气运的原因，向征看着不太痴傻了，只是显得特沉闷也不说话。<br/>　　<br/>　　    王惊蛰就问向缺道：“叔，既然气运已经到了，您怎么不带他出去碰碰运气？就像我这一次……”<br/>　　<br/>　　    向缺端着杯子，看了儿子一眼，摇头说道：“我也不太着急，现在向征的问题也不是那么非得要迫在眉睫地解决了，我反倒是觉得他这么无忧无虑地活着也未尝不是一件好事，人啊无论多聪明多有钱，但背后肯定烦恼一堆，向征这样反倒是活得轻松了，我和你陈婶要是老去了他还有个姐姐，他姐姐走了的话，陈向两家后代也还可以，我有几个朋友过得也挺不错，所以啊护着他一辈子不难，他这样活着可比一般人乐呵多了”<br/>　　<br/>　　    东北非著名网络作者困得睡不着先生就曾经说过一句话：“这世上只有一种人能活的最真实最没有烦恼了，那就是傻子……因为傻子，什么都不用想！”<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
