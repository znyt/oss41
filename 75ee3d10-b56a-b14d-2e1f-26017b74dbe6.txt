　　紫炎虽然能焚毁万物，但如今只是借助他的火焰温度而已。
<br/>　　在萧逸的控制下，只会融化掉药材，提取出药力，而并不会将药力也焚毁掉。
<br/>　　四人有条不紊地炼着药。
<br/>　　在场的武者，除了少数几人外，几乎都在大呼过瘾。
<br/>　　四品炼药师炼药的情景，可是很少能看到的。
<br/>　　四人的速度几乎差不多。
<br/>　　半个时辰后，药材的药力完全被提炼，而后便是按顺序地融合起来，这个过程极考验炼药师的耐心和控制力。
<br/>　　一个时辰后，四人的额头上明显布满了汗水。
<br/>　　事实上，只要参悟了丹方，炼药过程中一般不会出现意外。
<br/>　　整个炼药过程，非常枯燥，又非常磨人。
<br/>　　这时才是真正考验炼药师本事的时候。
<br/>　　白家主脸色紧张地盯着，一颗着急的心揪了起来。
<br/>　　在场的其他武者，也看得目不转睛。若能看到四品巅峰丹药雪魄丹的出炉，将是一件值得炫耀的事。
<br/>　　毕竟，这种丹药就算是五品炼药师都不一定能百分百成功，可以说难得一见之事。
<br/>　　在这紧张的气氛中，没有人注意到，北山慕容家那边，随同慕容胤一同前来的慕容杀执事，忽然离开了。
<br/>　　不多时，慕容杀与两个年轻人一起回来。
<br/>　　只见那两个年轻人极为恭谨地站在他身旁。
<br/>　　而他则一副高人模样，训话道，“你们两个小辈，只有后天境修为，本来是没资格来这里的。”
<br/>　　“现在四位炼药师原地炼药，我便破例带你们过来见识一番。”
<br/>　　“谢慕容杀执事。”两个年轻人感激地说道。
<br/>　　他们的声音并不大，怕打扰到四人的炼药。
<br/>　　但，先天境以上武者的耳朵是很灵敏的。除非不想听，若是特地地想去听，很容易就能听得到他们的对话。
<br/>　　当然，在场武者没人愿意干这闲事。
<br/>　　除了萧逸。
<br/>　　萧逸一直是个谨慎的人，无论有没有必要，他不会放过任何一个细节。
<br/>　　虽然慕容杀他们的对话，能在大庭广众下谈论，肯定只是无关重要之事。
<br/>　　然而，这时他听着两个年轻人的声音，却微微皱起了眉头。
<br/>　　两人的声音，有些耳熟。
<br/>　　萧逸一边精准地控制着炼药过程，一边认真听着。
<br/>　　在真气的增幅下，耳朵变得极为灵敏。
<br/>　　两个年轻人恭谨地问道，“慕容杀执事，不知何时能随我们回紫云城报仇？”
<br/>　　“放心吧。”慕容杀摆摆手，道，“慕容墨好歹是外门执事，有功于家族，不会让他白死的。”
<br/>　　“我乃洞玄六重武者，那萧家，挥挥手便灭掉了。”
<br/>　　两个年轻人顿时一喜，道，“那就谢过慕容杀执事了。”
<br/>　　随后，两人脸上露出了阴狠的目光，“哼，我要他们萧家通通死绝。”
<br/>　　“别急。”慕容杀随意地说道，“这次家族派我来百武城，还有些事要处理。待处理好了，过两天便随你们回去。”
<br/>　　“区区一个小家族罢了，到时候我一个个绑起来，随便你们对付。男的杀掉，女的留下，哈哈。”
<br/>　　“现在，先看慕容胤执事的炼药风姿吧。我们北山慕容家，随便一个执事，都是外人眼中高不可攀的强者，去到哪都受尽崇拜。”
<br/>　　慕容杀得意地笑着。
<br/>　　当然，在场没有人听他的笑声，也没有人注意他。
<br/>　　所有人的目光都在紧张看着中心四人的炼药。
<br/>　　忽然，慕容胤的炼药炉猛地一声轰鸣，一股激烈的爆炸，弄得火焰四射。
<br/>　　慕容杀的笑声戛然而止。
<br/>　　“炸炉了...炸炉了....”
<br/>　　在场武者惊呼一声。
<br/>　　“咳咳。”慕容胤咳嗽了两声，被炸得灰头土脸，却没有受伤。
<br/>　　但药炉里面的药力，可是毁于一旦了，也代表着这次炼药的失败。
<br/>　　数秒过后，又是一声爆响。
<br/>　　方鹤的炼药炉也炸炉了。
<br/>　　“方鹤大师也失败了。”在场武者失望地摇了摇头。
<br/>　　“唉。”方鹤叹了口气。
<br/>　　一分钟之后，叶铭的炼药炉也是一阵爆炸，炸炉。
<br/>　　“连叶铭都没能成功。”在场武者变得更加失望。
<br/>　　白家主更是咬了咬牙，慕容胤和方鹤的失败他尚且能接受。
<br/>　　但他一直对叶铭抱有极高的信心，连叶铭都失败了，难怪他会满脸灰丧之色。
<br/>　　至于易霄，他其实本来就没抱多大信心，也似乎早就预料到易霄也会失败。
<br/>　　果然，几乎是他念头刚出现之时，萧逸的火炉上虽然没有出现爆炸，但一阵澎湃的紫炎从火炉中倾泻而出，险些伤了其他武者。
<br/>　　这种情况，火焰失控，恐怕药力也被焚毁了。
<br/>　　“易霄也失败了，不过也在常理之中。”在场武者反而觉得理所当然。
<br/>　　毕竟他们也是一开始就不对易霄有信心，连其他三人都失败了，易霄就更不可能成功了。
<br/>　　然而，事实是，萧逸根本没有炸炉，也没有失败。
<br/>　　他可是有着冰鸾剑赋予的完美炼药术，完美过程，怎么可能失败。
<br/>　　只不过是刚才听了慕容杀等人的对话，一时心神不稳，才出现了紫炎失控的情况罢了。
<br/>　　紫炎本身就威势强大，稍微失控一些，动静就很大，看起来像是炸炉。
<br/>　　但实则，萧逸马上就反应了过来，并控制了火焰，里头的药力丝毫无损。
<br/>　　“慕容轩，慕容千军，你们竟然去北山慕容家了。”萧逸心头狠狠地想着，“早知如此，当初醒来时，就该不惜一切追杀你们。”
<br/>　　“想灭我萧家？我不会让你们有机会的。”萧逸心头涌现出一股滔天杀意，但此时正在炼药，他勉强压了下来。
<br/>　　这时，慕容胤、方鹤两人的脸色变得难看起来，叶铭也没了之前的神态自若，皱起了眉头。
<br/>　　方鹤没有理会慕容胤，而是看向叶铭，道，“叶铭，你也是同样的情况吗？到最后的凝丹步骤时，冰属性气息与火焰相冲，导致炸炉。”
<br/>　　“嗯。”叶铭点点头，道，“雪魄丹本身就是极寒之药，与我们的火焰产生抵触。火焰稍弱些，冰寒气息盖过了火焰，根本无法凝丹。”
<br/>　　“火焰稍猛些，冰寒气息又失控了，导致炸炉。”
<br/>　　方鹤点点头，道，“除非能找到两者的平衡点，以此下手，否则这丹根本无法炼制成功。”
<br/>　　叶铭点头道，“这个平衡点太难把握了，我们以前没有炼制过雪魄丹，没有经验，要靠自己去摸索的话，没个十次八次炼制尝试，几乎做不到。”
<br/>　　“可是，我们每人都只有两份材料，只能炼制两次，这次失败了，便只有最后一次机会。”
<br/>　　这也是叶铭皱眉的原因。
<br/>　　这时，慕容胤见没人和他说话，显得有些尴尬。看到萧逸还在控制着炼药炉里面的紫火，顿时嗤笑道，“臭小子，你都炸炉了，还装模作样做什么。”
<br/>　　三人的炼药炉炸炉后，都还残留着熊熊火焰在里面。
<br/>　　虽然萧逸还在继续炼药，但单从外表看，四人的火炉内都是一片火焰，一模一样。
<br/>　　自然外人也以为萧逸也炸炉了。
<br/>　　慕容胤见萧逸没有搭理他，变得更为尴尬和愤怒。
<br/>　　“臭小子，想来你也是没什么本事的，便不要浪费材料了。你那材料，便归我了，我多一次炼制机会，成功几率也大些。”
<br/>　　慕容胤说着，就欲用灵气将原本在萧逸身旁的另一份材料卷走。
<br/>　　“滚。”萧逸满含杀意的喝了一声，而后将身旁的另一份药材悉数投入炼药炉中。
<br/>　　“老子就是烧了也不给你。”萧逸冷冷地说道。
<br/>　　他现在对北山慕容家的人，只有杀意。
<br/>　　“你...”慕容胤顿时大怒，指着萧逸道，“易霄，你这样浪费材料，是在罔顾白家主女儿的性命之危。”
<br/>　　白家主在一边看着，脸色也变得极其难看。
<br/>　　在他看来，易霄就算真的是四品炼药师，但炼药术也是比不上其余三人的。
<br/>　　易霄如此浪费一次炼制机会，无异于拿他女儿的性命开玩笑，这让他心头涌现了杀意。
<br/>　　当然，易霄现在已经将药材投入炼药炉了，他也发作不得，免得影响了易霄。
<br/>　　就算他对易霄信心不大，但总归要试一试。
<br/>　　在场武者，在听到萧逸的话，以及看到萧逸的动作时，也是脸色一变。
<br/>　　“遭了，这易霄在和慕容胤斗气。”
<br/>　　“只是这气是出了，但得罪了白家主，此番炼药后，若他失败，怕是大难临头。”
<br/>　　“如此罔顾白家主唯一的女儿的性命，他也是自己找死。”
<br/>　　在场武者都在失望地摇头，仿佛已经认定了萧逸之后的下场会很看。
<br/>　　但萧逸本人，却充耳不闻。
<br/>　　他根本没炸炉，所谓的寒冰气息和火焰的平衡点，他了如指掌，怎么会失败。
<br/>　　冰鸾剑早就让他对火焰的控制把握得极为完美。
<br/>　　第一份药材，其实已经差不多要真正凝丹了。
<br/>　　刚才第二份药材投进去，不过是再一次开始炼丹罢了。
<br/>　　旁边，叶铭、方鹤、慕容胤三人也再次开始了第二次炼丹。
<br/>　　无论有没有把握，他们终归要试试。
<br/>　　一个多时辰后，毫无悬念地，三人再次炸炉。
<br/>　　雪魄丹，果真是太难炼制了。没有足够的炼制经验，几乎不可能成功。
<br/>　　在失败一次后，就能找到平衡点，这样的本事，怕是只有五品炼药师才能做到。
<br/>　　这三人不过是四品炼药师，和五品炼药师的差距太大了。
<br/>　　“唉。”三人都叹了口气，炼药炉中的药力，已经毁于一旦，他们再次失败了，也没有机会了。
<br/>　　就连叶铭，脸色也是一阵难过，毕竟这事关一条人命。
<br/>　　反倒是慕容胤，立刻跳了起来，愤怒的指着萧逸，道，“臭小子，都是你，若不是你刚才浪费了一份药材，我们三人其中一个，必然能再炼制一次，说不定就成功了。”
<br/>　　“哼，若白家主的女儿有个三长两短，就算白家主宽宏大量，不与你计较，本执事也饶不了你。”
<br/>　　事实上，白家主哪里会宽宏大量。慕容胤这时如此说，不过是想把责任都推到萧逸身上，自己撇清关系罢了。
<br/>　　在场武者也是纷纷摇头，“易霄这次确实过分了。”
<br/>　　“就为了斗气，竟然浪费掉一份材料，浪费掉一次炼制机会。”
<br/>　　“唉，白家主饶不了他的。”
<br/>　　一旁的白家主，脸色黑得似乎要滴出墨来，心头更是伤痛无比。
<br/>　　“完了吗？冰雪她，真的没救了吗？时间紧迫下，请不来五品炼药师，如今药材又耗尽了...”
<br/>　　叶铭、方鹤、慕容胤三人失败后，其实他已经没有任何信心了。
<br/>　　“易霄...。”白家主忽然变得咬牙切齿，但下一秒，他又叹了口气，“哎，算了，错不在他。”
<br/>　　孰料，他的叹气声刚刚落下，萧逸的炼药炉中，忽然一股骇人的冰冷气息冲天而起，让得在场武者同时一惊。
<br/>　　白家主则神色一震。
