        童筱颖听着都觉得揪心，一手拖着行李箱，担忧地往大哥傅夜沉的房门口走去。
<br/>
<br/>        大哥傅夜沉的房门是敞开着的，里面或许是因为窗帘被拉上的缘故，光线有些昏暗。
<br/>
<br/>        “大哥？大哥你还好吗？”童筱颖一边往里面走，一边试探性地问。
<br/>
<br/>        当她走进房间里，并未看到大哥傅夜沉的人影，正准备转身离开的时候，一丝敏感漫上心头，让她下意识地侧过身去，看向了那边书房的房门。
<br/>
<br/>        “大哥？”童筱颖又唤了声。
<br/>
<br/>        她刚走到门口，里面突然传来傅夜沉一声冷斥：“滚！”
<br/>
<br/>        童筱颖怔住，驻足在房门口，不敢贸然踏入半步。
<br/>
<br/>        “那个……大哥……你是不是发生什么事情呢？要不，我叫刘婶过来帮你看看？”童筱颖弱弱地问道。
<br/>
<br/>        因为，她在空气里，闻到了一股血腥味。
<br/>
<br/>        “你是筱颖？”傅夜沉的语气稍稍缓和了点，不再像起初那样冷酷无情。
<br/>
<br/>        童筱颖轻轻地应了声：“嗯。”
<br/>
<br/>        “你……不用管我。”傅夜沉的声音有些暗哑。
<br/>
<br/>        “大哥，你是不是受伤了？”童筱颖追问道，索性直接走了进去。
<br/>
<br/>        书房内，灯光昏黄染着几分暧昧温情，但在血腥味的渲染下，这份暧昧温情不再，反倒添了些许阴森。
<br/>
<br/>        傅夜沉微微颤抖着身子，屈腿坐在书房的落地窗前，厚重的窗帘被拉上，他低垂着眸子看不清神色，可半裸的胸膛，以及面上不正常的潮红和他因过度隐忍而暴起的青筋，让他的一切极其不正常。
<br/>
<br/>        “谁让你进来的？”傅夜沉抬眸看了一眼童筱颖后，迅速收回自己的目光，他随之从地上站起身来，脚步略显吃力地缓慢地走到沙发旁，慢慢地坐下。
<br/>
<br/>        童筱颖看向傅夜沉，弯弯的柳眉不经意间地挑了下，潋滟的眸底闪过一丝惊愕。
<br/>
<br/>        只见鲜血浸透了他的西装裤，然后顺着皮鞋流到地上，很快蜿蜒一地，看起来触目惊心。
<br/>
<br/>        “大哥，你等一下，我去给你拿药！”童筱颖转身跑到门外自己的行李箱旁，将行李箱放倒，打开行李箱后，从里面拿出了一个小药箱。
<br/>
<br/>        这是她的个人习惯，不管去哪里旅行，她都会带上药箱以防万一。
<br/>
<br/>        童筱颖提着药箱，径直走到了傅夜沉的身边蹲下身去。
<br/>
<br/>        他的西裤上破了个洞，洞口有些血肉模糊。
<br/>
<br/>        童筱颖从药箱里拿出剪刀，轻轻地捏起西裤，然后把血肉模糊的伤口周遭的布料全部剪掉。
<br/>
<br/>        她慢条斯理地替他处理伤口，他正襟危坐地坐在沙发上眉头都不皱一下。
<br/>
<br/>        傅夜沉只觉童筱颖的手很凉，碰到他的肌肤上，让他觉得很舒服。
<br/>
<br/>        刚刚被他压抑下去的那股躁动，又“噌”地一下蹿了上来，理智渐渐地往丧失的边缘滑动。
<br/>
<br/>        傅夜沉犀利的黑眸，目光灼灼地打量着正在埋头替他自己处理伤口的童筱颖。
<br/>
<br/>        只觉她的耳垂，白嫩圆润，缓滑的下颚骨线条，美丽地延伸至她的粉唇下。
<br/>
<br/>        傅夜沉滚动了一下喉结，腹下一直处于紧绷状态，令他燥热难耐。
<br/>
<br/>        他饱满的额头上，再次渗出细密的汗珠，呼吸越来越深沉。
<br/>
<br/>        慢慢地，体内的药效又发作了，一丝一毫地，吞噬着他的理智。
<br/>
<br/>        傅夜沉下意识地闭上眼睛，紧抿薄唇，努力克制着自己。
<br/>
<br/>        “滚！”
<br/>
<br/>        他想让自己对她语气尽量放温柔一点，但最终还是对她说了狠心的话。
<br/>
<br/>        童筱颖头也不抬地继续替傅夜沉包扎伤口，并且没好气地说道：“我给你包扎好就‘滚’行了吧！你这是怎么伤到大腿的啊？动脉血管都被你扎破了！要是再不及时给你处理伤口，你会失血过多而晕过去的！”
<br/>
<br/>        其实，不管是闭上眼睛，还是睁开眼睛，他的心里脑海里想要的女人，全是她童筱颖。
<br/>
<br/>        她若不说话还好，她一说话，他便受不了了，因为他已经忍到了极限。
<br/>
<br/>        傅夜沉赫然睁开双眼，并瞪大了嗜血般的眼睛，像一只雄狮打量着猎物一般死死地盯着童筱颖。
<br/>
<br/>        他的俊脸涨得通红，眼里布满血丝，头顶隐隐地冒着热气。
<br/>
<br/>        童筱颖给傅夜沉包扎完伤口后，又从药箱里掏出了一个小瓷瓶，从里面倒出两粒咖啡色的小药丸，抬起头来朝傅夜沉递了过去。
<br/>
<br/>        “喏，把这个吃了！止血止痛的！”童筱颖接着说道。
<br/>
<br/>        她递过来的药丸，傅夜沉迟迟未接。
<br/>
<br/>        童筱颖见他不理会自己，刚准备起身，猛然间腰就被一把抱住，猝不及防地跌坐在大哥傅夜沉的另一只大腿上。
