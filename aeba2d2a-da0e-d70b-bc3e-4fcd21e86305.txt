霍西泽觉得自家二哥还真是另类，人家来酒吧喝酒都会叫美女来陪，二哥这里就好像是这些美女会玷污了他一样，再美的美女都别想赚他的钱。
<br/>
<br/>        不过霍西泽可不敢惹陆寒霆，他麻溜的滚了出去。
<br/>
<br/>        陆寒霆又喝了几瓶酒，酒的后劲很大，他深邃的狭眸里染上了几分微醺，有点醉了。
<br/>
<br/>        这时豪华包厢门被推开，一道身影走了进来，是夏妍妍。
<br/>
<br/>        前段时间夏妍妍元气大伤，所以这些日子她十分的安静，不过安静的外表下她肯定在酝酿着什么大招。
<br/>
<br/>        夏妍妍走进来，来到陆寒霆的身边，她轻声唤道，“陆总…陆总…”
<br/>
<br/>        陆寒霆醉了，没有给予回应。
<br/>
<br/>        “陆总，我送你回房间吧。”夏妍妍扶起了陆寒霆，将他带出了豪华包厢送到了一个总统套房里。
<br/>
<br/>        ……
<br/>
<br/>        总统套房里，夏妍妍将陆寒霆放在了柔软的大床上，她爱慕迷恋的看着陆寒霆这英俊成熟的五官，然后伸手，去解陆寒霆身上的黑色衬衫。
<br/>
<br/>        一颗纽扣，两颗纽扣解开了，露出了男人精致的扇骨，夏妍妍漂亮的小脸迅速红了，整个人心跳加速。
<br/>
<br/>        正准备解第三颗纽扣时，一只大手突然拽了过来，一把扣住了她的手臂。
<br/>
<br/>        夏妍妍呼吸一紧，等她抬头看时，床上的陆寒霆已经缓缓睁开了眼，他那双微醺的眼眸又黑又沉的落在了她的脸上。
<br/>
<br/>        每当和这个男人对视时，夏妍妍都觉得畏惧又紧张，财富权势地位叠加的男人气场太过于强大，不动声色盯着你的时候，好像能将你的灵魂都给看穿。
<br/>
<br/>        “陆…陆总…”夏妍妍吓得磕磕绊绊的。
<br/>
<br/>        这时陆寒霆伸手一扯，直接将她扯了过去。
<br/>
<br/>        夏妍妍迅速倒在了他的怀里，男人身上那股干净清冽的味道还夹杂着淡淡的烟酒味瞬间就冲击进了她的鼻翼里，好闻到致命。
<br/>
<br/>        夏妍妍全身一软，几乎像水一样软在了他的怀里。
<br/>
<br/>        这一切就像是做梦，夏妍妍在梦里无数次的梦到自己睡在陆寒霆的怀里，还跟他翻云覆雨。
<br/>
<br/>        没想到这一天终于实现了。
<br/>
<br/>        “陆总…”夏妍妍娇滴滴的叫了一声。
<br/>
<br/>        陆寒霆将她抱在自己的怀里，英俊的眼睑敛了下来，沙哑的呢喃出声，“绾绾…绾绾…”
<br/>
<br/>        他一遍遍的呢喃着夏夕绾的名字。
<br/>
<br/>        夏妍妍脸上的笑容都僵住了，像是一盆冷水扑了下来，原来他将她当成夏夕绾的替身了。
<br/>
<br/>        不过，机会难得，不容错过。
<br/>
<br/>        夏妍妍的手顺着他精硕的腰身往下滑去，“陆总，你醉了，我帮你把衣服脱掉，我们一起睡觉吧。”
<br/>
<br/>        陆寒霆没有拒绝，他将自己的俊脸埋在了夏妍妍的长发里。
<br/>
<br/>        几乎是下一秒，陆寒霆倏然睁开了眼，眸底的微醺全部退去，他伸手一挥，直接将夏妍妍挥落下床。
<br/>
<br/>        轰，夏妍妍以极其狼狈的姿势摔在了地毯上，疼的惊呼出声。
<br/>
<br/>        陆寒霆迅速起身下床，他居高临下的冷睨着她，目光凌冽阴冷的几乎将她给凌迟，“夏妍妍，谁让你进来的，你是不是活腻了？”
<br/>
<br/>        夏妍妍觉得自己浑身的骨头都被他那一挥给挥断了，她脸色苍白的看着陆寒霆，“陆总，我看你醉了，只是好心将你送到房间里，刚才你将我当成绾绾了…”
