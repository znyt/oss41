　　当楚风眠从这道路上，冲上去的一刻，周围的压力，便是全部都散为无形之中。
<br/>　　战龙宝车，冲上这山峰之上，一眼望去，便是一道彩虹阶梯。
<br/>　　在这彩虹阶梯上，一眼望去，楚风眠便是看到了就在其中大约十分之一的地方，便是一枚闪耀黑色的灵石，清晰可见。
<br/>　　“黑耀十方石？”
<br/>　　楚风眠看到这黑色灵石的一刻，都是不由的开口一声。
<br/>　　这黑耀十方石，可是诸天奇物之中的极品，其中蕴含有着独特的灵力，可以将武者身躯之中血脉提炼纯粹。
<br/>　　一般武者的血脉，若是被这黑耀十方石所提炼过的话，就足以媲美一些例如真龙血脉的血脉。
<br/>　　而对楚风眠这样，本身便是有着其他血脉的武者来说，这黑耀十方石的作用还会更强。
<br/>　　楚风眠修炼战龙决，他所拥有的远古战龙血脉，虽然也称得上是一种极为极品的血脉，可是实际上，天地之间依然还有着足以超越战龙血脉的血脉。
<br/>　　像是鲲鹏，天凰，许许多多天地异兽，他们的血脉，可是比起远古战龙血脉都要强大。
<br/>　　若是现在楚风眠可以得到这黑耀十方石，便是足以将他的血脉，再度淬炼，倒时候足以让楚风眠的实力，大进一步。
<br/>　　这黑耀十方石，绝对是对任何武者来说，都是绝对的宝物。
<br/>　　放在九域中，这黑耀十方石在都已经是有着数万年一来没有出现过了，楚风眠也只是从一些古籍之中，才见到过关于这黑耀十方石的记载。
<br/>　　云羽天宫，果然富足，这黑耀十方石，在当初远古时代都是极为珍贵。
<br/>　　眼下这黑耀十方石，居然就放在这彩虹阶梯之上，简直是唾手可得。
<br/>　　不止是楚风眠，同样也有着无数目光，看向这黑耀十方石的所在，几乎刚刚冲上来山峰的武者，都被这黑耀十方石所吸引，目光炙热无比。
<br/>　　这种天地至宝，在归墟天中都是难得一见。
<br/>　　“是黑耀十方石！”
<br/>　　“天啊，这不是早已经在九域中消失的奇物吗？相传这黑耀十方石都不是九域之物，曾经是从域外而来，用一块少一块，居然如此在此！”
<br/>　　“我本就是阳虎血脉，要是让我得到这黑耀十方石，居然让我的血脉都再度蜕变，倒时候所向无敌！”
<br/>　　那些武者，一个个看向这黑耀十方石，恨不得现在就要将其夺走。
<br/>　　就连寒月漓，看向这黑耀十方石的目光，都是炙热无比，眼神中渴望得到。
<br/>　　这黑耀十方石，不同于其他诸天奇物，在无数诸天奇物之中，有着一些只对某些境界的武者有用。
<br/>　　像是之前楚风眠得到过的龙诞草，也只是对生死境之下的武者有用，对生死境武者，都没有任何用处，所以价值有限。
<br/>　　可是这黑耀十方石却是不同，黑耀十方石的效果，跟武者的境界，实力，没有任何的关系，他所做的，是蜕变血脉。
<br/>　　就算是对一些碎命之境，甚至是更高境界的强者来说，都有着大用处，人人都想要得到之物，这价值根本无法衡量。
<br/>　　但是这么多人，无数目光，却最多都只是在看看，而并没有真正有人，敢来上前争抢。
<br/>　　这一幕倒是让楚风眠稍微有些意外，归墟天的武者，一个个都是疯子，为了利益不择手段。
<br/>　　刚刚在山脚下，为了向前一步都可以混战起来，面对这黑耀十方石，这种实打实的宝物，应该是不可能如此冷静才对。
<br/>　　不过楚风眠观察了一眼，便是看见了，在这彩虹阶梯上的血祭。
<br/>　　好像是曾经有一些武者想要踏入这彩虹阶梯上，但是却当场陨落，尸骨无存，只留下一些血祭。
<br/>　　在这彩虹阶梯上，还弥漫有着一种七彩霞光，令人不敢踏足，一旦踏足其中，便是会承受无与伦比的轰击，没有任何人可以抵挡得住。
<br/>　　正因这样，现在那些武者，虽然满是贪婪，却没有一人敢上前一步，去争夺这黑耀十方石。
<br/>　　大部分的武者，观察到这一幕后，都是安静的站在了一边，等待着。
<br/>　　他们也清楚，这云羽天宫种种异变，应该都是传承快要开启的象征，甚至是已经开启。
<br/>　　刚刚那爬上山峰，便是第一道考验。
<br/>　　没有通过这一道考验的武者，就连见到这黑耀十方石一面的机会都没有。
<br/>　　大部分的武者，都是沉下心来，在一旁等待，不过却也有着一些武者，有些忍耐不住，想去冒险。
<br/>　　其中一名武者，刚刚踏入这彩虹阶梯的一刻，他的身躯，便是彻底泯灭，只在地上，留下了几滴鲜血。
<br/>　　这彩虹阶梯上的七彩霞光，其中的威力，远远不是现在这些武者，足以抗衡的。
<br/>　　楚风眠看到这一幕，心中自然也是清楚的很，他便是乘坐战龙宝车，在一旁安静的等待着。
<br/>　　“唰！唰！唰！”
<br/>　　一座巨大无比的天舟，突然冲了上来，在天舟上，二十几道身影，纷纷出现。
<br/>　　这些人，便是刚刚想要去对付楚风眠的那些荒古门弟子。
<br/>　　“哼！”
<br/>　　其中几名荒古门弟子，看到楚风眠的一刻，便是眼神中露出了几分杀意，一身灵力澎湃，都要即将出手。
<br/>　　但是在旁边，那年轻男子，却是摇了摇头，最后还是一群人在，站在了一旁去。
<br/>　　“荒圣君来了。”
<br/>　　看到这年轻男子，不少武者的眼神中都是露出了几分忌惮之色。
<br/>　　荒圣君，剑祖荒，鹿战神，这三人在天羽神域中，便是高高在上是三人，位列巅峰。
<br/>　　任何武者想要去跟他们为敌，都需要仔细考虑一二才行。
<br/>　　而除了荒古门弟子，逐渐的又一群武者，进入其中，这群人皆是目光深邃，是古月峡谷的弟子。
<br/>　　在人群中，刚刚被楚风眠打下山峰的古鹏，赫然在列，他的脸色还有些苍白，不过伤势却是好了不少，应该是服用过了一些丹药。
<br/>　　他上来的一刻，目光也是看向了楚风眠，死死的瞪了楚风眠一眼，随后站在了一旁。
<br/>　　“看来这古鹏，不会善罢甘休的。”
<br/>　　寒月漓在一旁，看到这一幕开口说道。
<br/>　　“无妨，一个蝼蚁罢了，谁敢挡我，谁就死！”
<br/>　　楚风眠平静的开口道。
<br/>　　一个古鹏，楚风眠还没有放在眼中。
