　　楚风眠一行人，离开黑市之后，便是来到了一处客栈，住下。
<br/>　　楚风眠回到房间之中，便是布置好了无数禁制阵法，将这房间中的一切，彻底隐蔽起来。
<br/>　　楚风眠盘膝坐在床上，扫视了周围一眼，这才将那独牛内丹，给拿了出来，观察着。
<br/>　　“这天晶的力量，已经是全部融入到了内丹之中了，看来这头独牛，应该是将天晶给吞噬了，所以才会成长都如此地步。”
<br/>　　看着手中的独牛内丹，楚风眠自言自语道。
<br/>　　天晶，就连人的血脉都可以净化，一头独牛，吞噬了他，踏入御风境，自然轻松无比。
<br/>　　这还是因为独牛的资质，实在是太差，若是换做强一些的武者，可能会直接修成妖神。
<br/>　　倒时候也许真的可以将这天晶的力量炼化。
<br/>　　毕竟不入神力境，就算是得到这天晶，也不能完全炼化，楚风眠感觉的到，这一枚天晶的力量，还剩余至少九成之多。
<br/>　　那独牛修炼到如此地步，也最多只是耗费了天晶不足一成的力量。
<br/>　　天晶中大部分的力量，如今还都是保存了下来，这对楚风眠来说倒是一个好消息。
<br/>　　虽然说如今天晶的力量，因为被那独牛吞噬，已经是彻底的融入到了他的内丹之中去。
<br/>　　但是对楚风眠来说，想要将这天晶的力量炼化出来，并非难事。
<br/>　　“还是先将天晶的力量，彻底炼化出来再说。”
<br/>　　楚风眠心中想到。
<br/>　　这天晶，若是被其他人发现，只怕是会引来无数强者的争抢，毕竟天晶这等奇物，对在强大的武者，都有着用处。
<br/>　　北邙学院的底蕴，远超楚风眠的想象，楚风眠如今还是趁早炼化了这天晶。
<br/>　　以增添他的实力，好让楚风眠在北邙学院中有着立足的资本。
<br/>　　想到这里，楚风眠便是心神一动，一股火焰，出现在他的掌心之中。
<br/>　　这一股火焰，呈现出金色，乃是龙炎，龙炎灼烧，这内丹便是开始融化。
<br/>　　几个呼吸之间，这内丹，便是被彻底融化。
<br/>　　其中的一股极为特殊的力量，缓缓的凝聚了出来，化为了晶莹的颜色。
<br/>　　楚风眠便是看的出来，这力量，便是天晶的力量，楚风眠将这独牛内丹，彻底融化。
<br/>　　所留下的，便是最为精粹的天晶力量。
<br/>　　“吞！”
<br/>　　楚风眠没有任何犹豫，看到这一股力量出现的一刻，顿时张口，一下子将这天晶力量全部吞入到了身体之中去。
<br/>　　同时楚风眠便是运转起身体之中的力量来，造化之力！战龙之力！同时催动起来，开始炼化这天晶的力量。
<br/>　　天晶之力，不难炼化，随着楚风眠力量的吞噬。
<br/>　　这其中的天晶之力，便是化为了一股股细小的灵力，融入到楚风眠的身躯之中去。
<br/>　　逐渐的，楚风眠便是都感觉到了一种畅快的感觉。
<br/>　　天晶，足以让武者身躯中的力量，都得到净化。
<br/>　　在强大的血脉，吞噬品级在高，灵力在精粹的丹药，都会在武者的身躯之中，留下杂质。
<br/>　　这杂质，虽然看起来不起眼，但是却影响这武者的身躯。
<br/>　　就算是楚风眠，这等强大的血脉，在这天晶的力量之下，楚风眠都感觉到，他的皮肤上出现了不少黑色的淤泥。
<br/>　　这些便都是武者体内的杂质，楚风眠没少吞服丹药，这些丹药产生的杂质，如今随着天晶的力量，彻底的都给排了出去。
<br/>　　楚风眠感觉到，他现在的身躯，已经是变的清爽了许多，似乎是一举一动之间，灵力在手中的控制，都是显得顺手了许多。
<br/>　　天晶，虽然不足以让武者的境界有着任何的进步，但是他的效果，可是一切丹药都比拟不了的。
<br/>　　就算是天元神丹，都不能跟这天晶媲美。
<br/>　　天晶的力量，是足以让武者的身躯都进行蜕变，整个血脉之力，都会纯净许多。
<br/>　　如今楚风眠的力量，虽然没有任何的变化，但是他感觉的到，他的实力比起刚刚，只怕是还要强大许多。
<br/>　　因为身体中没有杂质，完美的身躯，不会受到任何的影响，一举一动都可以做到最为完美。
<br/>　　可以说现在楚风眠的身躯，便是真正完美的身躯，将身躯塑造到了最为完美的境地。
<br/>　　已经是达到了现在做能够做到的一切极限，楚风眠如今所差的，便是进入到北邙学院之中去，然后寻找机会，踏入神力境。
<br/>　　“怎么回事？你的身躯怎么发生了如此翻天覆地一般的变化？”
<br/>　　一道声音，突然从楚风眠的脑海中响起。
<br/>　　这让楚风眠似乎都有些一愣，随后才想了起来。
<br/>　　“星玄，你苏醒了？”
<br/>　　这声音，正是一直隐藏在楚风眠身躯中的那把魔剑，星玄的声音。
<br/>　　星玄如今大部分的时间，都在沉睡，楚风眠都快要忘了他的存在了，没想到他如今居然是苏醒了过来。
<br/>　　“不错，你到底是吞噬了什么东西，怎么让你的身躯蜕化到了如此完美的地步？”
<br/>　　星玄观察这楚风眠的身躯，似乎也是微微一愣。
<br/>　　“我得到了一块天晶。”
<br/>　　楚风眠淡然回答道。
<br/>　　“天晶？”
<br/>　　听到这个名字，星玄明显有些一愣，似乎没有记得这个名字。
<br/>　　他一股灵力，扫视了一下楚风眠的身躯，楚风眠便是感觉到，他身上隐藏的一切，仿佛都被这星玄尽收眼底。
<br/>　　“这星玄曾经巅峰的时候，得有多么强大，我的身躯，让生死境武者都不可能看穿一丝一毫，在他的面前，居然是毫无办法隐藏？”
<br/>　　楚风眠有些吃惊的说道。
<br/>　　他刚刚故意隐藏身躯中的力量，想要试探一下。
<br/>　　但是他的隐藏，在这星玄的面前，却是没有任何意义。
<br/>　　“这是，星晶之力？”
<br/>　　星玄查看着楚风眠身躯中留下的力量，似乎有些吃惊的自言自语一句。
<br/>　　“这只有四级星域之上才会出现之物，怎么会出现在这里？”
<br/>　　“星晶？四级星域？”
<br/>　　听到星玄的话，楚风眠的心中也是极为吃惊，这星玄，居然是认识这天晶。
<br/>　　天晶其实便是九域起的一个名字，因为天晶来自域外天空，便是名为天。
<br/>　　但是如今看样子，星玄居然认识这天晶到底是什么来历。
