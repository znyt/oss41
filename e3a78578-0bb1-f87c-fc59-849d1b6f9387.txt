　　漫步在色彩缤纷的花海中，陆雪琪触景生情，臻首情不自禁的靠在慕清霄宽阔的肩膀上，空灵清绝的脸蛋上牵起一丝温柔的笑意。
<br/>　　她这一笑，天地都为之失色，犹如九天仙女下凡，不染一丝凡尘。
<br/>　　无数的美丽花朵在青天之下，突然间一起欢笑一般，衬着她绝世容颜，骄傲盛开，整个花海之中，她便是最靓丽清艳的那一抹颜色。
<br/>　　慕清霄不禁呆了一下，嘴角牵起一丝笑意，自从两人形影不离，他还是第一次见陆雪琪笑的如此幸福，哪怕到最后没有收获，此行也是值了。
<br/>　　右手攀上她的腰肢，极致柔软的触感传来，他的心中没有一丝杂念，眼中尽是溺爱，两人就如此互相依偎，行走在花海中，片刻后，终于停下了步伐。
<br/>　　眼前的树干，被粗壮的藤蔓所完全覆盖，鲜花争奇斗艳，自上而下，凝聚成一面墙，而在花海中央处，忽然耸立着一座石门。
<br/>　　石门高五丈，宽三丈，硬生生的嵌入树干之中，周围被无数藤蔓鲜花所淹没，只留出中间厚实的巨石，上边刻着古篆体的四个大字。
<br/>　　“天帝宝库！”
<br/>　　隐隐约约的，有什么黄钟大吕般的声音，回荡在青天之际，震动心魄。
<br/>　　慕清霄的视线落在前面，那座高大的石门之上，嘴角勾起一丝笑意，道：“天帝宝库，最后一卷天书就在其中。”
<br/>　　陆雪琪望着石门，道：“我们如何进去？”
<br/>　　如何进去，这是一个令人头疼的问题，虽然天帝宝库近在眼前，但如何进去，慕清霄还真不知道。
<br/>　　微微摇头，慕清霄走上前去，心神停留在天帝宝库四个古篆体上，隐隐约约觉得，这四个字与自己有什么联系。
<br/>　　于是，慕清霄伸出右手，指尖与字体接触的瞬间，体内涌出一窜窜金色光芒......
<br/>　　望着这一幕，慕清霄微微一愣，终于知道，天帝宝库为何会提前开启，因为他体内的四卷天书，形成了契机，从而提前开启宝库。
<br/>　　金色光芒宛若满天星辰，此刻汇聚在青天之上，在光芒的照射下，‘天帝宝库’四个古篆大字，闪闪发光。
<br/>　　就在慕清霄好奇的时候，天帝宝库的石门上，传来沉闷的轰鸣声，此刻，正是青天上的金芒，照射在了‘天帝宝库’四字古篆中‘天’字的最上一横上。
<br/>　　记得原剧情中，黑水玄蛇来袭，但在万蝠古窟，无情海中，慕清霄早就将黑水玄蛇斩杀，天帝宝库开启，自然就没有它出场的戏码。
<br/>　　伴随着金芒的照射，天帝宝库石门上的声音渐渐高昂，伴随着一声轰鸣，在金芒的照射下，本来完整一块的巨大石门，突然从中间裂开一条缝隙，然后缓缓向旁边移开。
<br/>　　金色耀眼夺目的光芒，自缝隙中轰然涌出，即使是在白天，竟然也是这般灿烂的不可逼视，连天际冉冉升起的太阳，此刻视乎也变得暗淡无光。
<br/>　　仿佛有什么东西，在天帝宝库之中咆哮着，在金色的光芒中轰鸣着！
<br/>　　青天上的金色光芒也逐渐变化，仿佛是在改变顺序，如同密码锁一样，步骤繁杂，光芒再次照射下来，从天帝宝库四个古篆中的‘天’字一横上又向上面移了一些。
<br/>　　陡然，原本开了一半大约三尺来宽的石门，突然停止了继续一动，片刻后，居然开始合拢，青天上的金芒也再次莫入慕清霄的体内。
<br/>　　回过神来，慕清霄眼疾手快，直接搂住陆雪琪，向缝隙中疾射而去。
<br/>　　石门正在低沉闷响声中渐渐合拢，石门缝隙之间的距离越来越紧密，剑光一闪，石门轰然关闭。
<br/>　　紧接着，在慕清霄与陆雪琪进入天帝宝库的瞬间，这道巨大的石门在巨响声中合拢，中间的那道缝隙，也不可思议的消失，无影无踪。
<br/>　　与此同时，天帝宝库外的天色突然昏暗了下来，只见九天之上，忽然出现一大片橙黄色彩，将数十仗方圆笼罩在其中，连阳光都给遮掩了去。
<br/>　　随后，天际仿佛传来一声凤鸣一般的清啸之声，那片云彩遮天蔽日一般的落了下来，神秘生物也逐渐露出其貌。
<br/>　　出现在上空的，居然是一只周身橙黄羽毛，高七丈的大鸟，展翅翱翔，盘旋于青天之上，黝黑的瞳孔盯着天帝宝库的石门，露出情绪化的光芒。
<br/>　　它好像在紧张！
<br/>　　此时，慕清霄已经带着陆雪琪来到石门内部，周围一片黑暗，然后，在前方不远处，缓缓亮起一道幽幽的光芒，就像是黑夜中的一盏微弱烛火，静静燃烧。
<br/>　　两人默默的望着深处，那道光是轻微的淡紫色，然后转换成青色，接着又慢慢转换成深绿，便是如此，不停的变换，闪烁着各色光芒，煞是好看。
<br/>　　周围一片黑暗，陆雪琪静静地问道：“这里就是天帝宝库内部吗，其中可能凶险万分，我们小心点。”
<br/>　　“嗯，走吧。”
<br/>　　说罢，慕清霄就牵着她，向那道闪烁着神秘色彩的光线中走去，当彼此拉近距离，映入眼帘的东西也变得清晰起来。
<br/>　　这是在黑暗中竖立着的一道木台，一米高左右，大约婴儿手臂粗细的一根圆形木柱连入地下，上端拖着一个一尺大小的平台。
<br/>　　平台中间，放着一只形状古拙的杯子，看着也似木头所做，仔细看去，便可以看出，这整个木台和杯子，竟然都是完整的连接在一起，与地下的树木连为一体。
<br/>　　然而，最吸引人的，却是存于这木杯中的东西。
<br/>　　木杯大概有三寸高，两寸宽，中间盛着一种透明的液体，而在杯中浮在那液体之上的，还有一颗小小的透明石头，做五面平整状，晶莹剔透。
<br/>　　而就是这颗透明的石头上，发射出柔和的光线，在小小木台上形成一道半圆形状的光幕，折射出各色光芒，向四周散发而出。
<br/>　　此时，两人已经站在木台前，向木杯望去，空气中，隐隐飘荡着奇异的香气，无所不在，仅仅是闻上一丝，便使人精神为之一振。
