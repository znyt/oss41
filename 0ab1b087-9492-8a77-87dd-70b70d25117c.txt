　　　　罗昊确认楼房安全之后，竖起左手食指在自己头顶上画着圈，示意其余三人向自己靠拢，集合。
<br/>　　打开绑在左手腕上的单兵PDA，查看了一下PDA上的卫星地图，罗昊看了眼身边的三个人，说道：“现在我们的位置距离被反政府武装人员所控制的区域，还有十公里，这剩下的十公里就需要我们自己徒步进行穿越，中间可能会遇到反政府武装势力的巡逻队，一切以规避为第一选择。如果不得已必须要交火，无声战斗歼灭敌人，动作一定要快，绝对不能让反政府武装的巡逻队呼叫支援抵达。我们这次是以侦察为主要目的，找到我们队员的下落，方便后面实施营救。”
<br/>　　“明白！”
<br/>　　李然、慕容飞、李毅峰三个人沉声回答。
<br/>　　李毅峰打量了一眼自己现在所处的楼房，说道：“这里以前似乎是一栋居民楼，但是现在这里的居民因为战乱，全都离开了。”
<br/>　　“也许是离开了，也有可能是被杀了。”
<br/>　　慕容飞指了指留在墙壁上的弹孔、散落在地面上的弹壳、还有那些喷洒在地上和墙壁上的血迹。这些血迹的颜色已经变黑，显然已经是留下了有一段时间。
<br/>　　“我更倾向于这里是一个行刑地。”
<br/>　　李然指了指墙壁上的弹孔和血迹，说道：“你们仔细看这些弹孔，不觉得它们的排列很整齐吗？如果那些遭到处决的人被迫跪在这个位置，反政府武装人员就站在我现在所在的位置对他们开枪射击，造成的弹孔位置和血迹喷洒的情况，不就是我们眼前所看到的样子吗？”
<br/>　　在李然的描述下，罗昊、李毅峰、慕容飞三人眼前出现了这样一幅画面。
<br/>　　几个被反政府武装人员所抓捕的平民被强迫着跪在他们面前的墙壁前，然后对着他们的后脑勺举起AK自动步枪，在一连串点射中，那些平民中弹倒地，随后他们的尸体被带走，可能是被仍在一处无人问津的角落里，一点点的腐烂发臭，最后化成一堆白骨；亦或者是被送去解剖，挖出他们身体中有价值的器官送去黑市贩卖。
<br/>　　“这群该死的混蛋！”
<br/>　　李毅峰紧握着拳头，用力的砸在墙壁上，眼中跳动着愤怒的火焰。
<br/>　　罗昊把手放在李毅峰的肩膀上，用力的按了一下，说道：“我们还有事情去做，必须要保持冷静。”
<br/>　　“我明白。”
<br/>　　李毅峰将M2010超级狙击步枪背在身后，将背包一侧的ACE-GAP39SB短突击步枪握在手中，在空间狭窄的城市街道中，结构紧凑短小的ACE-GAP39SB短突击步枪会比狙击步枪更适合，7.62mm口径的弹药能提供强大的火力。
<br/>　　楼房中四个人拉起缠在脖子上的幽灵面罩，遮住自己的口鼻，就如同四道幽灵一样，离开了这栋曾经被反政府武装势力当做刑场，对平民进行屠杀的楼房，进入街道、小路、弄堂纵横交错的小镇，朝着十公里外的敌占区前行而去。
<br/>　　就跟情报中所说明的那样，这个镇子在被反政府武装势力控制之后，里面的老弱妇孺或是被杀或是被抓之后，这个原本有着数千人口的小镇，彻底变成了一座死镇。
<br/>　　即便是知道了这座小镇上已经成了一座死镇，但是行走在这里的时候，罗昊、慕容飞、李毅峰、李然四个人还是不敢放松警惕，因为无法排除这里不会再出现反政府武装势力的巡逻队。
<br/>　　“我们在哪里跟蛰龙汇合？”
<br/>　　慕容飞眼睛盯着ACE-GAP39SB短突击步枪战术导轨上的全息瞄准镜的同时，眼角的余光注意着四周的情况。
<br/>　　“没有汇合点。”罗昊回答道：“他会主动来找我们。”
<br/>　　“他怎么知道我们在什么地方？”李毅峰对罗昊的回答表示怀疑。
<br/>　　因为到现在为止，他们也只是从宋萱的口中得知蛰龙会带着他的小队配合行动，但是他们并没有跟蛰龙进行过联系。在没有联系的情况下，蛰龙和他的小队能找到自己这边四个人吗？
<br/>　　不仅是李毅峰对罗昊的回答感到怀疑，就连慕容飞也同样是感到怀疑。唯一确信蛰龙一定能找到自己这边四个人的，就罗昊和李然。因为他们都是从红旗学校出来的，他们了解蛰龙，慕容飞虽然比李毅峰要早加入龙隐部队，但是却也并没有跟蛰龙接触过。
<br/>　　蛰龙虽然是龙隐部队的队员，但是他和他的小队却是独立在基地外的行动小队，他们只接受龙隐部队最高指挥官的命令，以前是野狼，现在是承影。
<br/>　　“这个你不需要怀疑，他自然有他自己的办法。”李然轻笑了一下，说道：“如果他不愿意出现的话，我们是没办法找到他的……，嘘！”
<br/>　　李然竖起左手，示意其他三个人停止前进。李然对着罗昊使了个眼色，罗昊会意的点了点头，示意所有人立刻分散隐蔽。
<br/>　　四个人立即朝着街道两边的建筑分散隐藏起来，在四个人隐蔽起来后不久，就有一队反政府武装势力的巡逻队从街道上经过。
<br/>　　巡逻队一共五个人，搭乘一辆加装了重机枪的皮卡车。
<br/>　　罗昊靠在墙壁背后，按住耳麦轻声说道：“不要惊动他们，让他们通过。”
<br/>　　“明白！”
<br/>　　“收到！”
<br/>　　“明确！”
<br/>　　在能不跟对方发生交火的情况下，对反政府武装势力的巡逻队采取规避是第一选择。
<br/>　　可是，有时候，武力接触不是想要规避就能规避得了的。那辆架设了重机枪的皮卡车在街道上停了下来，皮卡车上的五个反政府武装人员也是从皮卡车上跳了下来，嘴里不满的嘀咕着什么，似乎是在这次巡逻任务有所不满。
<br/>　　其中一个反政府武装人员拿出一包烟，自己抽了一支出来，叼在嘴里，随后把烟扔给身边的同伴，用一个不知道从哪里弄来，已经掉了漆的打火机，把烟点燃，靠着皮卡车默默地抽着，短时间内似乎不准备离开。
<br/>　　“狂龙，我们现在要怎么办？”慕容飞按住耳麦，通过无线电向罗昊问道。
