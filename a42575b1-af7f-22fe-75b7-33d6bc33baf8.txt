　　听到SHADOW的话，chen原本正在填写支票的手颤抖了一下，就如同看外星人一样看着坐在自己对面的SHADOW。
<br/>　　chen放下自己手中的圆珠笔，推了推自己鼻梁上那副向下滑落的重度近视眼镜，看着SHADOW，轻笑道：“马丁先生，您是在跟我开玩笑吗？一组代码，而且还是并不完全的代码，您开口就要五十万美金，您不觉得太过分了吗？我自己也是做代码的，对这个市场还是了解的，五十万新台币来收购您手里这组代码，已经是很高的价格了。看来马丁先生并不如我想象中那么有诚意，来完成这笔交易，既然这样，那么再见吧。”
<br/>　　“不要着急。”SHADOW对着chen压了压手，阻止了他要离开的举动，说道：“如果我没有诚意，就不会千里迢迢来这里跟你见面了。”
<br/>　　SHADOW拿起咖啡杯轻轻呷了一口，用小勺子轻轻搅拌着杯子中的咖啡，抬眼看着脸上带着怒色的chen，说道：“你说得没错，但是你说得价格只针对于一般的代码，如果我手里这组代码来自NSA的黑客病毒实验室呢？”
<br/>　　“EG？”
<br/>　　chen的脸上不再有怒色，相反变得有些凝重。NSA下属的“方程式小组”（EG）在黑客界的名气很大，世界上目前已知的几种有着巨大破坏性的电脑病毒都跟EG有关系，甚至有传言，这些病毒就是EG研发出来，投入到网络中进行试验的。
<br/>　　对于EG的黑客病毒实验室，chen也是有所听闻，那里有着海量的电脑病毒代码，有些已经完成的，有些是还没有完成的，有些是已知电脑病毒的变种，有些是世界上还未知的电脑病毒，那里对于每一个黑客来说就是一座巨大的宝藏库，但是因为EG隶属于NSA，所以没人敢去老虎身上拔毛。
<br/>　　chen坐在椅子上，沉默了很长时间，就连服务员把咖啡端上来他都没有发觉，直到他鼻梁上的那副近视眼镜又快要滑落下来，chen才是推了推眼镜，抬头看着SHADOW，严肃的问道：“马丁先生，您说的是真的？”
<br/>　　chen指了指SHADOW放在桌子上的U盘，问道：“这里的东西真的是来自EG的黑客病毒实验室？”
<br/>　　“如假包换！
<br/>　　SHADOW显得非常自信，从一开始他就很自信，仿佛他料定了chen一定会购买他手中的代码一样。
<br/>　　chen推了推眼镜，看着SHADOW放在桌子上的U盘，说实话，chen此时已经心动了。如果U盘里的东西真是从EG黑客病毒实验室里弄出来的，那么其中的价值就不言而喻了，chen甚至已经开始在幻想着，自己凭借这组代码，再次成为那个被无数人仰视的“电脑鬼才”时的情形了。
<br/>　　犹豫了一下后，chen对SHADOW说道：“我没有那么大的权利动用那么大一笔资金，我需要向公司请示，必须要得到他们的同意。”
<br/>　　“我理解，我可以给你时间，但是时间不会太多。”SHADOW摊了摊手说道：“你也知道，我这样的人是不会在一个地方待很长时间的，因为那样我就会很麻烦，我不想被警察找上门。”
<br/>　　“我知道，我曾经也是做这个的，当年如果不是有保护，我可能已经被抓去坐牢了。”chen有些为难的对SHADOW说道：“虽然我个人是愿意相信您的，但是您也知道，要想说服公司上面的人并不是一件容易的事情，仅仅凭我口头说，就算我说得在怎么天花乱坠，他们也未必会相信，所以……”
<br/>　　“所以，你想让我把代码给你，让你拿回公司？”SHADOW斜睨了chen一眼，问道。
<br/>　　chen咽了一口口水，最后还是点了点头，说道：“我知道，这样可能会让您很为难，但是希望您能理解。如果您不放心的话，我可以支付给您五十万新台币作为定金，如果那组代码真如您说的那么神奇，我立刻让公司给您结算尾款。五十万新台币，是我能动用资金的最大额度。”
<br/>　　SHADOW拿起桌子上的U盘，扔给chen，说道：“这个U盘我做了设置，你可以有一次访问权限，随后它就会被锁定，需要我的密码才能再次打开。不要试图去复制里面的代码，如果你这么做了，就会启动里面的删除程序，把U盘格式化，到最后，你们花了五十万新台币将什么东西都得不到。”
<br/>　　“谢谢您的理解。”chen接过U盘，小心的收好，拿起桌子上那杯已经变凉的咖啡，一饮而尽，随后起身向SHADOW告辞。
<br/>　　SHADOW目送着chen离开，随后从钱包中取出几张纸币放在桌子上，用水杯压好，收起自己的笔记本电脑，起身离开。至于那张写了女服务员电话号码的纸巾还被SHADOW仍在桌子上，从一开始，SHADOW就没有正眼去看过那张纸巾，不是他不知道，而是他不感兴趣。
<br/>　　……
<br/>　　chen拿到U盘之后，直接拦了辆出租车，报了公司大楼的地址，直奔公司大楼而去。
<br/>　　回到公司，chen搭电梯上了顶楼，推门进入了总经理办公室。chen进入办公室的时候，经理的咸猪手正放在年轻女秘书的屁股上，看到chen，经理立刻把手收了回来，然后对着年轻女秘书挥了挥手，示意她先离开。
<br/>　　经理脸色看到的盯着chen，问道：“东西带回来了？”
<br/>　　chen把U盘放在经理办公桌上，说道：“马丁先生对我们的价格并不满意，他需要五十万美金才愿意交易这组代码。”
<br/>　　“五十万美金！？”经理从椅子上弹起来，眼睛瞪得滚圆，恨不得要把chen给瞪死，说道：“五十万美金，就这么一组代码，他要五十万美金，他怎么不直接去抢银行？！”
<br/>　　“马丁先生说，这组代码是来自NSA隶属的‘方程式小组’的黑客病毒实验室，之前从未出现过。”chen推了推眼镜，解释道：“EG黑客病毒实验室里出来的东西，的确值这个价格，如果拿到黑市上去卖，价格只会比马丁先生开出的价格更高。”
<br/>　　“可是怎么能证明，那个马丁先生不是在编造谎话，欺骗我们呢？”经理皱着眉，疑惑道。
