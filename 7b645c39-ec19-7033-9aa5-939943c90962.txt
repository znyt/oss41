夏妍妍回到了房间里，她打开了抽屉，拿出了一块玉佩。
<br/>
<br/>        这块玉佩是六七年前陆寒霆送给夏夕绾的，不过被她趁机偷了过来。
<br/>
<br/>        夏妍妍的记忆回到了大雪纷飞的那一天，夏夕绾在冰天雪地里救下了奄奄一息的陆寒霆，并将他带到了一个山洞里依偎共眠了一晚，第二日清晨在山洞外的不止苏希和夏小蝶，她也在场。
<br/>
<br/>        她亲眼看着苏希愤然离去，夏小蝶追了过去，后来夏夕绾走出了山洞去找援兵了，当时就她在洞外。
<br/>
<br/>        夏夕绾不会知道她前脚走，陆寒霆的手下后脚就找来了，那些手下训练有素，态度恭敬的将陆寒霆带上了一辆加长版的商务豪车，当时她看了一下车牌，那是从帝都来的车，光车牌就是吊炸天的那种。
<br/>
<br/>        夏妍妍当时心跳的厉害，她已经猜到夏夕绾无意之中救下了一个大人物。
<br/>
<br/>        陆寒霆走的时候将随身玉佩赠给了夏夕绾，说很快就会来找她的，夏夕绾将这块玉佩放在了抽屉里锁上，夏妍妍就找了一个机会将夏夕绾给支开了，她将这块玉佩偷了出来，占为己有。
<br/>
<br/>        她一直在等待陆寒霆来找她，当某一天，一辆挂着帝都车牌的劳斯莱斯幻影豪车再次来到乡下时，她主动出现了，将玉佩交给陆寒霆，说她救了他。
<br/>
<br/>        后来陆寒霆答应了她三件事，让她任意提要求。
<br/>
<br/>        这一切，夏夕绾全然不知。
<br/>
<br/>        到现在夏夕绾还不知道当年她救下的是陆寒霆，陆寒霆也不知道这些年他念念不忘的那个救命女孩是夏夕绾。
<br/>
<br/>        但是，缘分兜兜转转，仿佛一切都在命中注定，谁知道一场精心策划的替嫁让夏夕绾从乡下回来，嫁给了陆寒霆，成为了他的陆太太。
<br/>
<br/>        夏妍妍将这块玉佩用力的拽在了手心里，她知道自己并没有输，她还是陆寒霆的救命恩人。
<br/>
<br/>        当年她曾经跟陆寒霆约定过，不能将救他的这件事情告诉任何人，她不想任何人知道这件事。
<br/>
<br/>        所以夏妍妍敢肯定陆寒霆是不会将当年那段救人的事情告诉夏夕绾的，他和夏夕绾无法相认，那她的这个谎言永远就不会被揭穿。
<br/>
<br/>        夏妍妍又想起他在电话里说这些年他从来没有忘记过那一夜，在宠爱夏夕绾的同时他心里也为当年那个女孩保留了一份柔软。
<br/>
<br/>        一个夏夕绾，竟然让他爱了两次。
<br/>
<br/>        不过，那个女孩现在是她！
<br/>
<br/>        夏妍妍知道自己在依仗什么，只要她还握着这块玉佩，陆寒霆就不会真的弃她于不顾，她还有机会！
<br/>
<br/>        这时“叩叩”的敲门声响起了，李玉兰推门而入，她手上端了一杯温热的牛奶，殷勤讨好的看着夏妍妍，“妍妍，你是不是在怪妈，妈真的不知情啊，这个世界上妈妈是最爱你的人啊。”
<br/>
<br/>        夏妍妍将玉佩放好了，然后扭头看向李玉兰，“妈，我相信你是最爱我的，现在我需要你为我做一件事。”
<br/>
<br/>        “什么事？”李玉兰问。
<br/>
<br/>        夏妍妍拿出了一把锋利的刀，放在了李玉兰的面前，“妈，你割腕吧。”
