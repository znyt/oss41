　　一天后，萧逸横跨三郡，直接去了东来郡。
<br/>　　当然，第一时间，便是开始调查。
<br/>　　然而，仅仅三天后，萧逸才刚刚灭掉一个势力，毁掉里头炼制血意丹的基石。
<br/>　　刚欲对付下一个势力时。
<br/>　　一个消息却忽然传出。
<br/>　　消息一出，整个东边六郡震荡。
<br/>　　连萧逸也感到愕然，只得暂时停下追查。
<br/>　　消息便是，千雄教与星盟两大霸主势力，将在两天后，彻底大战，不死不休。
<br/>　　挑起战斗的，是千雄教。
<br/>　　两大势力，虽然多年来一直不对付，一直明争暗斗。
<br/>　　但，像这次这般，直接开战，还是头一次。
<br/>　　而究其原因，竟是这两月来，星盟某位强者大肆在东边六郡杀人。
<br/>　　大肆对其他势力灭门。
<br/>　　如此‘残忍’、‘灭绝人性’的过分之举，终于让得千雄教忍无可忍。
<br/>　　甚至不惜就此对星盟展开大战。
<br/>　　此事一出，最近风头极盛，那位神秘的星盟强者，一下子成了众矢之的。
<br/>　　那位神秘的星盟强者，自然是一直被误认的萧逸。
<br/>　　而此刻的萧逸，也是满脸愕然。
<br/>　　愕然之余，他更多的是心头冰冷。
<br/>　　千雄教与星盟的纷争，由来已久。
<br/>　　两方可谓知根知底。
<br/>　　别人也就罢了，千雄教不可能猜不出这两月来之事，并非星盟强者所为。
<br/>　　可千雄教还是将开战的缘由，扣在了萧逸这位一直被误认的星盟强者头上。
<br/>　　很有可能，千雄教早就想开战。
<br/>　　这一次，只是随便拿萧逸之事，当个借口罢了。
<br/>　　当然，无论如何，是否真如自己猜测的那般。
<br/>　　萧逸都打算去看个究竟。
<br/>　　……
<br/>　　两天后。
<br/>　　东来郡，千雄教千里之外的某处荒郊森林。
<br/>　　那里便是两大势力越好的交战之地。
<br/>　　此时，两方强者，早已到来。
<br/>　　密密麻麻，两方都不下数千之人。
<br/>　　两个势力间的大规模战斗，计谋显得相当无力。
<br/>　　两方的武者，也不打算用什么计谋。
<br/>　　真正的实力，才是这场大战胜负的决定因素。
<br/>　　周围，是闻讯赶来观战的其他势力之人。
<br/>　　密密麻麻，在数千之数。
<br/>　　恐怕，东来郡以及周边各郡的势力，和一些名气颇大的独行武者等等，全都来了。
<br/>　　当然，为免被战斗波及，观战的人都离得远远的。
<br/>　　萧逸自然也在其中，只不过相当不起眼就是了。
<br/>　　入目所视，两方近万人的实力，尽在他的感知当中。
<br/>　　千雄教这边，为首的，是一个老者。
<br/>　　那便是千雄教教主，更在这东边六郡被誉为千雄之主。
<br/>　　实力乃天元一重。
<br/>　　手下，足有数百位地元境八重武者。
<br/>　　这样的数量，虽没有传言中的千位枭雄之数，但已经相当骇人。
<br/>　　其余的数千教内武者，则是良莠不齐。
<br/>　　几乎都在洞玄境至破玄境这个范畴。
<br/>　　而星盟这边，则没有天元境。
<br/>　　为首的自然是徐星，但真正的最强站立者，应当是身旁的五位老者。
<br/>　　五人，皆是地元九重，面前可算半步天元。
<br/>　　而地元八重修为的，大概在两百位左右。
<br/>　　而其他的普通星盟武者，倒是较之千雄教更厉害，清一色的破玄境修为。
<br/>　　“不愧是东边六郡的两大霸主势力。”萧逸淡淡自语一声。
<br/>　　当然了，那些地元境武者，看起来数量极多。
<br/>　　但，两方势力，可是遍布六郡。
<br/>　　手下的地元境武者构成，也来源于六郡。
<br/>　　一旁，某位观战的武者说道，“可不是嘛。”
<br/>　　“两大势力开战，震动六郡。”
<br/>　　“此战，无论胜负如何，都将造成极严重的后果。”
<br/>　　身旁，不少观战武者开始议论纷纷。
<br/>　　“哎，这都怪这两月来星盟那位神秘强者太过嚣张了。”
<br/>　　“四处杀人，四处灭门，难怪千雄教要替天行道。”
<br/>　　在六郡武者眼中，可不知道那些被灭的势力与千雄教有关。
<br/>　　只以为是各地的势力被灭。
<br/>　　千雄教看不过眼，才彻底出手。
<br/>　　“不过，星盟这些年来作恶多端，被灭了才好。”
<br/>　　“千雄教，实力浑厚，必能解决这一群恶人。”
<br/>　　一旁的萧逸淡淡道，“不一定吧。”
<br/>　　“谁是坏人，谁是好人，可不能从表面上来认定。”
<br/>　　“而且，就如今两方展现出来的武者力量，在伯仲之间。”
<br/>　　“胜负，可不好说。”
<br/>　　“切，你懂个屁。”周围之人不屑说道。
<br/>　　“星盟中人，人人得而诛之。”
<br/>　　“千雄教若是不敌，我等便上去帮忙。”
<br/>　　“这一次，必须好好教训星盟的混蛋。”
<br/>　　萧逸闻言，皱了皱眉头，没有再多说什么。
<br/>　　千雄教在东边六郡，传承百年以上，一直风评极好。
<br/>　　但暗地里的龌蹉，却无人知晓。
<br/>　　而他今日来观战，仅仅是观战。
<br/>　　并不打算插手两方的战斗。
<br/>　　不多时，数千米之外的两方，似在谈判，随后谈判忽然破裂。
<br/>　　两方武者，瞬间出手。
<br/>　　规模浩大的战斗，也瞬间打响。
<br/>　　星盟这边，五位半步天元，瞬间缠上了千雄教主。
<br/>　　两边的地元境，也瞬间交锋。
<br/>　　近五百位地元八重的战斗，威势还是相当骇人的。
<br/>　　原本，星盟足足差了千雄教近百位地元八重，应当落入下风才对。
<br/>　　可是，徐星的出手，却立刻弥补了差距。
<br/>　　在萧逸的感知中，此刻的徐星，修为达到地元六重。
<br/>　　也不知是服了什么丹药，还是用了什么秘法，强行提升了一重修为。
<br/>　　同时，徐星身上，一道诡异的武魂虚影凝聚着。
<br/>　　那是一条巨大的双头蛇虚影。
<br/>　　“双生蛇武魂。”萧逸微微吃惊。
<br/>　　双生蛇，亦称双头蛇，位列蓝色阶品。
<br/>　　且论稀有程度，丝毫不亚于星轮羽衣这种稀有武魂。
<br/>　　那两个硕大的蛇头，各有功效。
<br/>　　一为妖蛇幻音，有摄人心魄之效。
<br/>　　若实力低下者，仅仅听一丝妖蛇幻音，恐怕就会肝胆俱裂而亡。
<br/>　　另一蛇头，称之为五毒蛇头。
<br/>　　操控着五种阴寒剧毒，威力相当恐怖。
<br/>　　两者结合，若单独打斗下，徐星绝对可以碾压一切地元八重武者。
<br/>　　而此时，在他手上，正有一笛子。
<br/>　　笛子，一看便知极为不凡。
<br/>　　萧逸直接便看出，这是下品元器。
<br/>　　徐星通过这笛子，将妖蛇幻音，大大增幅。
<br/>　　那如同魔音一般的声音，顷刻间覆盖全场。
<br/>　　千雄教那边的地元八重武者，因此纷纷实力大减。
<br/>　　“厉害，不愧是星盟门主。”萧逸淡淡地说了一句。
<br/>　　另一边，两方普通武者间的战斗。
<br/>　　星盟清一色的破玄境武者，足以碾压千雄教良莠不齐的武者。
<br/>　　若是照这个情况下去。
<br/>　　星盟这边，在徐星的带领下，赢得这场大战，只是时间的问题。
<br/>　　周围之人，再次议论纷纷。
<br/>　　“千雄教落入下风了，该不会败给星盟吧？”
<br/>　　“肯定不会，我可听说了，千雄教好像请来了某位帮手。”
<br/>　　“这也是千雄教忽然宣战，且极有把握的原因。”
<br/>　　周遭的议论声刚刚落下。
<br/>　　远处的战斗中，忽然，一道蓝色流光，疾速而来。
<br/>　　同时，点点星光，伴随着凌厉幻影，轰然落下。
<br/>　　一抹骇然的剑影，一闪而过。
<br/>　　剑出，影落，星光涌动。
<br/>　　一直在大战中极为耀眼的徐星，瞬间被轰飞。
<br/>　　“好强的实力，好绚烂的一剑，那是何人？”观战之人，纷纷一惊。
<br/>　　远处，徐星被轰飞后，原地出现了一年轻人。
<br/>　　年轻人手持长剑，直直指着徐星，战意滔天。
<br/>　　其他人认不得刚才那一剑。
<br/>　　萧逸却认得。
<br/>　　“星影剑。”萧逸双眼一眯，“凌羽怎会来此。”
