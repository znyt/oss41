　　第418章
<br/>　　盘龙八打枪，顾名思义有八种打法，抽砸挑拨、进环收落，有青龙甩尾、黑龙拍水，白云入龙等八招。
<br/>　　八招打法各有特点，繁杂高深，同时要求使用者有极大的气力。简单来说，就是不仅要四肢发达，同样也需要头脑发达。
<br/>　　帝国那位开国皇帝，号称天下武尊，当然，这是他自己号称的。人一旦当了皇帝，便将什么好事都往自己身上揽，他还称自己：至尊明圣德武文极……
<br/>　　全称共有二十四个字，每年祭天的时候，明帝都要背好几天才背得下来。
<br/>　　在那些他的对手口中，他常常被称作为『李蛮子』。
<br/>　　蛮这个字首先说明他不怎么讲道理，另外也说明一点，这个人力气很大。
<br/>　　事实上，李蛮子不仅力气大，脑子也很好用，盘龙枪在他手中能发挥出最大威力。然而，一百年的时光过去，李家再没出现过如李蛮子这等天赋的人。在今天，明帝是李家唯一掌握盘龙枪八种打法的人，至于其他人，大多只能掌握一两种。
<br/>　　今天是李家枪和林家枪的对决。
<br/>　　林少羽知道自己小看了明帝，没料到这个老者还有如此的战斗力。但也仅此而已，他毕竟老了，曾经或许很强，但这个世界已不是他的舞台。
<br/>　　林少羽进步抢攻，双手握枪，左臂发力，枪去叼明帝的手腕。
<br/>　　铛！
<br/>　　明帝的枪轻轻砸在林少羽的枪杆上，四两拨千斤，林少羽一身力气被拨到别处。
<br/>　　林少羽一怔，收力不及，往前进了几步。忙稳住身子，却见明帝依旧摆出防御的姿态，倒是像头蹲坐在地上的蛤蟆，安坐不动，只等昆虫经过的时候，才吐出长舌。
<br/>　　那一双眼睛藏在堆满皱纹的脸中，眸底清澈，瞳孔聚拢，眼神仿佛再说：
<br/>　　纵然是你年轻，但真正的战斗不是你这种黄毛小儿懂的。
<br/>　　林少羽深呼吸，平静心情，同样寻找着明帝的弱点。随即忽然向前跳了一步，逼近明帝，一枪刺向明帝的膝盖。
<br/>　　比反应和速度明帝绝对比不过林少羽。
<br/>　　铛！
<br/>　　明帝又砸出一枪，顺着林少羽的枪势将他挑到一边，而趁这个时间，明帝与林少羽拉开了距离。
<br/>　　这下林少羽大感匪夷所思，同时心头震惊。他小看了明帝，明帝比他预估的还要强，必须重新判断明帝的实力。他或许，真的有能和自己匹敌的实力，甚至有机会胜过自己。
<br/>　　无论自己怎样攻击，明帝都能顺着枪势挑开，他双手握枪，一前一后，驼背弓腰，只等着林少羽出手。
<br/>　　盘龙枪第七打：画龙点睛。
<br/>　　所谓画龙点睛，不在攻，在于守，不在动，在于静，讲究以静制动。要诀不在出手的速度，在于眼力。必须在对方出手之前，便判断出对方的弱点，后发先至，攻击对方出手时的软肋。
<br/>　　当初的李蛮子，掌握八种打法，但画龙点睛却是最弱的，因为他的性情也是大开大合，这种小巧走心功夫实在和他的性情不符。
<br/>　　同样，画龙点睛也不是明帝最强的打法，但无奈，他年迈力衰，其他如青龙甩尾，黑龙拍水这样势沉力猛的打法已经用不出来。
<br/>　　林少羽的确有些心浮气躁，他已连攻七枪，枪枪被明帝挡住。过程中，他甚至卖了个破绽，想要引明帝出手。
<br/>　　明帝没有上当。
<br/>　　越是挫败，便是越是激起林少羽的战意。他一定要凭速度破掉明帝的画龙点睛。
<br/>　　但他忘了一件事，画龙点睛讲究的不是速度，而是眼力预判。明帝懂盘龙枪，也同样懂林家枪，少年时，他便和林问天一起切磋，即便登基称帝，诸事繁忙，两个老友也经常聚在一起切磋枪法。
<br/>　　月静花香，对月舞枪，不分君臣，只谈往事，却是人生一件快事。
<br/>　　此时此刻与彼时彼刻，何其相似，只不过，与自己对枪的不再是林问天，而是林少羽。今日也不是老友切磋，而是生死相争。
<br/>　　以生死争天下。
<br/>　　林少羽眉头皱紧，眼角滴落一滴冷汗，他不得不承认，此刻明帝是没有破绽的。
<br/>　　难道这画龙点睛真的无法攻破么？
<br/>　　恍惚间，林少羽忆起了当初，似乎也见过这一幕。那时他还很小，很小很小，大概只有六七岁左右。一日黄昏，明帝和自己父亲在花园对枪，明帝用画龙点睛，自己父亲用林家枪攻。
<br/>　　那日傍晚，父亲使出浑身解数，却攻不破明帝的枪围。自己不服气，在明帝走后还问过父亲：难道我们林家的破浪枪，真的胜不过李家的盘龙枪么。
<br/>　　记得本来还垂头丧气的父亲却是抚须大笑，对自己道：臣子怎么能胜过君王，有些事能赢也不要赢，至于画龙点睛……哈哈，破之不难……
<br/>　　记得当时年纪小，父亲的话听不懂，为何能赢却不要赢。但关于画龙点睛的破法，自己是记得的。
<br/>　　想到这里，林少羽泪如雨下。画龙点睛并非不可破，关于破法父亲曾经亲口告诉过自己，而自己，还记得。
<br/>　　『老爹，你的仇今日报了』
<br/>　　林少羽仰面向天，任泪水涌入口中，淡淡咸咸的味道，恍惚又回到曾经。
<br/>　　那个人已不在了，但他的话还留在心底。几年十几年没有感觉，觉得他的话没有任何用，然而有朝一日，你终会想起他的话。今日，自己便用他教的法子为他复仇。
<br/>　　林少羽一枪递出，横平竖直，堂堂正正，破浪枪中最普通的一招，同样也是最强的一招。
<br/>　　破浪七叠。
<br/>　　将全身力气灌输在一点，把七道劲凝成一道劲，在枪尖绽放。
<br/>　　这并不是太复杂的要诀，天下一百种枪法，大约九十九种都写着这句话。然而知道是一回事，能够做到是另外一回事，恰好，林少羽能做到。
<br/>　　七枪化一，一枪捅出，捅穿天地。
<br/>　　明帝用枪杆撞击对方的枪杆，想要借势挑开，画龙点睛，的确有四两拨千斤的能力。但，如果对方的力气不是千斤，而是千钧万斤呢。
<br/>　　四两便也就拨不动了。
<br/>　　摧枯拉朽，穿透层层阻击，这一枪，捅进了心脏。
<br/>　　
<br/>(本章完)
