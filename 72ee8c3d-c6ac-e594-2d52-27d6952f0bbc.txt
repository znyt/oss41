　　“那我们怎么办？”吉尔有些慌。
<br/>　　保护伞能调集五十名全副武装突击队员，还能偷偷派出十多人潜入蜂巢最深处。说不定那一天，自己等人在睡梦中就被人杀死！

<br/>　　“这次突击队直接潜入实验室，下一次不知道会是什么攻击，不能把希望寄托于保护伞高层的仁慈。”苏重看着众人道：“这里虽然安逸，但并不安全。”
<br/>　　“那我们去哪？外面到处都是丧尸。”雷恩皱眉问道。
<br/>　　“保护伞是有很多分部，只要控制这些分部，保护伞总部鞭长莫及，就不会发生这种偷偷潜入的事情。”爱丽丝笑道。
<br/>　　她已经明白了苏重的打算。
<br/>　　分部基地远离危险，设备资源齐全，能保证他们生存，也能给苏重研究提供基础。
<br/>　　“我们全都经过强化，再准备一辆坚固的装甲卡车，到达另一座基地并不困难。”马特眼睛一亮道。
<br/>　　几人闻言不由振奋，刚才他们面对十倍于己敌人，竟轻松把地方放到！更强的力量，更快的速度，更迅捷的神经反应，训练有素的突击队员，根本就不是他们的一合之敌！那些满大街的丧尸，还真奈何不了他们！
<br/>　　“去收拾吧，我也要做些准备。”苏重打发走众人，开始准备搬迁。
<br/>　　“实验数据不用带走，都在脑子里。”苏重扫了一圈：“制作一些抑制剂和强化剂，作为储备药品就好，保证路上安全。”
<br/>　　抑制剂和强化剂全都是他开发，制作起来驾轻就熟，很快就准备好了三十支抑制剂、五支一型强化剂、还有一支二型强化剂。
<br/>　　“要走也不能让保护伞好过。”苏重心里想着：“红后，我把你带走怎么样。”
<br/>　　苏重一边把药剂放入冷藏箱内，一边和红后闲聊。
<br/>　　“先生，我不得不提醒您，蜂巢基地的主板只是我的一个子程序。我的本体一直在地下深处。”红后遗憾道。
<br/>　　“没什么，带走你的子程序也好。”苏重不在意。
<br/>　　他打定主意控制保护伞分基地，有一个人工智能掌控全局，苏重会轻松很多。
<br/>　　沉默了一会儿红后才道：“谢谢，先生。可惜我已经被保护伞公司限制，不能给您提供更多帮助。病毒资料也被封锁，不能再给您更多实验数据。”
<br/>　　“没事，病毒我自己研究，不用他们的数据。现在如果还给我发数据，我还怕他们误导我呢。”苏重不在意。
<br/>　　他心头一动突然问道：“保护伞封锁了病毒数据，那有没有封锁其他数据？”
<br/>　　“您想要什么资料。”红后小脸一笑，格外大气。
<br/>　　“嘿，你把保护伞钻地的本事告诉我吧。对了，还有地热发电技术，如果有你的核心程序，就更好了！”
<br/>　　生化病毒被苏重基本研究透彻。苏重看上了保护伞钻洞的本事。
<br/>　　生化危机系列中，各个国家到处都有保护伞的地下基地。而且一个个的设施完备，空间巨大，地下建造技术显然非常成熟。
<br/>　　“先生，蜂巢基地设计施工方案已经打包存放入主板。”红后动作非常快：“核心程序无法给您，只能把给您一套简易子程序。”
<br/>　　“干的漂亮！”苏重只是突发奇想，没想到还真有收获。
<br/>　　“谢谢夸奖。”红后高兴的笑了起来，看着苏重继续忙活，不由问道：“您在制作什么仪器？需要我的帮助吗？”
<br/>　　“谢谢，不过这是我的独门秘方，是给保护伞准备的惊喜。”苏重手里不停。
<br/>　　他在制作吝啬蔷薇，临走了，苏重准备给保护伞玩个大的！
<br/>　　……
<br/>　　蜂巢地下，保护伞总部。
<br/>　　艾萨克从沉睡中苏醒，一脸不满的看着威斯克：“不是告诉你不要随便唤醒我吗？”
<br/>　　他是保护伞第二股东，生化末世的发起人，威斯克的顶头上司。
<br/>　　之前一直躲在保护伞总部休眠，等待生化末世过去，变相延长寿命。
<br/>　　“出了什么问题。”艾萨克压下怒气闻道，威斯克是他得力手下，被迫将他唤醒，肯定是发生了计划之外的事情。
