        童筱颖不假思索地回答：“我留在这里等你。”
<br/>
<br/>        “好。”傅夜沉微微一笑。
<br/>
<br/>        晚饭的时候，爷爷傅炎廷已经见过童筱颖了，和以前一样，爷爷对童筱颖的态度是不冷不热。
<br/>
<br/>        晚宴散席后，童筱颖和傅夜沉走在回屋的长廊上，她忍不住问他：“爷爷是不是不喜欢我？”
<br/>
<br/>        “爷爷是那性格，你不要太在意。”傅夜沉说话的语气很轻。
<br/>
<br/>        最近，他跟她说得最多的一句话，只怕就是“不要太在意”任何事情了。
<br/>
<br/>        不过，童筱颖自己倒是也没往心里去，反而更在意起他来：“你去滨江城要多久才会回？”
<br/>
<br/>        “明天晚上就会回来。”傅夜沉回答道。
<br/>
<br/>        起风的时候，风过长廊，带来了一丝寒意，长廊外的荷塘里泛起了涟漪，月光流淌而过。
<br/>
<br/>        傅夜沉连忙脱下了自己的外套，披在了童筱颖的肩上。
<br/>
<br/>        这段时间，和她相处下来，他觉得很轻松。
<br/>
<br/>        甚至在不知不觉中，他竟然不知道自己已经没有了烟瘾，整个人看上去也精神了许多。
<br/>
<br/>        童筱颖此时心里一暖，索性一头扎入傅夜沉的怀中，双手环上他的腰畔，脑袋靠在他的心口上：“那我也给你暖一暖。”
<br/>
<br/>        傅夜沉欣慰地笑了笑，很自然地搂住童筱颖的肩头，紧紧地拥着她一起回了房间。
<br/>
<br/>        自从将她接回自己的身边后，他并没有跟她行夫妻之礼，而且两人还是分床睡。
<br/>
<br/>        他知道，她现在只是依赖他，但是还没有到爱上他的地步。
<br/>
<br/>        他们的房间有内屋和外屋，晚上，童筱颖睡内屋的大床上，他傅夜沉则睡在外屋的卧榻上。
<br/>
<br/>        到夜深人静时，傅夜沉侧躺着，虽然是闭着眼睛的，但是大脑的思维却很活跃。
<br/>
<br/>        没过多久，他便听到了筱颖传来的细微的呼吸声。
<br/>
<br/>        月光从透过纸窗，朦胧了屋子里的一切。
<br/>
<br/>        傅夜沉因为睡不着，干脆坐起身来，点漆的眸子无意间瞄向了内屋。
<br/>
<br/>        透过珠帘，一大半的被子悬挂在大床边缘。
<br/>
<br/>        傅夜沉见状，从卧榻上下来，撩开珠帘，轻轻地走到内屋。
<br/>
<br/>        傅家大庄园是座古宅，所以屋内的陈设和古时候的王府差不多。
<br/>
<br/>        地面铺着地毯，走上去一点声音也没有。
<br/>
<br/>        他将被子拉起，然后又轻轻地替童筱颖盖好，顺道将这小女人的那只脚也塞到了被子下。
<br/>
<br/>        童筱颖是平躺着的，两只手摆成了一个“大”字，睡姿很随意。
<br/>
<br/>        月光柔柔的洒在她脸上，让她的皮肤看起来格外白皙，长卷的睫毛在眼睑上投下了一层阴影。
<br/>
<br/>        她瘦了一圈，养了这么久，都没有养胖。
<br/>
<br/>        傅夜沉难以想象，她被人拿去做实验时，会是个什么场景。如果，他看到了，一定会将连华生他父亲的那个实验室给一锅端了。
<br/>
<br/>        “大哥……”童筱颖忽然一声呓语，翻了个身。
<br/>
<br/>        傅夜沉身心一怔。
<br/>
<br/>        不知道她的梦里，他做了什么事，能让她如此惦记着。
<br/>
<br/>        “嗯——大哥……我想吃……核桃酥。”童筱颖蠕动着嘴唇，呢喃自语。
<br/>
<br/>        傅夜沉忍不住笑了笑，俯身吻了吻童筱颖的额头，应了声：“好。”
<br/>
<br/>        翌日。
<br/>
<br/>        童筱颖一觉睡到自然醒，虽然没有日上竿头，但也差不多了。
<br/>
<br/>        此时的屋内，早已没有了傅夜沉的身影。
<br/>
<br/>        童筱颖洗漱完，走出内屋，看到小厅堂中间的圆桌上摆满了各种点心，于是好奇地凑了过去。
<br/>
<br/>        除了一些名贵又好看的西式点心，最重要的一点是还有她昨晚梦里的核桃酥，再配上一杯散发着清香的茉莉花茶。
<br/>
<br/>        她还真的没想到，自己可以梦想成真。
<br/>
<br/>        童筱颖嘴馋，拿起一块核桃酥，咬了一大口。
<br/>
<br/>        好好吃！
<br/>
<br/>        她刚吃过点心，就有女佣捧着一套晚礼服走了进来，向她微微欠身，说道：“大少奶奶，夫人说，您用过早点之后，得陪她一起出席时尚明星慈善拍卖宴。”
<br/>
<br/>        “噢，好。”童筱颖应了声，让女佣将晚礼服放下便好。
<br/>
<br/>        临海城是省都，所以娱乐圈里大大小小的活动，差不多都会在省都举办。
<br/>
<br/>        傅氏集团作为这次慈善拍卖宴的赞助商，冷夫人自然也就成为了晚宴的特邀上座嘉宾。
<br/>
<br/>        这次慈善拍卖晚宴，虽然是娱乐圈里各位大明星的“走秀”盛宴，但也是各大豪门贵妇斗富争艳的地方。
<br/>
<br/>        再换句不好听的话，就是有钱砸钱，没钱抛头露面蹭人气的盛宴。
<br/>
<br/>        冷夫人之所以带上童筱颖，其实是傅夜沉的嘱咐，不过她也是想让自己的这个儿媳妇去见识见识一下大场面。
