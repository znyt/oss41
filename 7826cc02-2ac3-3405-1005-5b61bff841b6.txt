　　看到罗昊脸上露出思索的神情，云熙如何能不知道罗昊的想法。
<br/>　　“你是想要把这种药剂，在龙隐部队，甚至在全军推广？”云熙说道：“你这个想法很不错，但其中的难度并不小，首先这份资料上的内容是不是可行，按照这份资料是不是真的能把这种潜能激发药剂制作出来还是一个未知数；其次，就算是能成功制作出来，这种药剂会有什么样的副作用，也是一个未知数。退一万步讲，即便药剂全部满足要求，推广开来也会是一个难题，你必须明白，现在军队里面是有‘少爷兵’，‘太子兵’的，他们来参军的目的就是为了给自己‘镀金’的，他们始终认为战争距离自己很远，根本没有随时可能上战场的准备，你觉得让他们接受这种以对身体造成不可逆损害的药剂来激发他们的潜能，应对‘极小’概率会发生的战争，可能吗？”
<br/>　　“还有制造成本的问题。如果这种潜能激发药剂的制造成本过高，超过普通肾上腺素制造成本的话，就更加不可能在大范围内推广，甚至是被研究都很难被通过。”
<br/>　　罗昊不得不承认云熙说得那些可能都是存在的，而且也都是有很高概率发生的，这些情况都不是靠自己，或者靠云熙的努力就能被改变的。
<br/>　　叹了口气，罗昊关闭了资料，把那份资料拖到了桌面上，说道：“不管怎么样也必须要试一试，只要能完成研究，即便只是小批量的生产，对于我们来说，在关键时刻可以用来保命。至于对身体造成不可逆的损伤，只要能从战场上活着回来，那就是最幸运的事情。明天我就把这个东西交给老章头去，他对‘恶魔药剂’可是一直有想法，这个东西他一定会喜欢的，而且龙隐部队的武器装备研究所是有独立研究资金的，就算是没有独立研究资金，让老章头自己掏钱他都愿意干。”
<br/>　　“你啊，早就打算好了是吧？”云熙白了罗昊一眼，说道：“就想着去坑老章头。我突然觉得老章头认识你，是有多倒霉。”
<br/>　　“他倒霉？我还倒霉呢。”罗昊撇嘴说道：“这家伙每次都喜欢捣鼓一些奇奇怪怪的东西，然后就让我给他去试验，完全把我当成了试验品好吗？他做出来的那些特种子弹到底会不会炸膛，他自己都不清楚就让我去试验，你说我能手脚完整的活到现在是有多么不容易。”
<br/>　　“你们两个啊，就是一类人，他坑你，你坑他，谁都不是好东西。”
<br/>　　云熙轻笑着又打开旁边一个文件夹，这个文件夹里倒是没有什么仙岛夫留下来的研究资料，只有一张照片，这张照片看上去是很多年前，加上被粉碎过一次，后来又通过恢复程序恢复出来，所以照片很模糊，只能模糊的看到照片上是一男一女，却看不清他们的面貌。
<br/>　　“这什么也看不清楚啊。”罗昊皱眉说道：“不过仙岛夫会把这张照片单独放在一个文件夹里，可以说明这张照片应该很不一般，难道是仙岛夫这混蛋跟他初恋的？那我还是不看了，免得辣眼睛。”
<br/>　　“你用脚趾头想想就知道这是不可能的。”云熙白了罗昊一眼，打开一个软件，把照片放进去进行修复，很快照片就被复原出来。
<br/>　　当看到照片上的人时，罗昊和云熙脸上都露出诧异之色，他们两个人对视了一眼，盯着照片很是意外的说道：“小丑！”
<br/>　　云岳那次因为云熙的事情回国的时候，曾经带回来过一张小丑的照片，所以罗昊和云熙第一眼就把照片中的小丑认了出来。
<br/>　　“站在小丑身边的这个女人是谁？”罗昊指着小丑身边的玛丽科娃，说道：“这似乎是一个酒会，而且能让小丑合影，要说这个女人跟小丑没有什么关系，打死我也不会相信。”
<br/>　　“‘毒皇后’玛丽科娃！”云熙盯着照片看了好一会儿，终于是把照片中的玛丽科娃认了出来，对罗昊说道：“无常在认识佟千雪之前，他对女性有心理障碍，他的心理障碍就来自于他在外面接受针对特招队员的训练时所留下的，而中间就跟玛丽科娃有关系。”
<br/>　　“你怎么知道的这么清楚？”罗昊对云熙问道。
<br/>　　云熙指了指自己的脑袋，说道：“你忘了我之前在龙隐部队是干什么的？龙隐部队数据库里那些东西，我有什么是不知道的？”
<br/>　　“你这个到底是人脑还是电脑？”
<br/>　　“超级大脑！”云熙回答道。
<br/>　　“难怪那小悠儿这么聪明，全都是她老妈聪明。”
<br/>　　“那是，要不是被你拉低了平均值，她还能更聪明！”云熙理所当然的说道。
<br/>　　“云小熙，要是没有我，能有她吗！？”罗昊看着云熙问道。
<br/>　　“对哦，把这个忘了！”云熙说道：“你的存在感太低了。”
<br/>　　“你别逼我出手哦，我的忍耐是有限的。”罗昊咬牙切齿道，他恨不得把这个女人按在自己腿上，狠揍一顿屁股。
<br/>　　“你舍得对我动手吗？”云熙对罗昊完全就是有恃无恐，说道。
<br/>　　这个……自己还真不敢！
<br/>　　云熙的帮手太多了，不说别的，那两个小的，要是知道他们的妈妈被自己欺负了，还不得跟自己拼命？
<br/>　　“外面称呼玛丽科娃叫‘queen’，而且一直有人猜测，在queen背后的King是谁。”云熙把鼠标点在小丑身上，说道：“这个应该就是queen背后的King了！”
<br/>　　“没想到玛丽科娃跟小丑还有关系。”罗昊盯着照片说道：“我们跟‘暗月’交手了这么多次，可是从来都不知道小丑在哪里，如果能抓住玛丽科娃这根线，或许就能找到小丑，只要找到小丑，也许就能知道‘暗月’这个神秘组织的性质！”
<br/>　　罗昊能感觉到，玛丽科娃将会是一个非常关键的人物，如果不是因为仙岛夫存在硬盘中的这张照片，如果不是因为子午的程序把硬盘中的数据恢复了，或许他们永远都不会知道玛丽科娃跟小丑之间的关系。
