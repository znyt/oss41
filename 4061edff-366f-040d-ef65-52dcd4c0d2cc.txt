　　名满整个魔皇城的铁阿三，竟然被人秒杀了！
<br/>　　甚至，很多人都没有看清楚，铁阿三是怎么死了，只看到他的脑袋突然就飞了起来，便已经身首异处。
<br/>　　“好！”
<br/>　　唯独纳兰柔然，兴奋的拍手叫好，感觉大为解气，望向叶尘的目光中，露出一抹浓浓的崇拜之色。
<br/>　　“我乃赫连将军府三公子，赫连飞鹏，阁下到底是什么人？”
<br/>　　那位赫连公子先是愣了片刻，继而沉声开口道。
<br/>　　叶尘微微抬头，瞥了一眼赫连飞鹏，淡淡道：
<br/>　　“你惹不起的人！”
<br/>　　轰！
<br/>　　众人顿时再次哗然。
<br/>　　赫连将军府，乃是仅次于魔皇宫的超一流势力，府中有数位冲霄境老祖坐镇，便是魔界之主魔皇大人，也要给赫连将军府几分面子！
<br/>　　这家伙是谁，竟然敢和赫连将军府的嫡系公子，如此说话？
<br/>　　而那位赫连飞鹏，听到这话，则脸色发白，眼眸之中闪过一抹杀意，
<br/>　　“好狂妄的口气！阁下这是一定要和我赫连将军府作对了？”
<br/>　　叶尘冷冷一笑，
<br/>　　“和你作对？你还不够这个资格！再不滚，我连你一起杀！”
<br/>　　轰！
<br/>　　叶尘每说出一句话，众人便震惊一分。
<br/>　　这家伙！
<br/>　　他这是一点也没有把赫连将军府，这么一个超然大势力，放在眼里啊！
<br/>　　一旁的纳兰柔然，则两眼放光，小脸上的崇拜之色，越发的浓郁。
<br/>　　“叶前辈，真是……太帅了！”
<br/>　　赫连飞鹏顿时气得浑身发抖，若非他现在还摸不透叶尘的实力，只怕早已经扑上来了，
<br/>　　“好好好！你这百年以来，第一个敢不把我赫连将军府放在眼里之人！”
<br/>　　“百里长老！”
<br/>　　随着赫连飞鹏一声低声，
<br/>　　唰！
<br/>　　众人只感觉眼前一花，那赫连公子的前面，陡然出现了一道，身穿灰袍的老者。
<br/>　　就好像是，凭空出现在众人面前的一样，除了叶尘之外，没有人知道，他从哪里冒出来的。
<br/>　　“嘿嘿！”
<br/>　　那灰袍老者出现之后，顿时冷冷一笑，一双老眼在叶尘的身上，上下打量了一番，方才缓缓道：
<br/>　　“现在的年轻人呐，还真是狂的没边了！有那么一点实力，便不知道天有多高，地有多厚！”
<br/>　　“老夫百里长松，不知道小友，出自何人门下？”
<br/>　　灰袍老者报出自己名字的时候，一脸的傲然之色，显然应该身份不凡。
<br/>　　果不其然，
<br/>　　听到‘百里长松’四个字，酒馆里的所有人，纷纷惊呼开来，
<br/>　　“竟然是百里长松老前辈！”
<br/>　　“他老人家早在数十年前，就已经名满天下了！”
<br/>　　“后来隐退之后，便不知所踪，原来是进入了赫连将军府！”
<br/>　　“百里长松老前辈，早在三十年前，就已经是御空境强者，如今的实力，就算没有达到冲霄境，恐怕也相差不远了！”
<br/>　　……
<br/>　　听到众人的议论声，那百里长松的神色，顿时越发傲然。
<br/>　　纳兰柔然则脸色大变，不由得拉了拉叶尘的衣袖，低声道：
<br/>　　“叶前辈，对方说不定是冲霄境强者，要不我们还是赶紧走吧！”
<br/>　　在纳兰柔然看来，
<br/>　　虽然叶尘也极有可能是冲霄境强者，可是对方的背后，还有一个赫连将军府！
<br/>　　真要是再纠缠下去，恐怕事情将会有些不妙。
<br/>　　“哈哈哈！”
<br/>　　赫连飞鹏似乎也察觉到纳兰柔然的退意，顿时哈哈一笑，
<br/>　　“现在才想走，未免太迟了些！”
<br/>　　说到这里，赫连飞鹏两眼微微一眯，脸色露出一抹戏谑之色，缓缓道：
<br/>　　“纳兰小姐，看在你罗刹城主的面子上，本公子也不为难你，乖乖陪本公子喝杯酒，今天的事情，本公子可以既往不咎！”
<br/>　　“嗯？”
<br/>　　纳兰柔然听到这话，顿时大为心头，当即便忍不住要答应下来。
<br/>　　虽然，她对这个赫连飞鹏，简直讨厌至极，而且陪他喝酒，恐怕免不了要吃点亏，可是为了活命，也只能答应下来了。
<br/>　　不料，
<br/>　　赫连飞鹏说完这话之后，又转而望向叶尘，一脸悠然道：
<br/>　　“至于你嘛……你竟然敢杀我贴身侍卫，实在胆大包天！”
<br/>　　“不过看在纳兰小姐的面子上，只要你跪下磕头道歉，自断一臂，本公子或许可以考虑饶你一命！如何？”
<br/>　　众人听到这话，顿时纷纷暗暗摇头感慨起来。
<br/>　　“要说此人，年纪轻轻，便能秒杀铁阿三这样的高手，绝对算得上是一个天才人物！”
<br/>　　“可惜，他不该招惹赫连公子！”
<br/>　　“好在赫连公子大度，只是让他自断一臂，虽然今后实力会大受影响，不过也算不幸中的万幸了！”
<br/>　　“面对百里长老这样的强者，他也只能乖乖接受了吧？”
<br/>　　……
<br/>　　众人议论纷纷，
<br/>　　显然都一致认为，
<br/>　　在百里长松这样的老牌强者面前，叶尘等人根本没有任何反抗的余地。
<br/>　　然而，
<br/>　　等叶尘一开口，所有人全都愣住了，
<br/>　　“既然如此，那我也给你一次机会！”
<br/>　　“跪下磕一百个响头，然后自断双臂，我便饶你一条小命！”
<br/>　　众人顿时再次哗然，继而纷纷露出一副看傻瓜的表情，
<br/>　　“真是不知死活啊！”
<br/>　　“这小子完了！”
<br/>　　“自寻死路！”
<br/>　　就连纳兰柔然，俏脸也一阵发白。
<br/>　　魔皇城毕竟不比罗刹城，这里高手如云，叶尘如此嚣张，今天这事怕是难以善了。
<br/>　　赫连飞鹏则两眼一瞪，
<br/>　　“你找死！”
<br/>　　“百里长老，给我废了他！”
<br/>　　“是！”
<br/>　　百里长松当即应了一声，直接朝叶尘伸手虚抓，
<br/>　　轰！
<br/>　　一道巨大的手掌虚影，突然凭空出现，继而闪电般，一把便将叶尘握在了掌中。
<br/>　　众人看到这一幕，顿时再次惊叹，
<br/>　　“举手之间，便可调动天地之力，而且真元凝聚如实质，这至少也是御空境后期，才能具备的实力吧？”
<br/>　　“秒杀呀！不愧是百里前辈！”
<br/>　　“那小子虽然厉害，毕竟太年轻了，如何会是百里前辈的对手？”
<br/>　　……
<br/>　　就在所有人都以为，叶尘已经被百里长松一招制服。
<br/>　　这时，
<br/>　　一道冷漠无比，犹如魔神般的声音，陡然在酒馆中响起，
<br/>　　“蝼蚁一般的东西！也敢对我出手！”
<br/>　　“找死！”
<br/>　　彭！
<br/>　　随着‘死’字落下，握住叶尘的那道巨掌虚影，轰然破碎！
<br/>　　与此同时，
<br/>　　蹬蹬蹬！
<br/>　　百里长松一连倒退了七八步，那张老脸之上，尽是骇然之色。
