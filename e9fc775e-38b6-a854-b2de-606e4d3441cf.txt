<br/>				    听到这话，几人沉默下来，他们相视了一眼，不知该不该说？于是，他们的目光都落在几人当中的两名老者身上，以眼神询问着。<br/>　　<br/>　　    见此，两名老者沉默了一会，这时，其中一名老者才道：“主子，在此之前，能否容我们问几个问题？”<br/>　　<br/>　　    闻言，凤九挑了挑眉，从空间中摸出酒葫芦拧开盖子喝了一口酒，神色带着轻松与悠哉，道：“问吧！”<br/>　　<br/>　　    老者看着凤九，道：“我观主子身上并无多少血腥味，杀气也并没有，与这里面的人显得很是不一样，且问，主子原本是何许人？又为何会来到这里？”<br/>　　<br/>　　    听到这话，凤九轻笑出声：“原来是这个啊！”她看向他们，笑道：“我呢，是前段时间刚进来的，如你们所见，我非魔修，亦非邪修，更不是哪个宗门的人，也就算是个散修吧！是在跟魔族魔主手底下的血罗交手时，被他打开的黑暗之门级吸了进来，就这样。”<br/>　　<br/>　　    她笑了笑，又喝了一口酒，继而看向他们：“这回，可以说了吧？”这些人，还真是谨慎。<br/>　　<br/>　　    闻言，两名老者这才看了一眼后，微点下头，道：“主子，其实我们是被仇家所害才进了这里的，我们是一个隐世家族，当年被吸进这里面时，除了我们之外，还有我们的妻小，因此，我们在这里面除了我养活我们自己之外，还要养活我们的妻小，保护他们。”<br/>　　<br/>　　    “咦？”<br/>　　<br/>　　    凤九诧异的看向他们：“你们是一个隐世家族的？还有妻小也进来了？”她微讶，这倒是没想到，据她所知，这里面多为恶人和魔修，小孩一类的更是少见，不过，若真是一个隐藏家族，那他们有妻小倒也正常。<br/>　　<br/>　　    “是，除了我们的妻小之外，还有一些族人。”老者说着。<br/>　　<br/>　　    她将手中的酒葫芦拧起收了起来，问：“你们总共有多少人？在这里面没被其他人发现？”以他们的实力生存倒是没什么问题，不过，若是他们还带着妻小的话，那若是被其他魔修或心术不正的修士所发现，那想要活命可就难了。<br/>　　<br/>　　    “加上我们几个，一共有三十九人。”<br/>　　<br/>　　    闻言，凤九一手托着下巴，道：“三十九人啊！那人这可不少，这么两头凶兽也不够分啊！”想了想，她站了起来，道：“既然这样，那我带你们再去打些猎物吧！免得三天两头进来一趟麻烦。”<br/>　　<br/>　　    几人微怔，相视一眼，继而应着：“好！”他们吃了些肉食后，体力也渐渐恢复了，只要体力恢复，那战斗力也随着恢复，就可以跟那些凶兽战斗了。<br/>　　<br/>　　    她看了几人一眼，道：“那就你们两个伤势较轻的跟我走吧！其他的就先在这里休息一下等我们回来。”<br/>　　<br/>　　    “主子，我们也一起去吧！”老者说着，担心他们三人应付不来。<br/>　　<br/>　　    “行了，你们身上有伤，就先休息着，打些野味而且，两个人足够了。”她摆了摆手，在周围给他们布下了一个结界。<br/>
