　　“炼药？”萧逸一愣。
<br/>　　“不然呢？”易老白了他一眼，道，“不靠丹药提升速度，单凭控火兽武魂，你连破丹融脉都不一定能做到。”
<br/>　　这时，易老见他脸色愕然，以为自己的话重了些，伤了他的傲气和自尊。
<br/>　　便放缓了语气，道，“你虽武魂资质差，所幸，你的领悟力极强，控火兽又恰好是能够操控火焰的武魂。”
<br/>　　“所以，你要成为炼药师，问题并不大。”
<br/>　　“记下来，待会儿去内门宝库换取几本炼药典籍。”易老说着，将几门炼药典籍的名字道出。
<br/>　　“你在第一个考核中得到的分数，足以兑换这些炼药手册了。”
<br/>　　“还有，别再去外门宝库了，那里的东西，对你帮助不大。”
<br/>　　这时，萧逸没有动作，他在考虑。
<br/>　　易老见状，再次道，“还站着做什么？走阿。哦，对了，别担心你外门弟子的身份，这是我的令牌，你拿着即可进入内门宝库兑换宝物。”
<br/>　　说着，易老递给萧逸一个令牌。
<br/>　　萧逸没接，尴尬道，“那个...易老，你刚才报的几门炼药典籍，我都会，所以，不必换了。”
<br/>　　以萧逸如今的炼药本事，单听典籍名字，大概就能判断出这些典籍的程度。
<br/>　　最高不过四品炼药师的知识。
<br/>　　而他刚才之所以没有动作，只是在考虑是否要告知易老自己会炼药。
<br/>　　思索了一下，他衡量再三，觉得并无所谓，便现在才说话。
<br/>　　“你都会？”易老一愣，“小子，你本身就是个炼药师？”
<br/>　　“嗯。”萧逸点点头。
<br/>　　“几品的？”
<br/>　　“三品。”
<br/>　　“三品？”易老再次问道，“可炼制过更高级别的丹药？”
<br/>　　萧逸刚想说话，易老打断道，“我提醒你，最好如实说出，这关系到你之后的修炼，否则，我剩下半天的教导，可能会出错。”
<br/>　　“额。”萧逸皱了皱眉，想到前半天易老的手段，短短时间就能让自己学到这么多东西，不敢乱说。
<br/>　　“炼制过四品的洞玄丹。”
<br/>　　萧逸没说炼制过雪魄丹，那是四品巅峰丹药，要是说出来，也不知会不会有麻烦。
<br/>　　而洞玄丹，乃是四品基础丹药。
<br/>　　两者同为四品，应该问题不大。
<br/>　　“还炼制过洞玄丹，好，很好。”易老脸上，忽然写满了满意和莫名的喜悦。
<br/>　　“这是四品基础丹药，也就是说，你算是准四品炼药师了。”
<br/>　　“啧啧，准四品炼药师，就是放到内门药堂，也足以碾压大部分弟子了。”易老笑着说道。
<br/>　　整个内门药堂，四品炼药师，只有三人，而且是药堂长老的亲传弟子。
<br/>　　而准四品炼药师，也只有寥寥几人。
<br/>　　“你跟谁学的炼药术，又学了多久？”易老忽然凝重地问道。
<br/>　　萧逸道，“跟家族中的一位三品炼药师，那是我们家族最厉害的炼药师。嗯，学了几个月吧。”
<br/>　　“几个月？”易老瞪大了眼睛，通过他对萧逸的接触，他确定萧逸是个很狂傲的人，不会说这种谎。
<br/>　　“我算是明白过来了，为何你会是控火兽武魂。”易老摇了摇头。
<br/>　　“嗯？”萧逸一愣，难道武魂还和这些因素有关。
<br/>　　熟料，易老的下一句话，差点没让萧逸气死。
<br/>　　只见易老煞有介事地说道，“武道天赋、实力、领悟力，你在三个考核中都一一表现了。三项考核皆为第一，可见是何等出色。”
<br/>　　“如今，又是如此的炼药天才。”
<br/>　　“若还让你拥有强大的武魂，那你岂不是成妖孽了，上天，是公平的，不可能什么便宜都给你。”
<br/>　　“这...”萧逸无语了。
<br/>　　这时，易老继续道，“你家族中最强炼药师不过三品，看来只是个小家族。”
<br/>　　“不过，也幸好你那家族前辈只教了你数月，还不至于到误人子弟的地步。”
<br/>　　“再者，以你的年纪，就算放到内门药堂，也是最小的，炼药成就，可谓无可限量。”
<br/>　　“易老过奖了。”萧逸躬身道。
<br/>　　易老摆摆手，道，“既然你本身是准四品炼药师，那就好办了。”
<br/>　　“这半天时间，我便指点你炼药上的知识。”
<br/>　　“看在你炼药天赋如此出色的份上，我再给你一次机会。日后，你还是可以来寻我，我会指点你。”
<br/>　　“还是那句话，你我并无任何关系，你也爱来不来。今天的指点，是我代表剑派给你的补偿；以后的指点，则是我个人的喜好，你来不来，我都不会在意。”
<br/>　　“待你突破洞玄境后，我会推荐你进内门药堂，你的天赋，不该被埋没。裂天剑派，也不应失去这样一位出色的弟子。”
<br/>　　易老一口气说完，再没有说其他事，传授起炼药知识来。
<br/>　　萧逸也发现，易老虽不管剑派的事务，但对于剑派下一代弟子，却十分上心。
<br/>　　两人在炼药知识上，一个发问，一个解答，讨论得甚是兴致。
<br/>　　越是这样的兴致时间，越是过得特别快。
<br/>　　半天时间，眨眼便过。
<br/>　　夜幕降临，天色幽暗，易老停下了指点。
<br/>　　“在三品中的炼药知识，你无比熟悉，甚至可以说是完美；可到了四品的炼药知识，你却只懂皮毛。”
<br/>　　“这证明，你是个了不起的三品炼药师；可在四品，却屁都不是。”
<br/>　　易老训斥了两句。
<br/>　　萧逸撇撇嘴，他看透了当初大长老给的炼药心得，自然在三品中的炼药知识极为出色。甚至是完美。
<br/>　　可在四品，他只炼制过雪魄丹，也只是从那丹方上了解过一点儿知识；自然在四品这个层次很差劲。
<br/>　　嗖的一声....
<br/>　　易老招呼都不打一个便离开了。
<br/>　　萧逸暗暗腹诽他性格古怪，而后摇摇头，再次盘膝坐下，开始他日常的修炼。
<br/>　　在修炼上，他从来都是刻苦得严苛。
<br/>　　翌日清晨，天蒙蒙亮，萧逸从修炼中醒来，只觉神清气爽。
<br/>　　武者在修炼状态，比睡眠更容易补充体力和经历。
<br/>　　下一秒，他毫不犹豫地往外门宝库而去，找易老指点炼药知识。
<br/>　　昨日，易老的指点，虽说只是举手之劳，但对他，却是极大的帮助。
<br/>　　所以，这人情是百分百欠下了，便也不在乎继续欠下去了。
<br/>　　如今，他的武道知识已经不用指点了；只差突破洞玄境。
<br/>　　而这个过程，破丹融脉都需要庞大的力量，暂时是做不到了。
<br/>　　只能先提升自己的炼药水准，好为自己多炼制些高级丹药，为之后突破洞玄境做准备。
<br/>　　外门宝库中，易老。
<br/>　　“小子，来了。”易老淡淡一笑，似乎早有所料。
<br/>　　“早上没空，下午再来吧。”易老却毫不犹豫地将萧逸撵走。
<br/>　　显然，他在‘报复’萧逸。
<br/>　　下午，萧逸再次而来，他没有为难，如同昨天那般，认真地指点萧逸。
<br/>　　.....
<br/>　　第二日...第三日....
<br/>　　萧逸几乎每天都去外门宝库寻易老。
<br/>　　只因，易老似乎有让萧逸更快突破洞玄境的办法；而这办法，需要在四品炼药知识中，有一定的水准。
<br/>　　萧逸只得不断请教，提升自己的炼药水准。
<br/>　　.....
<br/>　　时间，就这样有规律地过着。
<br/>　　早上，去寻易老；晚上，回后山修炼。
<br/>　　一个月后。
<br/>　　以萧逸的天赋，炼药水准大升自然不必多说。
<br/>　　他的修为，也有一定的增幅；他感觉，自己要不了多久就能达到半步洞玄了。
<br/>　　所谓的半步洞玄，便是体内真气超过先天九重；却又没有完成破丹融脉凝气泉过程的这个境界。
<br/>　　今日，萧逸再次去外门宝库。
<br/>　　易老却道，“小子，今日不用来找我指点了。”
<br/>　　说着，他满意地打量了萧逸一眼，道，“不错，一月前，你不过刚刚稳固先天九重修为；如今，竟已接近半步洞玄了，武道天赋确实过人。”
<br/>　　“现在，你该出去历练了。”
<br/>　　“历练？”萧逸面露疑惑之色。
<br/>　　易老言简意赅道，“你需要兑换内门宝库中的一物，但这东西，需要五千剑点，你暂时换不了。”
<br/>　　“剑派内，自有各种历练任务，完成后可得剑点。待你攒够足够的剑点，再回来寻我。”
<br/>　　“去吧。”
<br/>　　易老说罢，身影已经消失在原地。
