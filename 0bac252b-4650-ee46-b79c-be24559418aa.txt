　　“原来叶兄弟是想要找治愈神魂的天材地宝，那可真是巧了，我万宝阁别的没有，天材地宝却多得是！”
<br/>　　金舟之上，张凌云在得知叶尘，前往天玄城的目的之后，顿时哈哈一笑，一脸得意道。
<br/>　　说着，张凌云长袖一挥，
<br/>　　唰唰唰！
<br/>　　两人面前的桌子上，顿时便出现了一排奇形怪状的植株。
<br/>　　“这个是九阳天心葵……这个是玄天龙魂芝……还有这个，是极品铁魂兰！”
<br/>　　张凌云向叶尘一一介绍了一遍，脸上也越发的得意，
<br/>　　“叶兄弟，这些可都是地宝级的灵花灵草，而且对医治神魂有奇效，你若是需要，尽管拿去！”
<br/>　　叶尘两眼微微一眯，在这些灵花灵草上打量了一番，方才缓缓道：
<br/>　　“张公子一番好意，我先行谢过！不过，地宝级的灵物，对我的意义不大，不知道贵阁之中，可以天宝级治愈神魂的灵物？”
<br/>　　“什么！天宝级？”
<br/>　　张凌云听到叶尘这话，顿时眉头一皱，脸上已经浮现出一抹不悦之色。
<br/>　　在修真界，天材地宝而根据等级，又可以分为人宝级，地宝级，天宝级，道宝级，以及那传说中的先天灵宝和混沌异宝。
<br/>　　而在天玄星域这么一个四级星域之中，地宝级的天材地宝，已经是极为珍贵的存在了！
<br/>　　张凌云为了拉拢叶尘，可是忍痛割爱，才将这些天材地宝拿出来，却没有想到，对方竟然还看不上，一张口就要天宝级！
<br/>　　要知道，即便他们万宝阁，也只有那么寥寥几件天宝级的灵物罢了。
<br/>　　这家伙当天宝级是大白菜不成？
<br/>　　张凌云越想越气，甚至误以为，叶尘是为了拒绝自己的拉拢，故意才这么说，好让自己知难而退。
<br/>　　叶尘见状，也知道张凌云想多了，于是右手一番，从乾坤镯里取出了一个紫色药瓶，放在张凌云的跟前，道：
<br/>　　“这是一瓶伏龙丹，为四阶丹药，应该足以兑换你这些灵草了吧？”
<br/>　　虽然张凌云拿出的这些灵草，还达不到叶尘的要求，不过聊胜于无，至少应该可以延缓曦月残魂消散的速度。
<br/>　　张凌云和一旁的那位雷叔，听到叶尘这话，顿时全都微微一愕，
<br/>　　“四阶灵丹？”
<br/>　　灵丹共分九品，四阶灵丹只能算是中等级别的丹药，不过天玄星域地处荒僻，炼丹师极为稀少，所以四阶灵丹也是很少见的存在。
<br/>　　叶尘还以为两人，对价格不满意，于是接着又道：
<br/>　　“我手上只有这些东西，倘若你张公子觉得价格低了，等我到了天玄城，可以再为你炼制一炉！”
<br/>　　不料，叶尘这话一出口，张凌云和雷叔，顿时全都两眼一瞪！
<br/>　　张凌云更是猛然坐直了身体，死死的盯着叶尘，道：
<br/>　　“叶兄弟，你的意思是说，这伏龙丹，是你的亲手所炼？你是炼丹师？而且还是四品炼丹师？”
<br/>　　说到最后，张凌云声音都有些微微颤抖，显然十分的激动。
<br/>　　叶尘微微一怔，很快便反应了过来，
<br/>　　炼丹师，在修真界，是一个极为受欢迎的职业，尤其是在一些低级星域，更是可遇而不可求，也难怪张凌云会如此失态。
<br/>　　“算是吧！”
<br/>　　叶尘淡淡点了点头，一脸平静道。
<br/>　　得到叶尘肯定答复之后，张凌云的眼神，顿时变得更加热切起来，不过热切之中，却又透着一丝怀疑。
<br/>　　毕竟，叶尘实在太年轻了，以他只有金丹境的实力，就可也对抗五阶妖兽，这本身就已经十分逆天，如今居然又多了一个四品炼丹师的身份，这简直让人难以置信。
<br/>　　三人又闲聊了一会，叶尘便带着那些天材地宝离开了。
<br/>　　叶尘一走，张凌云当即让人，将手下的一名炼丹师请了过来，来检查叶尘所给的那瓶丹药。
<br/>　　“回四公子！这的的确确是伏龙丹！筑基巅峰的修士服用一颗，可以提高三成突破的几率！”
<br/>　　那名炼丹师，一番检查之后，顿时满脸的激动之色，
<br/>　　“而且，此丹的品质极高！甚至比我们万宝阁的柳大师，还要高出一线！不知道这是出自那位大师的手笔？”
<br/>　　张凌云听到这个结果，顿时霍的一下，便站了起来，满脸的不可思议之色，
<br/>　　“什么？你的意思是说，此人的炼丹水平，比柳天生还高？哈哈哈！真是天助我也！天助我也！”
<br/>　　原来，他口中的柳天生，乃是他们万宝阁的首席炼丹师，在万宝阁之中，有着举足轻重的地位。
<br/>　　只可惜，柳天生曾公开支持张凌云的二哥，一直和他过不去，也是他成为万宝阁继承人的一个巨大障碍。
<br/>　　如今，他得到了一个比柳天生还要强大的炼丹师，而且实力又如此逆天，一举压过他那位二哥，也就指日可待了！
<br/>　　这时，一旁的雷叔忽然开口道：
<br/>　　“四公子，你莫要高兴的太早了！此子究竟是不是四品炼丹师，还需要进一步的确定！”
<br/>　　“更何况，我观此人眼高于顶，而且城府颇深，想要驾驭此人，只怕不太容易啊！”
<br/>　　张凌云听到这话，脸色的笑容瞬间凝固，虽然他不想承认，可是却也不得不承认，雷叔说的很有道理。
<br/>　　天才往往都是骄傲的，又岂会轻易被驯服？
<br/>　　“传我的命令下去，等回到天玄城，立即帮我搜查天宝级神魂类的天材地宝！”
<br/>　　张凌云一声令下，众人领命而去。
<br/>　　等到所有人都离开之后，雷叔忽然长袖一挥，在房间周围设下了一道禁制。
<br/>　　张凌云见状，顿时眉头微微一挑，
<br/>　　“雷叔可是还有什么话要说？”
<br/>　　雷叔深吸了一口气，方才缓缓道：
<br/>　　“四公子，作为上位者，施恩固然重要，可是手段，同样必不可少！”
<br/>　　“若是拼资源，你无论如何也比不上二公子，倘若此人被二公子拉拢了过去，你又该当如何？”
<br/>　　张凌云顿时一呆，
<br/>　　和他那位二哥相比，他的确没有什么优势，难保叶尘不会投靠到对方的阵营之中。
<br/>　　“那雷叔你的意思是？”
<br/>　　雷叔微微一笑，脸上闪过一抹阴险之色，
<br/>　　“依我看，想要彻底收服此人，恐怕是不可能了，不过我们却可以利用他，直接除掉二公子！”
