　　“尼米兹”级核动力航母，联合行动司令部。
<br/>　　军事监狱中的行动过程全都通过罗昊、李然、李毅峰、慕容飞四个人身上所佩戴的战场视频系统，实时传送到联合行动司令部的主屏幕上。
<br/>　　当看到罗昊跟影子部队接火，掩护李然的时候，云熙双手不受控制的握紧，手心中全都是汗水，这个时候，云熙恨不得自己也在行动现场给罗昊提供一些帮助。
<br/>　　站在房间正前方的詹姆斯看着屏幕上的画面，忍不住点头说道：“这是我见过的最优秀的特种部队之一，而他也同样是我见过的最优秀的指挥官之一。”
<br/>　　这位太平洋舰队司令毫不吝啬对罗昊的赞扬。
<br/>　　詹姆斯看了眼站在一旁的云熙，说道：“如果他是我手下的兵，他现在军衔绝对不会只是少校。”
<br/>　　“将军，即便他不是您手下的士兵，如果他愿意的话，他现在的军衔也不会是少校。”云熙用流利的美式英语对詹姆斯说道。
<br/>　　云熙的话自然不是无的放矢。
<br/>　　当年，因为林晓阳的事情，罗昊在选拔过程中，面临第二次开除的时候，如果他接受那三尊“大佛”中任何一个人的“诏安”，罗昊现在的军衔就绝对不会是少校，他很有可能是上校，甚至是大校，如果不是因为年龄的关系，他或许将会是共和国建国以来，最年轻的将军。
<br/>　　“‘黄蜂’1号和‘黄蜂’2号还有多少时间抵达现场？”詹姆斯对作战参谋问道。
<br/>　　“差不多还需要二十分钟，将军。”作战参谋回答道。
<br/>　　“联系Bravo小队。”
<br/>　　詹姆斯对一旁的作战参谋吩咐。
<br/>　　“将军，跟Bravo小队联系上了。”
<br/>　　片刻之后，作战参谋把耳麦递给詹姆斯。詹姆斯接过耳麦，问道：“Bravo，报告你们现在的位置。”
<br/>　　“‘波塞冬’00，我们大约还有十五分钟可以抵达军事监狱。”Bravo小队队长，海豹突击六队“红鱼”小组组长麦克米兰的声音对詹姆斯回答道。
<br/>　　“Bravo1，Alpha已经发起突袭，他们正在对‘八爪鱼’进行抓捕，我现在命令你们以最快速度赶到军事监狱，准备支援并且接应他们撤离。‘黄蜂’1号和‘黄蜂’2号将会在二十分钟发起空袭，你们必须在空袭开始前，带着那个该死的‘八爪鱼’撤离军事监狱。”詹姆斯对麦克米兰命令道。
<br/>　　“Copy-that！”麦克米兰回答道。
<br/>　　“黑鹰”直升机机舱中，雨点不断的被吹进直升机中，机舱地板上已经被打湿，就连海豹队员身上也是被雨水给淋湿，但是他们对此却并没有什么特别的反应，这对于随时准备被投放到世界各地，执行各种危险任务的海豹六队来说，就是小意思。
<br/>　　坐在舱门口，戴着阻燃面罩，FAST头盔下只露出一双淡蓝色眼睛的麦克米兰把头盔上的战术挡风镜移到自己眼前，对着机舱中其余的Bravo小队队员大声喊道：“‘波塞冬’00命令我们，在最短时间内赶到军事监狱，为Alpha提供火力支援并且接应他们撤离，我们必须在空袭开始前完成撤离。Alpha已经进入‘八爪鱼’的住所，准备对他实施抓捕，我们应该怎么办？”
<br/>　　“绝对不能落后他们！”
<br/>　　“就算我们这次担任支援和接应，我们也是这个世界上最精锐的特种部队！”
<br/>　　“……”
<br/>　　麦克米兰用几句简单的话，就把机舱里这些海豹队员刺激的嗷嗷叫，恨不得立刻在军事监狱上空空降，跟罗昊、李然、李毅峰、慕容飞他们好好的比试一番，看看谁才是这个世界上最精锐的特种部队！
<br/>　　看到自己的目的已经达到，麦克米兰唇角轻轻扬了扬，不再说话。雨夜中，黑色的“黑鹰”直升机加速朝着军事监狱的方向飞去。
<br/>　　……
<br/>　　军事监狱，“八爪鱼”老巢。
<br/>　　Parker刚冲下楼梯就看到从楼下传来的火光，听到从楼下传来的爆炸声，立即折身往回跑，打算从走廊另一头的楼梯离开楼房，可是他刚回到走廊，就看到有两道人影出现在走廊另一头的楼梯口。
<br/>　　“Fuck！”
<br/>　　Parker走到窗户边往外看了眼，一只脚跨上窗台，打算跳窗离开，这是目前他唯一能够离开楼房的办法。
<br/>　　“噗！”
<br/>　　一颗子弹飞来，落在Parker另外一只脚的脚边。
<br/>　　慕容飞手中握着ACE-GAP39SB短突击步枪，对着Parker大声喊道：“Parker，放下武器，站在原地，不然下一颗子弹打得就是你的脑袋。”
<br/>　　Parker自然不会听慕容飞的话，站在原地不动，他不傻，知道自己如果不动，等到被抓回去，送进军事监狱终身监禁都是轻的，而且他也知道慕容飞刚才的话，只是吓唬自己，如果他们真的想要自己命的话，之前那颗狙击步枪子弹就能把自己爆头。
<br/>　　但是那颗狙击步枪子弹只是把自己逼回到房间里，没有狙杀自己，显然对方是想要抓活口！
<br/>　　Parker看向慕容飞和李毅峰，嘴角露出一丝不屑，另一只脚也跟着跨上窗台，准备朝着楼房外挑去。
<br/>　　看到Parker的举动，慕容飞咒骂一声，朝着已经爬上窗台的Parker冲去，如果被这混蛋从这里跳下去，再想要抓他就不可能了。
<br/>　　Parker对着朝自己冲来的慕容飞竖起中指，找准落地，就准备往下跳；可就在Parker身体要离开窗台的时候，一股力量从他背后传来，一把揪住了他的衣服，把他又硬生生的抓了回去，然后重重的扔在走廊中，身体跟坚硬的走廊地面来了个亲密接触。Parker还没反应过来，黑暗中一道人影便是朝着他扑来，紧接着一拳头就打在他的鼻子上。
<br/>　　“妈的，还想跑！”罗昊一拳头打断了Parker的鼻梁骨，又是一拳打在他的脸上，把他嘴角都打得开裂开来，狠声说道：“你再跑一个给我看看，看我不打断你的腿。”
<br/>　　想到死在“战斧”导弹下的孙远、沈飞、张子涛，还有被西伦科特抓走关押起来，现在生死不明的其他队员，罗昊怒从心起，这两拳下去，完全没有留情。
