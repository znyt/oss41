　　港口通向外面的唯一通路是一道水泥直道，不过此时路已经不再是以前的青灰色，殷红的血液已经将整条路染成了血红色，时间一长血液干涸，整条路又变得黑红，看上去带着一股阴森的感觉。
<br/>　　一具具高句丽人的尸体被拖到路边堆成了几个巨大的京观，而他们的头颅则是被砍下来，全部被悬挂到树梢之上。
<br/>　　“背叛者的下场！”六个大字被刻到石碑之上，然后再将石碑立于道路的两头。
<br/>　　不管是砍头还是立石碑，对于‘獠牙’来说都是他们经常干的事情，不过，京观他们却不常堆，所以在堆的时候费了些手脚，最后竟然是在惠日的指点下才堆成的，这让李承乾对惠日这老货多少有些刮目相看。
<br/>　　杜构此时已经快要把胆汁吐干净了，整个人萎靡的蜷缩在李承乾的马车里面，而他对面坐的是小武和杨雨馨两个大小姑娘。
<br/>　　“小雨，我们出去看看怎么样？就看一眼。”小武试着向马车的窗口位置凑了凑，跃跃欲试的想要掀开车窗的帘子。
<br/>　　“公主，殿下说了，不准您去外面，也不准看的。”杨雨馨将李承乾的命令执行的十分彻底，还没等小武的手碰到车帘，便伸手过去将帘子按住，同时拒绝了小武的要求。
<br/>　　“哼，真小气！”小武嘟起嘴，有些郁闷。
<br/>　　她和一般的小孩子不同，幼时的流浪生活让她懂得了许多事情，所以她并没有对杨雨馨发公主脾气，因为就算是发脾气也没用，弄不好把李承乾招回来，还要教训她一顿。
<br/>　　“杜大哥，我们不能出去因为我们是女孩子，你一个大男人为什么也不出去？”可能是在车厢里太过无聊，小武将目标对谁了杜构。
<br/>　　结果杜构只是轻轻抬头看了她一眼，便又一个人低下头去，整个人颓废的像一只瘟鸡。
<br/>　　“小武，你又欺负人了是吧？”就在小武打算再找几句话刺激一下杜构的时候，车厢的门被打开，李承乾从外面钻了进来，坐到杜构的旁边，看着小武笑着问道。
<br/>　　小武眼中闪过一抹狡黠，强词夺理的狡辩着说道：“我才没有，人家只是在分散杜大哥的注意力，省得他胡思乱想。”
<br/>　　“胡说八道。”李承乾伸出手，远远的够着在小武的脑袋上按了一下，然后说道：“行了，事情已经解决了，我们回登州。”
<br/>　　“还要回登州？不是说要出海么？”小武略有些失望，她还惦记着快点南下，去金山寺看看有没有所谓的法海和尚呢。
<br/>　　“出什么海，死了这么多人，总要有个交待吧。”李承乾没好气的翻了小武一眼，对这小丫头神经的大条有了新一层的感悟。
<br/>　　就在李承乾和小武有一句没一句的聊着的时候，杜构像是回魂了一样，坐正了身子犹豫着问道：“高，太子殿下，真，真的都杀了么？”
<br/>　　“是的，一个不留。”李承乾点点头。
<br/>　　老实说，其实他的心里也并不是那么舒服，不过祸是他自己闯的，就需要他自己去摆平，哪怕心里再不舒服，也必须把这件事情应付下来。
<br/>　　人是他李承乾买回来的，管理这些人的主管也是他李承乾安排的，很难想像，如果这些高句丽人突然爆发，给大唐造成巨大损失之后他李某人会面对怎样的弹劾。
<br/>　　所以这件事他必须担起来，哪怕心里再不愿意也必须去扛，这是他这个大唐太子的责任，无论如何都无法逃避。
<br/>　　杜构在听到李承乾的回答之后，再一次陷入沉默，不知是在自怨自怜还是在怨天由人。
<br/>　　李承乾对他说的以及刚刚发生的一些事情，让他明白，高句丽人并不如他想的那样老实，为什么自己以仁心、仁义去对他们，而他们却要这样来回报？
<br/>　　当初释放他们时那些保证与誓言现在犹在耳边回荡，可现在他们却变成了一地的尸体，是应该埋怨李承乾心狠？还是埋怨这些高句丽人狼子野心？又或者真的是当初自己错了？
<br/>　　当初是否真的不应该放他们出来呢？如果当初不放他们出来，现在这些人应该还活的好好的吧？尽管干活会累一些，但至少他们能活着，虽然活的不是那么自由。
<br/>　　杜构并不是没有脑子的人，沉默中他想了很多，从契丹人手中购买奴隶这件事是李承乾的错，可是站在大唐的角度上，似乎又没什么错，必竟用奴隶来修路、建筑要比用自己人省事、省力、省心，就算是不考虑工钱什么的，最低也可以避免一些施工中出现的意外受伤。
<br/>　　可是如果李承乾没有错的话，那么错的是谁？高句丽人？
<br/>　　不，他们已经是奴隶了，还能有什么错的，如果说错那就只能是他们投错了胎，成了一个高句丽人。
<br/>　　而所有人都没有错的话，那就只能是杜构自己错了，他就不应该把那些奴隶释放出来。
<br/>　　突然获得的自由让他们有了不该有的野心，开始试图解救他们的同胞，假如他们解救成功的话……。
<br/>　　杜构不知道还会发生什么，他知道，李承乾说的对，人的欲望总是不穷的，一件事情得到满足之后，很快就会对另外的事情产生欲望。
<br/>　　如果登州营地中的那些奴隶获救之后，杜构已经可以百分百肯定，这帮高句丽人一定会继续折腾新的花样，攻占一两座县城啥的也不是不可能。
<br/>　　经历了这一天的事情，杜构心中隐隐产生一股后怕的情绪，而更多的感悟则是对自己能力的怀疑。
<br/>　　他到了登州所有的一切动作基本上都是在李承乾和老杜头的遥控指挥下完成的，而唯一一件自己做主搞出来的‘业绩’——释放高句丽奴隶，却差点酿成大祸。
<br/>　　想到此处，杜构突然抬起头：“高明，让我回长安吧，我……我不是个当官的料子！”
<br/>　　“当官的料子？”李承乾愕然回头，一时没想明白杜构说的是什么，半晌之后才反应过来，原来这货是怂了。
