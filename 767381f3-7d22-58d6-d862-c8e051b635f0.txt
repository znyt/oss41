　　回到剑派，萧逸来到外门任务处。
<br/>　　“嗯？这么快就回来了？”
<br/>　　仍旧是那外门执事，正在任务处整理各种书籍卷宗。
<br/>　　看到萧逸的到来，他放下了手头上的工作，疑惑地问了一句。
<br/>　　对于面前这个自从分配到外门，就从未正式到外门居所报到的少年，他印象很深。
<br/>　　“20个任务，20只指定妖兽，已经全部斩杀。”萧逸回答道道。
<br/>　　“哦？”执事脸上略微闪过一丝惊讶。
<br/>　　而后似是想到了什么，沉声道，“萧逸，我提醒你，这些任务，必须靠自己实力完成，不能假手于人。”
<br/>　　“你取回来的妖兽内丹，都要经过检验的。”
<br/>　　“另外，冰晶球你可用了？必须要有完整的记录画面，否则，我很可能会断定你作弊。”
<br/>　　萧逸知道，面前这执事是误会了。但他没有解释，而是将冰晶球拿了出来。
<br/>　　事实胜于雄辩。
<br/>　　执事接过冰晶球，看着光幕上不断闪过的一个个战斗画面，脸色也愈加变得惊骇。
<br/>　　“黑铁妖豹，三级巅峰妖兽，媲美半步洞玄武者，一剑斩杀。”
<br/>　　“银风血狼，三级妖兽，群居，清一色先天九重，足有二十头，一剑斩之。”
<br/>　　“泥泽妖蟒，半步洞玄实力，藏身于沼泽数十米之下，环境极为恶劣危险，你竟潜入其中将其斩杀。”
<br/>　　“.....”
<br/>　　执事看着冰晶球上的画面，惊讶的自语着。
<br/>　　萧逸淡淡一笑，他本身就是半步洞玄修为。
<br/>　　在得到冰火道体后，体内气丹、真脉都变得空前强大，实力自然也大增。
<br/>　　虽未突破洞玄，但按照他遇到过的其他半步洞玄来说，他无疑是其中最强。
<br/>　　自然的，斩杀那些妖兽轻而易举，也让他仅用半月时间便完成了20个任务。
<br/>　　这时，执事看着萧逸，不可置信地问道，“萧逸，你真的才来裂天剑派外门一个来月吗？”
<br/>　　“年纪轻轻就有如此实力，进入内门都绰绰有余了。”
<br/>　　萧逸摇摇头，仍旧没有解释，道，“执事，我现在可以得到剑点了吗？”
<br/>　　“当然。”执事说道，“令牌拿来，我会为你添加剑点。”
<br/>　　.....
<br/>　　得了剑点，萧逸便离开了任务处，往外门宝库而去。
<br/>　　如今，他已经有足够的五千剑点。
<br/>　　怀着期待，他进入外门宝库，却并未能找到易老，顿时感到有些失望。
<br/>　　一贯的桌案前，坐了一个中年人，萧逸认得他，乃是外门中一个颇有实权的执事，实力不俗。
<br/>　　“萧逸？来找易老吗？”
<br/>　　显然，这位执事知道半月前萧逸与易老走得很近。
<br/>　　“嗯。”萧逸点点头，问道，“不知执事可知易老去哪了？”
<br/>　　这位执事摇摇头，道，“半月前易老便离开了，也不知去了哪里。我是代替易老来这守宝库的。”
<br/>　　“执事可知易老什么时候回来？”萧逸问道。
<br/>　　“说不准。”这位执事道，“或许今天，或许明天，或许下个月。易老一向行踪飘忽，只有他自己现身，我们是找不到他的。”
<br/>　　“是吗。”萧逸失望地摇摇头，道，“如此，萧逸告辞。”
<br/>　　萧逸转身离去。
<br/>　　这位执事却喃喃自语着，“易老心性古怪，平日里沉默寡言，谁都不待见。”
<br/>　　“也就只有每年新弟子考核时，他才会略微多说些话。”
<br/>　　“这小子是怎么回事，竟得易老如此青睐，更悉心指导了这么久。”
<br/>　　......
<br/>　　第二日，萧逸再次前往外门宝库。
<br/>　　这一次，他没有失望。
<br/>　　因为，那一贯的桌案前，坐着一贯应有的人，那位总是面无表情的老者，正拿着一本书籍细细看着。
<br/>　　“易老。”萧逸行了一礼，笑了笑。
<br/>　　“回来了。”易老放下书籍，淡淡地说着，那张仿佛永远没有表情的苍老脸皮上，此刻竟多出了一丝笑容。
<br/>　　只不过，这丝笑容，很快便褪去，再次恢复了一贯的古井无波。
<br/>　　“倒是厉害。”易老淡淡地说道，“半月时间便完成任务归来了。”
<br/>　　说罢，易老随手一抛，将一块令牌抛来。
<br/>　　“拿我的令牌，去一趟内门宝库，将凝丹五行录换来。”
<br/>　　易老说罢，再次低下头看书，没有看萧逸一眼。
<br/>　　“凝丹五行录？”萧逸接过令牌，自语一声，而后转身前往内门宝库。
<br/>　　裂天剑派，分内门，外门。
<br/>　　最珍贵的修炼资源，几乎都在内门之中。
<br/>　　萧逸跨过外门区域，来到剑派中央位置，这里的风景，确实比外门好太多了。
<br/>　　亭台楼阁、小桥流水，雄伟宫殿、庄严殿堂，让人看得心驰神往。
<br/>　　但，现在的萧逸，仅仅扫视了一眼，而后心思再没有放到这上面去。
<br/>　　内门宝库，在一宫殿之中。
<br/>　　门外，有两个洞玄境武者常年把守；里面，则有一位洞玄九重看守，以及各式禁制，可谓守护森严。
<br/>　　“嗯？外门弟子？”守门的两个洞玄境武者看了眼萧逸，呵斥道，“非内门弟子，不得进入内门区域，你不懂规矩吗？”
<br/>　　“内门宝库，更是内门重中之重，何时轮到你来此，还不速速离去？”
<br/>　　裂天剑派弟子，有着统一的衣服。
<br/>　　单从衣服，就能区别弟子是内门还是外门。
<br/>　　萧逸眉头一皱，道，“我....”
<br/>　　“我什么我。”那守门武者呵斥道，“看你生面孔得很，应该是今年刚加入剑派的弟子吧。”
<br/>　　“念你初来剑派，对地方不熟悉，这次姑且当你是误打误撞来这了。”
<br/>　　“现在立刻退去，下次若再敢随意擅闯，定治你个违背门规之罪。”
<br/>　　萧逸摇了摇头，只能拿出易老的令牌，“奉易老之命，前来兑换东西。”
<br/>　　“易老？”两个守门人顿时一惊，在看到令牌后，则连忙闪过身，让开一条道。
<br/>　　“原来是易老谴你来的，我们事先不知，还望见谅。”两个守门人的态度发生了180度转变，还对着萧逸行了一礼。
<br/>　　萧逸点点头，心中想着，易老在剑派的地位，怕是很高吧。
<br/>　　进入内门宝库，花了五千剑点，萧逸终于将凝丹五行录换到手。
<br/>　　大致看了几眼，萧逸大概明白过来，脸色也瞬间变得惊骇。
<br/>　　因为，这上面说的，竟然是对丹药进行精炼，而后完全除去丹药上的杂质。
<br/>　　众所周知，淬体丹、后天丹、先天丹、洞玄丹....等一系列能直接提升武者体内真气的丹药，是含有杂质的。
<br/>　　故武者不能大量吞服。
<br/>　　每吞服一粒，便需要待得完全消化，且将杂质排出，才能继续吞服。
<br/>　　但，凝丹五行录，却能将杂质完全除去。
<br/>　　这代表着，通过这些手段，日后炼制的丹药，将能无限服用。
<br/>　　只要有足够的丹药，突破修为，简直如喝水般简单。
<br/>　　如此强大的炼丹手段，难怪需要五千剑点才能兑换。
<br/>　　不过，凝丹五行录上的知识，极为玄奥。萧逸一时也难以完全悟透，看来还是得回去请教易老。
<br/>　　回到外门宝库。
<br/>　　“易老，凝丹五行录已经换来了。”萧逸说道。
<br/>　　“嗯。”易老点点头，微微放下书籍，道，“虽然凝丹五行录上的内容，我早已知晓。不过，剑派有剑派的规矩，我不能私传。”
<br/>　　“你先自己看，不要问，今日，只需完全看一遍。有什么不懂的，明日再来寻我。”
<br/>　　说罢，易老再次拿起了书籍。
<br/>　　“是。”萧逸点点头，他早已习惯了易老的古怪性格，行了一礼，转过身，离开了外门宝库，往后山而去。
<br/>(本章完)
