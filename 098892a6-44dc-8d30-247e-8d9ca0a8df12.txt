        &lt;h3&gt;第三卷第一百七十六章  刘诚之&lt;/h3&gt;
<br/>
<br/>        刘诚之告别了凌辰，踏上了回家的路，董亚子那女人看着大大咧咧很是活泼，其实心细如发，就拉了一下他的袖子就能猜测他的身份，原本他一个工作人员是不可能和董总坐在一起吃饭的，老狐狸父女肯定是调查过他，要不然不会叫上他和凌辰一起吃饭的。
<br/>
<br/>        他和凌辰在一起工作了两年，凌辰都没有发现他的身份，而董亚子就发现了。董亚子临走之时说的那句话他也知道是什么意思。
<br/>
<br/>        董亚子也和他一样选修了安老师的课程，所以年后他是会和董亚子在安老师家里相见的。
<br/>
<br/>        想起和董亚子第一次的first  kiss，他的心跳就加速，那个疯女人竟然夺了他的first  kiss，他是一个大男人，要夺也是他夺好不好？想一想他的心里就很憋屈，这个场子一定要找回来，找不回来他就不算是个男人。
<br/>
<br/>        刘诚之一路上都在想着要怎么找回他的场子，什么时候能找回他的场子，对了，找回场子的时候也让凌辰看着，免得面子上挂不住。
<br/>
<br/>        下了飞机，家里安排的司机姬叔早早的等在了机场：“诚之，刘总让我来接你。”
<br/>
<br/>        “姬叔，我打个车就回去了，就不用麻烦你来接了啊。”刘诚之并没有让姬叔替他拿行李箱，而是自己将行李箱放到了后备箱。
<br/>
<br/>        姬叔跟着父亲很多年了，也是看着他长大的，对他就像是对自己的孩子一样，小时候父亲忙，姬叔没少带他玩。
<br/>
<br/>        “你这孩子每次走都是一年，过年了你才回来，让人很揪心啊。”姬叔打开车门，看着刘诚之坐上车。
<br/>
<br/>        “鸟儿总是要离巢的，我总不可能一辈子都在父亲的庇护下生活吧？”刘诚之笑嘻嘻的说。
<br/>
<br/>        “这次刘总估计不让你出去了，你怕是要好好的回来工作了。”姬叔给刘诚之打预防针。
<br/>
<br/>        “我外面的工作还没有辞，做人不能言而无信，这一年怕是回不来的。”
<br/>
<br/>        “签合同了？”
<br/>
<br/>        “签合同了啊，手头上的事情还没有做完，老板一时半会找不到合适的人接手，我只能继续在他们公司干了。”刘诚之撒了个慌。
<br/>
<br/>        “唉，赚钱是小事，主要你要能学到一点有用的东西，你做的那个行业能学到东西嘛？”姬叔忧心的问。
<br/>
<br/>        “专业知识是学不到，但是社会关系分析学倒是学得很透彻，也算是一门学问吧。”刘诚之考虑了一下。
<br/>
<br/>        “什么社会关系学？我怎么不知道？”姬叔不解的问。
<br/>
<br/>        “就是在一定意义上，社会可以看作人们在实践活动中所形成的人与人之间的关系整体。说白了就是我学到了很多和人相处打交道的知识。”
<br/>
<br/>        “哦。这个你确实是需要好好学学，别学你爸一根筋，总是会得罪人。”
<br/>
<br/>        “不是，我爸不是经常教育我对人要懂礼貌，要多替别人着想吗？”
<br/>
<br/>        “你爸的一根筋和对你有礼貌没啥关系啊，这不前几次开会还和老陆家怼起来了，把他自己还气得不行。”
<br/>
<br/>        “那也是我爸怼该怼的人吧。一根筋也有一根筋的好处，要不然都是墙头草两面倒这个世界就乱了套了。”刘诚之对父亲还是充满了尊敬。
<br/>
<br/>        刘家世代是读书人，爷爷伯伯叔叔都是教授，只有父亲特列，父亲在一个部门上班，是老总级别的人，工作性质也不一样，他的学习也是很好的，记忆力特别的好，父亲要让他也努力一下，后期干和他一样的工作，只是刘诚之心里还是来来回回的。
<br/>
<br/>        回到家，父亲母亲，哥哥也都回到了家，他们家其实平时都是各忙各的，只有过年的时候才在家里，自小刘诚之就养成了独立的个性，上大学的时候都是他一个人住在诺大的别墅里。
<br/>
<br/>        父亲温和而严厉，经常鼓励他们要做一个有担当的人，当然父亲自己也是这样的一个人，随说有过刚易折这样的话，但是刚正不阿确是父亲的性格，碰到不好的事情是坚决不让步，这也就是姬叔嘴里的“一根筋”，不愿意和有些人同流合污。
<br/>
<br/>        中国人有辞年的习俗，刘诚之回到家自然是要走走亲戚的或者拜拜师长。
<br/>
<br/>        刘诚之和父亲一起去接回了爷爷奶奶，也拜访了一些亲戚朋友，在回家的路上，俩个人就目前的态势进行了讨论。
<br/>
<br/>        “现在的社会发展太快，总是感觉睡一觉起来身边又是另外的一个样子，但是个体和个体之间却没有因为发展太快而产生隔阂，反而会协同合作，不得不感慨人们的适应性。”刘诚之老成的分析着。
<br/>
<br/>        “时代的变化是不会停止不前的，人们只能努力适应，那些老的思想和观念一定会被淘汰，现在网络信息化的发展早已经起到了媒介的作用，那些个乌七八糟的东西也会藏不住的。”刘父似乎有些火气。
<br/>
<br/>        “听姬叔说您和别人怼起来了？是不是那口气还没有消下来？”刘诚之关切的问。
<br/>
<br/>        “唉，消不下来啊，有些事情不是说我能消口气的问题，而是立场和道义的问题。”
<br/>
<br/>        刘诚之沉默了，对父亲的工作他不能评头论足，因为他没有参与，平时和父亲也只是讨论一些看法。
<br/>
<br/>        “你还打算在外面浪多久？”刘爸爸见刘诚之沉默着，便问刘诚之来年的打算。
<br/>
<br/>        “我们公司今年出了一点小小的变故，厂家要我们配合他们打击假冒伪劣产品，年年都会有三一五，我们明年会借助那个机会进行整顿，我可能会被派到甸城。”刘诚之并没有告诉父亲，是他自告奋勇的去的，只是说被安排去的。
<br/>
<br/>        “其实我觉得你可以辞了那份工作的。”
<br/>
<br/>        “您不是教育我们做事情要有始有终吧，等这一年过了吧。”他们父子之间时如老友，谈话是很轻松的。
<br/>
<br/>        “也好，只是甸城太远，你过去怕是生活不习惯。”儿女总是父母亲的心头肉，走到哪里都会牵挂。
<br/>
<br/>        “总得要适应的，再说，我们凌总也没有合适的人过去，我也算是为公司出一份力吧。”
<br/>
<br/>        “你打算什么时候去看安教授？”刘爸爸问。
<br/>
<br/>        “你也想去吗？我约好了是明天下午的。”刘诚之问父亲。
<br/>
<br/>        “那不如一起过去，我也想见见安教授。”刘父心情很不错，他们家和安教授家关系不错，虽然他不做学问，但是不妨碍和安教授做朋友。
<br/>
<br/>        刘家这边是父慈子爱，凌辰确因为没有见到宁雅而度日如年！
