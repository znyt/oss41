　　欧洲某国，一栋写字楼内，SHADOW正在他那台笔记本电脑上十指如飞的敲打着，眼睛中跳动着兴奋的光芒，“来吧，GHOST，我知道你一定会出现的，我现在迫不及待的想要跟你过招了，我会让这个世界变得更加疯狂起来。”
<br/>　　高雄市那家通讯公司的刘经理和chen，只不过是SHADOW手里掌握的“肉鸡”之一，除了刘经理和chen之外，SHADOW还有十几个“肉鸡”在帮着他在网络上利用这种勒索病毒进行攻击。
<br/>　　SHADOW在黑客界有着强大的号召力，从当年由他主导，对东南亚各国以及俄罗斯发起的黑客入侵就可见一斑，只要SHADOW希望，或者说需要，就会有大量黑客为他效力，其中有“暗月”的人，也有网络上的自由黑客。
<br/>　　在网络上成为黑客的人，不是在现实生活中失意想要在虚拟世界得到在现实生活中所得不到的尊重和敬畏之人，就是一心为了追求刺激，带有浓烈冒险精神的人。
<br/>　　想要煽动这种人，对于SHADOW来说，轻而易举。

<br/>　　杭山市国际机场。
<br/>　　原本慕容飞是打算自己一个人来机场，但佟千雪执意要来送机，慕容飞拗不过她，只能答应佟千雪的要求，让她来送机。
<br/>　　“自己一个人在家多注意，别没胃口就不吃饭，注意休息，感觉不舒服就去医院，有问题就打电话给我妈……”
<br/>　　领完登机牌，慕容飞站在安检口絮絮叨叨的对着佟千雪进行一系列的叮嘱，生怕她会忘记一样，一件事情都要叮嘱好几遍。
<br/>　　佟千雪在慕容飞背后推了他一把，说道：“行了，赶紧过安检吧，不然一会儿广播里就要叫你的名字了。”
<br/>　　“我跟你说得，你别忘了啊。”慕容飞又一次叮嘱道。
<br/>　　“我知道了。”佟千雪无奈的白了慕容飞一眼，说道：“我自己就是医生，我还能不知道怎么自己照顾自己，唠唠叨叨的简直比我妈还要烦。”
<br/>　　慕容飞背着背包朝着安检口走去，走了几步又停下退了回来，对着佟千雪说道：“佟医生，你是不是忘了什么？”
<br/>　　佟千雪没好气地白了他一眼，踮起脚尖在他脸颊上亲吻了一下，可是还没等佟千雪退回去，她的后脑勺就被慕容飞伸手勾住，狠狠地亲吻着她的嘴唇，舌头挑逗着撬开佟千雪的牙齿，跟她的小舌纠缠在一起。
<br/>　　机场，是个充斥着离别和相聚的地方，每天都会有人在这里挥手分别；每天也会有人在这里嬉笑着相逢。
<br/>　　机场中来往的旅客，还有机场的工作人员对慕容飞和佟千雪之间的拥吻，已经是见怪不怪了。
<br/>　　慕容飞一直吻得佟千雪快要喘不过气来了，才把她放开，在她的娇嗔声中，转身走向安检口。
<br/>　　窗口背后的警察形式化的向慕容飞索要证件和登机牌，当慕容飞被自己的军官证和登机牌递给窗口背后的警察，警察打开军官证看了一眼后，看向慕容飞的眼神立刻变得尊敬起来，就连把登机牌和军官证递还给慕容飞的时候，都是站起身来双手递还。
<br/>　　看着慕容飞通过安检，身影消失在安检口，佟千雪才背着小包往回走去。佟千雪并没有注意到，在人群中，一个带着鸭舌帽的男子正拿着手机，把刚才慕容飞和她拥吻、分别的一幕全都记录了下来。
<br/>　　得到了自己想要的东西，鸭舌帽男子嘴角勾起一抹得意的微笑，把手机塞进牛仔裤口袋里，然后转身离开从另外一个出口，离开了候机楼大厅。
