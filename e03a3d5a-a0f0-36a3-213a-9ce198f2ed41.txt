　　而就在王玄策与李承乾讨论望远镜到底应不应该卖的时候，城外马球场上的表演赛已经进入了高潮阶段。
<br/>　　比分四比四，皇室与程家各进四球，随着比赛进入尾声，眼看再有不到一刻钟整场比赛就要结束。
<br/>　　此时此刻不管是皇室的队伍还是程家的队伍全都打出了真火，看着方就像看着生死仇敌，一阵阵惨烈的杀气开始在赛场上弥漫，刑部来的裁判早就退到不知哪里去了，赛场上除了两人只球队之外再也没有第三个不相干的人。
<br/>　　“小子，看不出来，你们还满厉害的，如果遇到的不是老子们，听怕这次还真让你们赢了。”程家的愣货早就已经忘了对方是谁，代表着谁，甚至连家里小姑奶奶的话也忘到了脑后，数次进攻失利，数次被对方反击得手使得这位老程的得力下属开始变的疯狂。
<br/>　　不过张启元经过了三个小节的比赛之后紧张的情绪已经得到了缓解，面对一脸狰狞的程家愣货微微一笑：“比赛还没有结束，你还有机会！”
<br/>　　“废话少说，开球！”程家的愣货撇撇嘴：“希望你的能耐能和你的嘴一样厉害。”
<br/>　　“那你就看着好了。”张启元还一个十分雷同的表情，手下球棍微微一拨，红色的马球便直直飞向程家的队伍。
<br/>　　而就在同时，一直唿哨响起，程家的队伍和皇室的队伍双双放弃了那个红色的实心球，十分默契的举着球棍撞到了一起。
<br/>　　一场好好的表演赛，最后竟然演变成了群殴，这大大出乎所有人的预料，看着球场上棍影翻飞，战马嘶鸣，几乎让人身临杀斗场中。
<br/>　　大赛的主办方礼部的官员们面对这样的情况也有些懵逼，完全不知道应该如何去应对，因为他们从来也没有想过比赛打到最后竟然会出现这样的情况。
<br/>　　而与礼部官员不同的是，观众们全都沸腾了，正所谓看热闹不怕乱子大，比赛比到打起来让人一看就知道是比出真火的表现，如果盛景可是毕生难得一见。
<br/>　　所以就在两支球队撞到一起的瞬间，赛场中的观众再也抑制不住心中的那份狂热，纷纷跳起来开始为自己喜欢的队伍加油，呐喊助威。甚至一杆杆插在场边的大旗被拔出来，一个个鼓手被推开，那些有过从军经验的家伙们竟然击起大唐军阵突击时的战鼓。
<br/>　　旌旗、战鼓、呐喊，一瞬间让赛场上的两支队伍忘记了一切，只觉得又像是回到了那血肉横飞的战场。
<br/>　　“大唐，万胜！杀！”
<br/>　　“大唐飞凤，血染长空！飞风军，随吾杀敌！”
<br/>　　虽然赛场之上每只队伍只有九人，但战吼声依旧带出了大唐军人无所畏惧的气势，两支由精锐士兵组成队伍就这样撞到了一起，人吼马嘶战成一团。
<br/>　　至于那个红色的小球……，爱谁谁吧，只等对对方揍趴下了，想进几个都行！
<br/>　　……
<br/>　　李承乾在收到消息的时候被惊的目瞪口呆，还没有离开的王玄策也被震的目眩神迷！
<br/>　　这特么叫什么事儿啊，太丢人了啊！明明是一场表演赛，最后怎么就打起来了呢？！
<br/>　　李承乾终于知道为什么后世的一些团体比赛不用那些配合默契的军队了，敢情特么这帮家伙太有血性，稍微控制不好就会把比赛演变成一场群殴。
<br/>　　……
<br/>　　而与李承乾不同的是，程咬金这老货在收到消息之后，似乎没觉得有什么不对，只是淡淡的问了一句：“谁赢了？”
<br/>　　“老爷，平手！”管家苦笑着答道。
<br/>　　老程闻言之后点了点头：“那就好，没输就好，若是输了回头老子拨了那个小崽子的皮。”
<br/>　　管家除了点头还能说什么呢，他家老爷是个什么德性他又不是不知道。
<br/>　　不过消息传到程琳那彪悍的女人耳中之后，乐子可就大了，这彪娘们竟然带着一票飞凤军的大头兵直接把那几个参与斗殴的家伙全都从营地拖出来好一顿暴打。
<br/>　　原因很简单，丫的，不听老娘的话，打不死你！
<br/>　　此前程琳曾经特地嘱咐过那几个去参加比赛的家伙，这只是表演赛，该让的时候就让，大家都知道怎么回事儿，不丢人。
<br/>　　可是这几个家伙倒好，打起比赛把所有的嘱咐全都忘了，拼了命的打球不说，最后还特么在赛场上玩起了群殴。
<br/>　　而另外一边，张启元他们这批人也不好过，堂堂的飞凤军，大唐第一精锐，竟然与程家那几个痞子打成了平手，这特么不是扯蛋么？还第一精锐，哪里精锐了？！
<br/>　　所以在程琳教训了自家的几个愣货之后，又回了飞凤军大营，把张启元他们几个又是一顿帝国主义式的毒打，末了还警告他们几个，若是以后再打平手全部开除军籍，都特么滚回家种地去。
<br/>　　对于这样的处理方式，李承乾并没有多说什么，他是皇帝，这种事情还轮不到他来管，甚至在他看来程琳都不应该插手到这件事情里面。不过既然程琳已经伸手管了，那就让她管到底好了，从今往后所有比赛的赛场稳定工作就让程琳接手。
<br/>　　……
<br/>　　不过对于这一切，长安的百姓却是不关心的，在他们看来那一天的表演赛就是这辈子看过的最精彩的比赛，足够他们津津乐道一辈子。以至于接下来的正式比赛那些看过表演赛的家伙们都没有了兴致，总是念念不望表演赛的精彩。
<br/>　　不过有幸看到那场比赛的人毕竟还是少数，大多数普通百姓还是没有机会看的。
<br/>　　毕竟开幕式那天的票价卖的很贵，又没有什么正式的比赛内容，很多普通百姓还都留着有限的几个钱准备去看自己心中喜欢的球队。
<br/>　　而就在这个时候，第一支非大唐的队伍跨海而来，他们是倭国代表队，因为距离近的关系，他们直接从釜山登陆，然后乘坐火车，只用了不到十天的时间使到了大唐长安。
<br/>　　这种速度简直就是他们以前不敢想像的，带队的秦吉胜不止一次的眼中露出贪婪的目光，似乎很想把铁路这个东西带回到倭国。
