　　罗马不是一天建成的，办法也不是一天能够想出来的，但解决的办法李承乾已经想了整整十年了，却还是没有想出来，不得不说，这就是一个天大的讽刺。
<br/>　　“哥，想什么呢，这么出神！”就在李承乾一边走一边想着问题的时候，身前突然出现一个影子，害他险些一头撞上去，待看清是长乐之后，李承乾气鼓鼓的瞪了她一眼：“臭丫头，想死我啊。”
<br/>　　“哥哥胆子那么大，怎么可能被吓死。”长乐俏皮的皱了皱可爱的小鼻子，两只小手背在身后，上身不断晃来晃去娇声说道：“刚刚哥哥是不是看到人家发脾气的样子了？是不是那个样子吓到哥哥了？”
<br/>　　李承乾摇摇头：“没有，你做的很好，有些时候宫里闹的不成样子，你这个长公主就是应该管管。”
<br/>　　“那还好，人家还担心你心疼苏玫他们，生人家的气呢。”长乐的小脸绽放出一丝笑容，不过很快又收敛下去，正色说道：“不过大哥，我最多也只能帮你一时，帮不了你一世啊，你的那几个老婆还是要你自己解决。”
<br/>　　“我怎么解决？我现在一个头两个大！”李承乾没好气的说道。
<br/>　　要知道，他现在不是不想解决，而是真的没有办法解决。
<br/>　　正所谓手心手背都是肉，他又能真的狠心处理谁呢？再说就算是处理了一个，难道另外几人就能消停了？只怕未必。
<br/>　　欲望这种东西只要开始产生，就不是那么容易压制的，能压制一时也压制不了一世。
<br/>　　“哥，你真的想不到办法？这可不像是你哦。”长乐那丫头似乎有什么话想说，但却一直没有挑明，总是旁敲侧击，若有若无的指点着着李承乾，就好像她已经有了办法一样。
<br/>　　李承乾自然能够听出长乐话里有话，于是对着身后的夜魅和小白挥了挥手，示意她们都退的远些，等到身边再无碍眼之人才看着缓缓说道：“有什么话你就说吧，现在没有外人了。”
<br/>　　“我说了你怎么感谢我？”长乐大眼睛忽闪忽闪的，带着别样的情感。
<br/>　　怎么感谢？钱？这丫头不缺啊！男朋友？这个貌似有点难，不过也不是没有，大不了把那个波斯王子抓过来给长乐当男宠养。
<br/>　　想到这里，李承乾露齿一笑：“要不我给你找个男朋友吧，大唐的番邦的，国王也好、王子也罢，看上谁了你说话！”
<br/>　　“啪”李承乾刚刚说完，小腿上就被长乐狠狠踢了一脚，同时听那丫头娇嗔道：“臭大哥，你说什么呢！”
<br/>　　“行行行，不说，不说了啊。”李承乾一边跳着，一边在小腿上揉着。
<br/>　　“狗嘴吐不出象牙，本来还想提醒你一下，现在本宫心情不好，不想说了。”长乐鼓着腮帮子，小脸被气的通红。
<br/>　　这个臭大哥也真是的，就像一根木头一样，别人家哥哥都是把妹妹宠上他，偏生他就会说一些不着调的话来调侃自己，也不知自己上辈子造了什么孽，竟然会有这样一个大哥。
<br/>　　可是，想归想，气归气，却还是不能不管他，想到当初在那个山洞里面，哥哥挡在自己前面那一幕，长乐在心底发出一声长长的叹息，瞪着傻呆呆的李承乾说道：“其实你当初的办法就很好，让苏玫她们几个每个人都有事情做，这样就可以避免很多事情。”
<br/>　　“啊？不是说……”李承乾瞬间愣住了。
<br/>　　要知道，当初苏玫她们几个的确是各自都有自己的事情做，像苏玫就是帮着长乐处理医学院的事情，孔雯处理教育方面的事情，小武那丫头则是负责一些财货。
<br/>　　可是后来因为种种原因吧，这几个丫头的差事都是被放下了，每天都缩在宫里没事儿打打麻将，看看小书。
<br/>　　时间长了，李承乾也就习惯了这样的生活，所以刚刚一时间他还真的没想起来这个办法。
<br/>　　“要我说你就是灯下黑，现在大哥你已经是皇上了，难道这点事情还处理不了？后宫不得不干涉朝政，但是做些生意，关心一下教育没问题吧！”长乐看着若有所悟的李承乾，继续给他加码。
<br/>　　“对，对对对，长乐，你可真是我的好妹子！”兴奋中的李承乾乐的哈哈大笑，最后竟然习惯性的抱起长乐原地转了好几个圈子。
<br/>　　要知道，虽然长乐的提醒只是一个方案，甚至可以说连个完整的计划都没有，但对于李承乾来说这就是一个契机，如果引导得当，还真的可以分散那几个老娘们儿的注意力。
<br/>　　要知道，之所以后宫内斗不休主要原因一是因为权利，二就是因为无聊。
<br/>　　毕竟皇宫就那么大，好几大千人住在一起又不能出去，怎么可能没有矛盾，无聊的时候又怎么可能不胡思乱想。
<br/>　　而想的多了，自然也就会生出事端，慢慢的那几个可以发号施令的位置也就成了众人眼中的终极目标。
<br/>　　这些说白了，其实还是一个眼界的问题，毕竟后宫太小了，不能干涉外事的情况下能给这些精明的女人施展‘才华’的地方也就剩下的宫半斗。
<br/>　　可是如果把这些娘们儿放出宫呢？放一些权利给她们，让她们有更多的地方发泄多余的精力，估计很快这几个娘们儿就会对那个只能管理几千人的位置没了兴趣吧。
<br/>　　想到这里，李承乾不禁对后世那个要求妇女解放，参加工作的人佩服万分。
<br/>　　其实真正说起来，得到解放的何止于妇女，对于广大男同胞来说其实这也是一种解放不是。
<br/>　　否则试想一下，诸位的老婆大人天天都在家里守着，每天所有的注意力都在诸位的身上，什么私房钱，什么女性朋友，每天都能给你翻出来，这样的生活还有什么意思？
<br/>　　问题得到解决之后的李承乾真是太高兴了，高兴到忘了长乐已经不是当初的那个小丫头，而是一个二十多岁的大姑娘，他这一抱瞬间就把那丫头弄了个大红脸，羞的一颗螓首差点垂到胸口里面去。
