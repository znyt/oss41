　　原来，
<br/>　　方才那句话，
<br/>　　对于叶尘来说，意义非凡！
<br/>　　……
<br/>　　那是进入修真界的第五百多个年头，
<br/>　　彼时，叶尘已经成为了修真界，最年轻的合道真君。
<br/>　　他意气风发，狂傲自负，以合道之境，击败无数渡劫期强者，自号为狂帝，风头一时无两，几乎可与八大仙门的巨头并驾齐驱！
<br/>　　为了寻求突破，他毅然踏入修真界十大绝境之一的大罗天境。
<br/>　　也就是在那里，叶尘邂逅了修真界第一美女，落霞宗圣女曦月仙子。
<br/>　　大罗天境在十大绝境之中，也是排名首位，里面自然险境重重，两人在生死之间，相互扶持帮助，渐渐生出了些许情愫。
<br/>　　但是叶尘向来独来独往惯了，再加上前世心魔作祟，于是在两人费尽千辛万苦，终于逃出绝境后，叶尘也打算直接狠心离开。
<br/>　　而就在那时，
<br/>　　两人在青缈峰顶，夕阳之下，一袭紫衣的曦月仙子，缓缓说出了那番话，
<br/>　　“叶尘，当你身在高处，却发现自己总是孤独，无人与你说话，那你身在高处，又意欲何为？”
<br/>　　这句话，就犹如醍醐灌顶，狠狠击中了叶尘内心深处，最为柔软的那个角落，让他如梦方醒。
<br/>　　刹那之间，叶尘忽然发现，自己之前的五百多年，似乎都白活了！
<br/>　　他看似风光无限的外表之下，内心深处竟是如此的独孤、寂寞，外人眼中嚣张跋扈的一代狂帝，原来也不过只是一个，用围墙将自己封闭起来的叛逆青年罢了。
<br/>　　而就是这么一句话，瞬间击溃了叶尘所有的伪装，让他懂得了直面自己的内心，勇敢的迈出了那一步。
<br/>　　也正是因为这句话，叶尘彻底爱上那个聪慧而美丽的女人，决定要和她永生永世，长相厮守……
<br/>　　而如今，这句话，突然从萧若曦的口中说出，他焉能不惊不喜？
<br/>　　这极有可能，是曦月的残魂，已经苏醒！
<br/>　　然而很快，叶尘失望了。
<br/>　　从萧若曦的眼神之中，叶尘并没有寻找到曦月的踪迹。
<br/>　　萧若曦被叶尘身上突然升起的强大气势所惊，俏脸微微发白，
<br/>　　“叶……叶尘，你……你怎么了？”
<br/>　　叶尘这才意识到自己失态了，当即收起身上的气势，两眼死死盯着萧若曦，道：
<br/>　　“刚刚那句话，你是听谁说的？”
<br/>　　萧若曦一脸迷茫，歪着脑袋，想了好一会，道：
<br/>　　“我也不知道，就是不由自主的说出了这句话，就好像……就好像本来就在我的脑袋里似得……”
<br/>　　叶尘听到这话，又顿时一喜，
<br/>　　“你别动，让我进入你的识海看一看！”
<br/>　　一边说着，叶尘当即在萧若曦旁边坐下，将手掌缓缓按在萧若曦的小脑袋上。
<br/>　　萧若曦顿时俏脸微微一红，同时又浮现一抹黯然之色。
<br/>　　她其实也清楚，叶尘之所以对她如此重视，无非是因为她识海中的那道残魂。
<br/>　　不一会，叶尘的手掌缓缓挪开，脸色露出一抹喜色。
<br/>　　曦月的残魂，比之前强大了许多，不出意外的话，随时都有可能苏醒。
<br/>　　这让原本还有些郁闷的叶尘，一下子便振奋了起来，心情大好。
<br/>　　“若曦，我之前给你的养魂丹，现在应该也快吃完了吧？走，我马上再炼一炉给你！”
<br/>　　说完这话，叶尘拉起萧若曦，便朝回走去。
<br/>　　听到这话，萧若曦的小脸，顿时更加幽怨。
<br/>　　……
<br/>　　回到宗门之中，
<br/>　　众弟子在楚飞烟的指挥下，正在卖力的维修着。
<br/>　　看到叶尘回来，楚飞烟走了过来、
<br/>　　“狂仙门？”
<br/>　　叶尘望着上方那个硕大的金色牌匾，不由得哑然失笑。
<br/>　　楚飞烟也笑了，
<br/>　　“这是尹庄主的意思，他说他们既然已经奉你为主，门头自然也要改一改，所以我就以你的名号，取了狂仙门这个名字，门主大人若是嫌这个名字太土，我可以再让人去改！”
<br/>　　提及尹百川，叶尘不由得脸色微微一黯，
<br/>　　“不必了！既然是尹庄主的意思，那就这样吧！”
<br/>　　楚飞烟点了点头，道：
<br/>　　“那我先带你去看看你的住处吧！”
<br/>　　很快，
<br/>　　在楚飞烟的带领下来，三人来到进入宗门之中，穿过几道走廊之后，来到一处环境优雅的院落之中，而且刚好位于整个宗门的正中央位置。
<br/>　　院落很大，小桥、流水、凉亭、绿竹……应有尽有，风景如画，诗意十足。
<br/>　　除此之外，还有客厅、卧室、炼丹房，也完全依照古式风格建造。
<br/>　　“怎么样？门主大人，你可还满意？”
<br/>　　楚飞烟一脸得色。
<br/>　　叶尘也不由得点了点头，
<br/>　　“不错！可是缺少了一些灵气！”
<br/>　　说着，叶尘手掌一翻，直接从乾坤镯中取出了几枚灵石，然后随手朝四周一丢，这些灵石，顿时便散布在整个院落的个个角落。
<br/>　　叶尘双手一阵虚点过后，
<br/>　　“阵起！”
<br/>　　轰！
<br/>　　整个院落微微一阵，
<br/>　　滋滋滋~~
<br/>　　不但四面八方的灵气狂涌而来，院落之中的流水、绿竹之中，也开始有灵气升腾起来。
<br/>　　一转眼的功夫，整个诺大的院落之中，已经是灵气缭绕，恍若仙境。
<br/>　　看的楚飞烟和萧若曦两人，一阵目瞪口呆。
<br/>　　“飞烟，从现在开始，全力采购各种灵石美玉，我要将我们狂仙门，打造成真正的修真门派！”
<br/>　　叶尘一脸意气风发道。
<br/>　　尹百川和姜长老的死，让叶尘对逍遥山庄的众弟子，多少有些愧疚之心，自然要想办法补偿一下。
<br/>　　而且，他也早就打算，以逍遥山庄的弟子为班底，打造出一个震慑全球的宗门势力。
<br/>　　楚飞烟听到这话，顿时欣喜不已，连忙道：
<br/>　　“好！我这就让人去准备！”
<br/>　　楚飞烟和萧若曦离开之后，叶尘直接走入炼丹房，从乾坤镯中取出炼制养魂丹的丹药，又取出一尊炼丹炉，准备开始炼丹。
<br/>　　然而就在这时，吞天鼎器灵奥维斯的声音陡然响起，
<br/>　　“主人，你这炼丹炉也太差了点吧，还不如用我呢！”
<br/>　　叶尘顿时一愣，
<br/>　　“你还能炼丹？”
<br/>　　奥维斯傲然道：
<br/>　　“何止！主人，我不但可以炼丹，还能炼器呢！这天底下，我不能炼化的东西，还真不多！”
<br/>　　“炼器？”
<br/>　　叶尘顿时两眼一亮，
<br/>　　之前和叶无伤交战之时，草雉剑和十拳剑这两件圣器，全部都被砍断，他正为此事犯愁呢！
<br/>　　若是用两把剑的残骸，打造一件新的兵器，或许会是一个不错的选择……
