　　第562章
<br/>　　楚云生脸色不是很好看，作为一个读书人，虽然在琅琊城憋屈了三年，但这种血淋淋的局面却是第一次亲眼目睹。
<br/>　　张明理将楚云生引到山间一座小亭，直到楚云生脸色勉强恢复正常，他方才开口道：“公子请用茶。”
<br/>　　楚云生双手将茶杯捧起，手腕颤抖，指尖发白。
<br/>　　张明理注视着他的一举一动，用温和的声音道：“公子勿忧，我们已调查清楚，这件事和楚公子并无关系。”
<br/>　　楚云生深吸一口气，压下忐忑的心情，老实说，他刚才的确被吓着了。
<br/>　　“你们要我做什么？”
<br/>　　张明理怔了怔，虽然一闪而逝。他笑笑道：“公子想多了。那程贼作恶多端，对他，以及对他的同伙，我们绝不会心软。但对于楚公子这样的清白人，我们定不会有半分不敬。”
<br/>　　楚云生像是松了口气，手中的茶杯也放下了。
<br/>　　“恶贼不除，天下怎平。”张明理道：“我辈读书人，以天下为己任。公子好好想想，这程大雷有没有与你说过什么话，帮你又是什么目的，这些话，公子要原原本本的告诉我们。若能除了此贼，天下当记公子的首功。”
<br/>　　楚云生沉默着，想程大雷曾与自己说过什么。老实说，程大雷并未告诉过他什么，一些扯淡的话，似乎也算不上秘密。
<br/>　　忽然，他长身而起，双手抱拳，向张明理深施一礼。
<br/>　　“张先生，你的话我都已听见，先生的心意我也都明白。可我当真是无可奉告，莫说没什么我知道的秘密，便是有我也是无可奉告。程先生与我有恩，他是贼也好，是官也罢，终究是与我有恩的。为人滴水之恩，当涌泉相报，且不可恩将仇报。”
<br/>　　这番话说得结结巴巴，态度倒是很坚决。张明理脸上的笑容僵住，道：“你是要帮那恶贼？”
<br/>　　“若有武艺，就算帮不了程先生，也得救刘先生一救。可惜，百无一用是书生。我能做的或许只有不与书院合作，借此表明态度。”
<br/>　　张明理如同第一次真正认识楚云生，道：“你可想清楚了。以你在数算上的天赋，到任何一路诸侯门下，都会被奉为上宾。若再在我书院修行一二年，未来天下定然有你一席之地。如果你选不对路，世间未扬名便夭折的天才可是不少。”
<br/>　　“道不同不相为谋，我的态度以表明，张先生并非我的同道人。”楚云生将双手负在身后：“张先生是不是也要将我丢进大牢中受那酷刑，我怕疼，还请下手轻些。”
<br/>　　张明理盯着楚云生，上下将他打量好几遍，最后竟失魂落魄的摆摆手，独自起身离开。
<br/>　　楚云生倒也准备好了去大牢受刑，虽然这似乎也无济于事，但读书人么，总要有些死给你看的精气神。他倒是也没想到，张明理竟然没难为自己。
<br/>　　或许，是自己没有被难为的资格。
<br/>　　半晌，他发出一声苦笑，穷书生还真是穷书生呐，穷得一文不值。
<br/>　　他并未离开，而是又在亭间坐下。张明理走了，自然不会有小童留下来伺候他。他自己给自己倒满茶，书院的茶，当然是好茶，平日楚云生可是喝不起。
<br/>　　然后，他开始想。
<br/>　　如今刘发财被扣在大牢，承受严刑拷打。程大雷消失不见，当还没有离开琅琊城，书院已派出搜捕的人，这是楚云生目前掌握的所有信息。
<br/>　　他开始想，任一条条线自由发展，终所有线索撞在一起。而自己，在这个局中，实在是微不足道的存在。但这枚微不足道的棋子，已试图影响最后的结果。
<br/>　　但楚云生实在太微不足道了，掌握的情报又实在有些少。蝴蝶闪一闪翅膀，或许真能引起一场风暴，然而，在风暴中，蝴蝶其实影响不了什么。
<br/>　　那么，就继续想。
<br/>　　楚云生不是个很有口才的人，诗情画意更是一窍不通。但他的『算』真的很厉害，他最擅长的一件事就是——想。
<br/>　　继续想。
<br/>　　张明理离开后，与宋寒之碰在一起。这件事对书院来说，当真算一件大事。但这样的大事，也未必需要夫子出面。
<br/>　　张明理与宋寒之是主持这场大局的人。
<br/>　　他们来到了书院的大厅，宋寒之拱手与大厅内的诸人见礼。
<br/>　　“大家当已都知道消息，那姓欧的便是我们要找的程大雷。他泼天大胆，竟然混进书院，如今行迹暴露，已经逃了。当然，他绝逃不出琅琊城。而他既然来到此地，我们便绝没有让他活着离开的道理。”
<br/>　　在场诸多人，都是东海一地参加伏魔大会的高手。因为盗书的事情，伏魔大会被推迟了。如今，算正式进入伏魔大会的节奏。
<br/>　　马猛开口道：“该如何做，宋先生安排就是。”
<br/>　　宋寒之点点头：“书院已开始全城搜捕，但那贼人的本事毕竟了得，就算能找到他，也未必能擒住他。所以，必须找一个绝地，摆下天罗地网，来一计引君入瓮，然后再关门打狗。”
<br/>　　马猛皱起眉头，江湖人么脑子一般不太好使：“他会来么？”
<br/>　　“我们已擒到他一名同伙，以他为饵，当有七成把握。”宋寒之如此说，心里倒不是很有把握，很难相信，义气这种东西，当真在江湖人之间真的存在。
<br/>　　“那绝地在何处？”马猛问：“书院么？”
<br/>　　“书院未必不可，但也得提防贼人逃了。”宋寒之深吸一口气：“所以夫子老人家将地方定在了岛上。”
<br/>　　海岛上……
<br/>　　大厅内诸人皱起眉头，那的确是一处绝地，任程大雷三头六臂，只要他没有飞天遁地的功夫，就绝对逃不了。
<br/>　　宋寒之站起身，抱拳道：“九江城马老爷子，立地太岁孙大哥，徐州城的周兄弟……到时候还请诸位一同出手，替天下杀了此獠。”
<br/>　　“宋先生放心便是。”大厅内的人都站了起来，爽朗答应。他们都是东海附近的高手，人人都算一地枭雄。
<br/>　　“到时候，也要见识书院剑阵的威力。”马猛笑道。
<br/>　　“这个自然，这个自然。”
<br/>(本章完)
