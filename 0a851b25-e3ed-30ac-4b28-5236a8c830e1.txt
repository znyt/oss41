　　“你知道的，少阳很爱她，对不对？”
<br/>　　“我……”瓜瓜急的快流汗了，“那个，我只知道，老大很爱你啊，对小九的话……好像也挺喜欢的，嫂子你不要吃醋啊，老大从来没想过跟小九怎么样，他一直都想跟你在一起，想和你结婚的。”
<br/>　　“是啊，我们本来都快要结婚了的……”
<br/>　　芮冷玉的头低了下去，神情中明显带着悲伤。
<br/>　　后卿忍不住提醒了一声：“冷玉。”
<br/>　　芮冷玉仿佛想起了什么，神色变得决绝起来，对瓜瓜说道：“如果小九复活，少阳跟她在一起，一定也会很幸福。”
<br/>　　瓜瓜当场怔住，嘴巴蠕动着，想说什么来劝劝她，但又不知道该怎么说，他的心性就是一个孩子，对于成人所谓爱情，明白是怎么回事，但毕竟没什么深入的体会，这种问题上，他也不知道怎么说。
<br/>　　“你是少阳最亲近的人，连你也赞同，可见完全没问题的。”芮冷玉强撑着微笑说道。
<br/>　　“不不，不是这样，只是……我不知道怎么说。”瓜瓜着急地挠着脑袋。
<br/>　　“好了，我找你来，就是想让你告诉少阳……将来如果你见到他的话，告诉他，不用来找我了，我留在这里不走了。”
<br/>　　“什么！”瓜瓜震惊，“为什么？”
<br/>　　“我……我要嫁给他。”芮冷玉看了一眼后卿，吸着气说道。
<br/>　　瓜瓜当场就石化了，甚至以为自己听错了，指着后卿说道：“你……要嫁给他？”
<br/>　　瓜瓜似乎这回过神来，一下子跳了起来，“不是吧！你居然要嫁给他！我说嫂子，你没搞错吧！你嫁给他，老大怎么办啊！”
<br/>　　“他有小九，足够了。”
<br/>　　“可是……哎呀，你喜欢他吗？”
<br/>　　芮冷玉微微低头，调整了一下情绪，说道：“阿呆很喜欢我。”
<br/>　　“喜欢你有什么用，问题是你喜欢他吗？”
<br/>　　后卿听了这话，脸上现出阴鸷的表情，但是强忍住没有发作。芮冷玉沉吟了一下，道：“很重要吗？我只有留在这里，才是最安全的，有他保护我，我可以永远住在这里……”
<br/>　　瓜瓜还想说什么，芮冷玉道：“行了，你不要说了。我找你来就是告诉你这件事。你去告诉叶少阳，我嫁给阿呆了，我的魂魄，会在尸族永远住下去，他……永远都不要再来找我了，我也永远不想见他。”
<br/>　　“不！我不相信！”瓜瓜叫起来，“你是我主母，是我老大的媳妇，怎么可以跟别人结婚！绝对不行！”
<br/>　　“这是我跟少阳的事，用不着你来管，你只要把我说的话转告给他就行了。”
<br/>　　“不，就算老大同意，我也不同意！！”瓜瓜暴跳如雷。
<br/>　　芮冷玉和后卿，对他的表现都有点吃惊。他们不明白，瓜瓜在某些方面是一根筋的，在他的认知中，芮冷玉跟叶少阳就是一对，一向都是一对，是铁定要在一起的，芮冷玉突然说不在一起了，他无论如何是不能接受的。
<br/>　　猛然间，瓜瓜想到了什么，转头恶狠狠地朝后卿看去，吼道：“一定是你逼她的，是不是，你把她关在这里，然后逼她，是不是！”
<br/>　　后卿微微一笑，道：“你觉得，她是容易被人逼迫就范的吗？你的主人叶少阳，保护不了她，所以她才会留在这里，跟我在一起，我可以保护她。你懂不懂。”
<br/>　　瓜瓜捏着拳头，目眦欲裂。
<br/>　　“瓜瓜！”芮冷玉怕他动手，呵斥了一声，“你既然知道了，就快走吧。路上……你注意一点。”说到最后几个字的时候，芮冷玉特意用了重音。
<br/>　　瓜瓜还想说什么，后卿猛然上前，一只手抓住了他的脖子，带着他飞出了洞府。芮冷玉看着他们离去，瘫坐在沙发上，泪流满面。
<br/>　　大约过了小半个时辰，关闭的洞府再度打开，后卿走了进来。
<br/>　　这时候芮冷玉情绪已经缓和下来，看到后卿走近，说道：“他回去了？”
<br/>　　“放心，我说过不会杀他的。两军交战，互有死伤，杀他一个不多，没这个必要。”后卿一直走到芮冷玉面前，低头看着她，说道：“刚刚，太阴山的右君，派了一个使臣过来见过，让我把你交还给他们，只要我答应，他们愿意派兵来帮我对付空界这些家伙。”
<br/>　　芮冷玉看也没看她，说道：“你没答应。”
<br/>　　“当然没答应，就算是用整个太阴山来换你，我也不会换的。江山我要，美人我也要，只要我打下空界，将来就算是太阴山，又能奈我如何？”
<br/>　　芮冷玉这才抬起头，盯着他看了一会，说道：“女魃他们同意你这么做，他们不怕交恶太阴山？”
<br/>　　“恰恰相反，他们很支持，因为我这么做，不会得罪他们，因为你在我手中。他们只会用更多的筹码来换，这也是我跟女魃所说的，留下你在这里的原因。”
<br/>　　说完，后卿笑了一笑，对芮冷玉说道：“刚才，我可是配合了一次，你怎么谢我。”
<br/>　　一句话，让芮冷玉又陷入了之前那件事情中，叹了口气，喃喃道：“不知道少阳会不会相信。”
<br/>　　“他知道你真嫁给了我，或许会放弃吧。”
<br/>　　芮冷玉摇摇头，“以他的性格，应该不会，不过，倒是先算提前有了铺垫，将来总是要好一些。”
<br/>　　后卿笑道：“你想用这办法来让叶少阳死心，自己却失去了名节，何苦如此。”
<br/>　　芮冷玉没有回答。
<br/>　　“你还没说，你怎么谢我？”
<br/>　　“你总不能让我真的嫁给你吧，就算我愿意，你又怎么跟女魃交代？”
<br/>　　后卿没有再说什么，他知道，芮冷玉不会真心想嫁给自己，而且女魃那边也绝不会同意。至少目前绝不会。
<br/>　　“等我打下这空界，再与你慢慢处理这件事，总之……”后卿声音放缓，柔和地说道，“我相信你一定会喜欢上我，到那个时候，一切都是有办法的。”
<br/>　　不明年代，一片无人的山林之中，有一汪清泉，清泉的四周，是用青石板搭建而成，是一个八角形，每一面的石板上都刻着不同的道纹。
