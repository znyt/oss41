　　哗！
<br/>　　叶尘这话一出口，众人顿时再次哗然，
<br/>　　眼前的这位老者，乃是烈日宗长老，堂堂金丹境的强者。
<br/>　　这神秘青年虽然高深莫测，可是以他刚才出手的情况来看，最多也就是金丹境，竟然敢口出如此狂言？
<br/>　　“小子！你这是在找死！”
<br/>　　灰袍老者气得双眼欲裂，若非南黎川没有开口，恐怕他早就对叶尘出手了。
<br/>　　南黎川两眼微微一眯，在叶尘身上打量了一番，冷冷道：
<br/>　　“小子，本公子不管你是什么来头，如今在我烈日宗的地盘，就算现在把你杀了，也不会有人知道，所以我劝你最好不要多管闲事！”
<br/>　　叶尘却仿佛没有听到一般，已经开始慢悠悠的数着数，
<br/>　　“七……八……九……十！”
<br/>　　当‘十’字落下，叶尘似笑非笑的抬起头，
<br/>　　“既然不走，那就，全都留下吧！”
<br/>　　说完这话，叶尘猛然一掌拍出，
<br/>　　轰隆！
<br/>　　天空陡然出现一道金色巨掌，朝南黎川等人狠狠拍落下去。
<br/>　　“什么！！”
<br/>　　直到这时，南黎川和灰袍老者，这才开始慌了，
<br/>　　“元婴境？不！这怎么可能！他明明只有金丹境的气息，怎么会有这么强的实力？”
<br/>　　两人百思不得其解，可惜他们永远都不会有答案了。
<br/>　　纵然如今叶尘只能发挥四五成的实力，也不是区区一个金丹境初期修士可以抵挡的。
<br/>　　这一掌落下，南黎川连同灰袍老者，乃至其余十多名烈日宗弟子，连丝毫反抗和逃跑的机会都没有，瞬间全部被拍成了一堆肉渣！
<br/>　　嘶~~
<br/>　　就连云灵儿等人看到这一幕，也不由得纷纷倒吸了一口凉气，望向叶尘的神色，也越发的恭谨。
<br/>　　“前辈救命之恩，我等感激不尽，如若前辈不嫌弃，请到我流光派一叙，让我等略尽地主之谊！”
<br/>　　云灵儿再次开口。
<br/>　　“也好！”
<br/>　　叶尘略微想了想，倒也没有拒绝。
<br/>　　前世，流光派对他有恩，他也勉强算是流光派弟子，既然人到了这儿，理当回去看看。
<br/>　　而且，他在天狼星域不会待太久，前往其他高级星域，则需要先找到传送阵，刚好可以打听一下此事，说不定流光派之中，便有通往其他星域的传送阵。
<br/>　　云灵儿原本只是随口邀请，根本没有指望叶尘会答应，毕竟像对方这样的强者，应该对流光派这种小门派不会太感兴趣，万没想到叶尘居然如此干脆的答应下来。
<br/>　　“太好了！前辈愿意驾临我流光派，是我派的荣幸！”
<br/>　　接下来，众人收拾了一下，将南黎川等人的尸体销毁干净，这才动身离开。
<br/>　　云灵儿等人才只有筑基境，因而只能御剑飞行，而叶尘则可以踏空而行，一步迈出，便是数里的距离，缓缓跟在众人之侧。
<br/>　　“贵派的玄离真人，不知可在派中？”
<br/>　　闲来无事，叶尘便主动和云灵儿交谈起来。
<br/>　　云灵儿听到这话，顿时美眸一亮，满脸喜色，
<br/>　　“前辈可是和家师认识？”
<br/>　　叶尘淡淡一笑，摇了摇头，
<br/>　　“不认识！不过尊师的大名，我倒是有所耳闻，只是未曾一见！”
<br/>　　听到这话，云灵儿脸上微微露出一抹失望之色，道：
<br/>　　“那实在不巧，家师云游去了，并不在派中，否则那烈日宗，也不敢如此明目张胆对我们出手！”
<br/>　　听到这话，叶尘也微微有些失望，
<br/>　　虽然这一世的玄离真人，并不认识他，不过前世的恩情，叶尘一直记在心里。
<br/>　　两人又聊了一会，叶尘对天狼星域的状况，也有了一个了解，
<br/>　　基本上和前世的情况差不多，整个天狼星域，一共有数十颗星球，而且他们如今所在的天狼星，是天狼星域的主星。
<br/>　　别看这天狼星只是二级星域，即便是那些数十颗附属星球，也都比地球大得多，天狼星更是地球的数百倍！
<br/>　　整个天狼星上，一共有三十多个门派势力，而烈日宗自然是首屈一指，流光派和另外两个门派，则紧跟其后。
<br/>　　也正因为如此，烈日宗一直视流光派为眼中钉，生怕被流光派取代了龙头老大的地位，所以才会趁流光派核心弟子试炼的机会，要将他们一举铲除。
<br/>　　“叶前辈，您连烈日宗都不放在眼里，想必应该是来自外星域的大宗门了？不知叶前辈您师承何方啊？”
<br/>　　流光派其中一名男弟子，忽然开口询问道。
<br/>　　众人听到这个，顿时纷纷望向叶尘，全都露出一抹期待之色。
<br/>　　以叶尘如此年轻的容貌，具备如此强悍的实力，必定是一个天才妖孽无疑。
<br/>　　虽然，修真界有很多驻颜之术，可是一般来说，在修真界一直流传着一个铁律：
<br/>　　在同一级别的强者之中，容貌越是年轻的修士，就越可怕，因为他们既有可能，是某个大宗门的绝顶天才！
<br/>　　然而，
<br/>　　等叶尘一开口，众人全都愣住了，
<br/>　　“我是散修，没有宗门！”
<br/>　　很快，叶尘可以明显感受到，众人对他的态度，除了云灵儿没有太大变化之外，其他人全都冷淡了许多，再也没有了方才的热切和敬畏，甚至有些人，已经开始暗暗摇头，面露凝重之色。
<br/>　　对于众人的想法，叶尘自然心知肚明，
<br/>　　所谓散修，就是无门无派之人，没有门派，就意味着没有靠山，只要对方实力比你强，谁都能踩上一脚。
<br/>　　纵然，叶尘之前表现出的实力，极有可能是元婴强者，可是公然斩杀烈日宗长老和南黎川，还是有些太冲动了！
<br/>　　要知道，那南黎川，可是烈日宗宗主南日山的宝贝儿子。
<br/>　　区区一介散仙，竟然敢和天狼星域的霸主叫板，这简直就是以卵击石啊！
<br/>　　“叶前辈也是为了救我们，所以才杀了南黎川，大伙一定要牢记，切不可将今日之事告诉任何人，否则我云灵儿绝不会放过他！”
<br/>　　云灵儿沉吟片刻之后，忽然脸色一肃，冲着其余几人告诫道。
<br/>　　众人自然纷纷称是，不过望向叶尘的目光，已经全无敬意，甚至还带着淡淡的抱怨。
<br/>　　毕竟，倘若此事一旦被揭发，叶尘固然要倒霉，他们也要跟着遭殃。
<br/>　　叶尘自然也察觉到了众人的变化，但也都在预料之中，
<br/>　　修真界之人，大多天性凉薄，像云灵儿和玄离真人这样热心肠的人，终究还是少数。
