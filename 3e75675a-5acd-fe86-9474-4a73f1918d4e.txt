　　“天！”
<br/>　　仓颉大喝一声，同时用手中的刻刀在眼前的巨大骨匾之上刻下一个奇怪的字符条纹，如天之浩远广阔。
<br/>　　“轰！”
<br/>　　与此同时，天地一震，苍穹之上，一个与骨匾上一模一样的奇怪符文悬挂起来，正是天字。
<br/>　　“地！”
<br/>　　仓颉再次大叫道，同时手下不停，继续刻着奇怪符文条理，厚重载物，苍穹之上的天字旁，再次出现一个地字。
<br/>　　“玄！”
<br/>　　“黄！”
<br/>　　“日！”
<br/>　　“月！”
<br/>　　“洪！”
<br/>　　“荒！”
<br/>　　“……”
<br/>　　仓颉每一次开口，便会在骨匾上刻下一个奇怪的字符，同时苍穹之上也会出现一个同样的字符，能够令洪荒所有人清晰看见。
<br/>　　此等手段，真是匪夷所思！
<br/>　　“咦！”
<br/>　　道音天传，声音传遍整个洪荒天地，那高高悬挂于苍穹之上的大字，好似天地至理一般，巨大的动静，瞬间便引动了无数闭关的大能。洪荒众多强者纷纷探出神念，汇聚到人族上空，关注此刻所发生的事情，就连圣人也不例外。
<br/>　　文字，是文明传播中必不可少的媒介，此刻是文字的初生，代表的是一种全新的大道。正所谓，天之道，损有余而补不足。
<br/>　　天道不全，人道补之！
<br/>　　开天三道，天、地、人，唯缺人道，女娲造人，致使人道大兴，玄天创武道，补充武道及人道，人道再兴，如今仓颉造字，创文道，人道又一次大兴！
<br/>　　天地人，三才平衡！
<br/>　　初创的文道，如之前玄天开创武道一般，会传遍天下，此刻是文道最好领悟的阶段，每一个字符，都代表着文道之中的一种道，以后便没有这样的机会了。
<br/>　　众多洪荒大神都在纷纷领悟这种新出现的文道，就连诸圣也不例外，元始更是第一时间关注仓颉。
<br/>　　元始大道，万道归元。文道作为一种能与武道睥睨的大道，自然也不例外。
<br/>　　这样的道，自然是多多善意，领悟的越多，他的元始大道自然也就越强。
<br/>　　元始眼中道光闪烁，他没有去看苍穹之上显现出来的大字，而是去看仓颉亲手刻下的字符。在仓颉的刻刀之下，元始感受到一种不一样的天地至理，一种玄奥的力量围绕在那些字符之间。
<br/>　　悟道古茶树悬于元始脑后，造化之光，智慧之光散于元始心间，助他去领悟大道。种种玄妙至理流淌于心间，化为一道道神秘符文，被他的元始大道吸收。
<br/>　　一个字符，便是一种道，种种字符，汇聚成一条浩瀚的大道，那便是——文道。
<br/>　　文道又汇聚元始字符之中，为其所吸收，那是一条更加壮阔的道——元始！
<br/>　　一个刹那，或者是无数年，又或者是永恒！
<br/>　　元始无数念头齐动，快速领悟着其中的道理，这天地间，除了仓颉本人之外，恐怕就是元始收获最多了。
<br/>　　昆仑山，万寿山，西方须弥山，太华山，天庭，地府，幽冥血海，北冥，首阳山……
<br/>　　洪荒各处名山大海，都显现出浩大惊人的气势，显然是洪荒顶级强者都出动了。特别是人族祖地首阳山，人祖殿，女娲庙，武宫，都显现出惊人的气势。
<br/>　　人族的气运之海在翻腾不休，浩荡的气运汇聚而来，气运在壮大，在沸腾，在兴奋。一枚五方大印镇压其间，镇压着人族的气运，那是人族气运至宝——崆峒印！
<br/>　　“人族大兴！人族大兴啊！”
<br/>　　好多人族祖老看见这一幕都兴奋不已，直叹人族大兴之势势不可挡。这些都是人族神话时期就存在的，好多都参与过三族大战，如今都留在人祖殿中，闭关修炼，其中大多数都是转轮皇者，堪比准圣强者。
<br/>　　有熊氏部落，轩辕正在与众多仙神商议与蚩尤的最终决战之事，见到这惊天动地的一幕后，直接走出房屋，大叫道：“来人，速与我去拜访仓颉！”
<br/>　　“是，共主！”
<br/>　　九黎部落，魔神蚩尤同样走出来观看，直叹人族好运。
<br/>　　部落之中，仓颉依旧在不断刻字，只见那骨匾之上，刻着密密麻麻的符文，随着刻录符文越多，他就越是吃力，但他身上的气息也越来越强。
<br/>　　“轰隆隆！”
<br/>　　九天之上雷霆炸响，无边雷云汇聚而来，其中蕴含着毁天灭地的威能，好似是要毁灭万物一般。
<br/>　　却是仓颉造字，每个字符代表着一种天地至理，因此泄露了天机，上苍欲将之毁灭。
<br/>　　“哗啦——”
<br/>　　一道耀眼的裂痕闪现，划破天际，向着仓颉直直劈了下来。
<br/>　　唰唰！
<br/>　　数道神光冲天而起，那是仓颉刻录的字符中冲出的神光，其中带着一股难言的沧桑古老之感，咔擦一声，与天上降下的雷霆相撞，一黄一白，相互交织，蕴含着生死之机，毁灭，重生，造化，凋零……
<br/>　　砰砰！
<br/>　　神光与雷霆相撞，兹兹兹声不断响起，最终，化为一片灵气消散不见。
<br/>　　“轰隆隆！”
<br/>　　上苍震怒，颇为不满，降下更加强大的雷霆，欲毁灭这些字文。仓颉猛然加速，快速刻录字符，同时在他的口中，吐出一个个神光璀璨的大字，冲向九天，阻挡上苍降下的雷罚。
<br/>　　大日横空，月华冲霄，雷龙暴怒，暴雨如瀑，电若火花……
<br/>　　一道又一道的异象冲天而起，内蕴强大气机法则，气势恢宏，大气磅礴。
<br/>　　雷霆越来越强，方圆数万里的灵气席卷而来，仓颉脸色苍白如纸，汗如雨下，期间，好几次都差点陨落于雷霆之下，还是元始出手，控制了雷罚，仓颉这才险而险之的渡过去。最终造字两千九百九十八。
<br/>　　“轰隆隆！”
<br/>　　上苍更加暴怒了，风雷怒吼，电光破天，无尽雷霆汇聚而来，演化出一片世界，世界之中，又演化出各种惊人异象，有浩瀚星域，有缥缈仙宫，有九重天地，有雷龙翻滚，四象飞舞……异象惊人！
<br/>　　仓颉于道台之上站了起来，他岿然不动，任由无边雷霆落下，却是不能伤他分毫。紫金色的雷光变成一道巨大无比的雷柱，携着毁天灭地的威势，直直地朝着仓颉轰炸下来。
<br/>　　天地昏暗，万籁寂静，大日隐蔽，群星退却！
<br/>　　仓颉表情十分严肃，他抬头仰天，万象雷霆深处，在骨匾之上快速刻下“文道”两个大字。

<br/>　　“今日我仓颉欲以文证道，开创文道修炼，证道文圣！”
<br/>　　仓颉猛然大喝，声若雷霆，响彻九霄。
<br/>　　“轰轰轰！”
<br/>　　仓颉那苍老的身躯变得仿佛凡人一般，此刻，他身躯中的气息忽然强横起来，而且还在不断攀升，不断升腾，似乎永无止境。

<br/>　　三千古字，犹如三千大道一般，最终一个个古字钻进“文道”两个字符之中。两字变得越来越强大，气势也越来越磅礴。
<br/>　　最终，只剩下一文一道两字。仓颉一手掌文字，一手握道字，演化万道。
<br/>　　轰！
<br/>　　一道与武道完全不同的气势从仓颉体内升起，天地正气汇聚而来，从天灵盖冲天而起，惊天动地！
<br/>　　“日月入我怀，山河藏我心；
<br/>　　天地生我胸，万物育我灵。
<br/>　　以文证道，当封文圣！”
<br/>　　此刻，天地雷霆尽散，却有一道比雷霆还要浩大，正气，神圣，璀璨而夺目的光柱，正缓缓消散。
<br/>　　这一刻，天地失声！
