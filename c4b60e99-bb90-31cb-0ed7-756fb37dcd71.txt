　　响亮的声音一出，整个狼牙便开启了最高级的防守模式，一名防守队成员立即爬上最高的那个建筑，一旦有任何不明物体靠近，他便会开启时光炮弹时刻准备开炮。
<br/>　　顾凉晨第一个冲出屋子，与狼牙所剩不多的防守队成员聚集在一起。
<br/>　　“方位？”顾凉晨一上来就强势出声，丝毫看不出曾在外游荡了好几年，那一身下意识的反应及狼牙特有的气场，一点都没落下。
<br/>　　快速聚集的队伍一开始都愣了愣，长时间的安静和平淡，让他们都快忘记了曾经的紧张感，也都快记不起被长官如此冷静问话，是多久之前。
<br/>　　但他们都以最快的速度反应过来，一名成员走上前语速极快的吐出两个字：“禁地。”
<br/>　　“什么！？”这回换成了顾凉晨呆愣震惊。
<br/>　　禁地，敌袭来自禁地？
<br/>　　“快速汇报情况！”顾凉晨话落之后，便直接朝着禁地所在地狂奔。
<br/>　　一队人员立即跟在其后，带队的人与顾凉晨并肩而行，边跑边说道：“是尚九娣城主发出的警报，人已死在白允儿手中，期间我们所有防守人员、设备以及侦查，都没有发现此人的混入。”
<br/>　　话说的不明不白，也没说敌人有多少，是否只有一个，目前正在彻查哪些地方。
<br/>　　但这看似不清楚的话语之中，却透露给顾凉晨一个惊人的信息，那就是这个混入的人，有能力躲过他们狼牙引以为豪的侦查。
<br/>　　既然一个人能做到，那就代表还有人能做到！
<br/>　　所以……
<br/>　　“速度汇报给陆玟丞，调集所有人去禁地！快！”顾凉晨二话不说的下令，甚至一阵冷汗瞬间浸·湿了后背。
<br/>　　第一个敌袭情况是发生在禁地，那么对方的目标便显而易见。
<br/>　　楚涵有危险！
<br/>　　刷刷刷！
<br/>　　一道道快速至极的身影快速在基地内穿梭，一切有一战之力的人全部拿起了武器，通红着双眼朝着禁地狂奔。
<br/>　　尤其是那一队跟着顾凉晨的防守队成员，他们很清楚顾凉晨平日里是个什么样子，冷冰冰的样子可不比白允儿级别弱，也就陆玟丞死不要脸才能跟这个大冰山混熟。
<br/>　　而能让一向扑克脸的顾凉晨如此大惊失色，足以说明敌袭的目标，那个谁都不允许靠近的木屋里住着的人，以及让他们都没能发现的可怕敌人想杀的人……
<br/>　　只能是他们狼牙的最高长官，楚涵。
<br/>　　狼牙没有傻·子，哪怕再笨的人在一次又一次的训练灌输，以及熟悉到铭记于心的种种套路里，也能训练到分析能力和应变速度惊人，更别说这种显而易见的推理。
<br/>　　于是想通之后的众人，一下子便心态爆炸，速度爆发的史无前例，甚至超过了他们这一队人的最高冲锋记录。
<br/>　　在警报响起的同时，陆玟丞也来到狼牙最中心的建筑里，这里是狼牙的中枢所在，也是整个狼牙地界情报最为集中的地方，很多部门都因为南方的战事而搬走，但情报部总有最深层的一部分留下。
<br/>　　“怎么回事？”他大步走进屋，开口便问。
<br/>　　“中将，我们刚想找您。”一人走上前，手持一份刚拿到的情报，神态复杂道：“禁地有人闯入，已被白允儿当场杀死，同时尚九娣以城主身份拉响了最高级别的警报。”
<br/>　　陆玟丞一下子有些懵，下意识问道：“什么？哪？”
<br/>　　“禁地，有人逃过了我们的全部防守侦查，悄然无息的靠近到木屋十米处。”那人又说了一遍，并且说的更加细致，紧接着也不给陆玟丞反应时间，继续道：“我们已经初步分析过了，目前推断狼牙内还有敌人尚存，并且数量不少……”
<br/>　　“你等一下，我先……”陆玟丞直接打断对方，并且想要转身就走。
<br/>　　“中将请留步，顾凉晨少将已经带着狼牙所有的战力部队前往禁地。”那人快速出声，将陆玟丞留住，又有些缓解压力的开口道：“反而是眼下的最深层情报部已经无人把守，所有人都跑去禁地了，您还是待在这里，并担心一下万一此处被人发现，狼牙所有秘密被公布于世该怎么办吧。”
<br/>　　最后一句话让差点暴走冲动的陆玟丞瞬间冷静下来，他反手将门一关，并直接坐下取出纸和笔：“说，分析到哪一步了？”
<br/>　　情报部的人嘴角带笑：“只剩最后一步。”
<br/>　　“那就汇报，还废什么话！”陆玟丞皱眉呵斥。
<br/>　　那人递上一张分析表，道：“一共三·点，第一点敌人为何能瞒过我们的视线，根据当下末世纪元的各大势力情况来看，除了神秘家族的出手别无其他可能。”
<br/>　　“第二。”那人伸出两根手指，有理有据的说道：“肖家不可能，因为就一个肖七自保还来不及，高家不可能，因为对方最小的嫡系少爷与我们狼牙最高长官的关系，所以这些人的背后只能是白家和罗家。”
<br/>　　“第三，来者是否为神秘家族正统成员？我们分析的结果是否，因为如果是那些高贵的神秘家族成员，他们根本没有必要这么偷偷摸·摸，也不可能被白允儿当场秒杀吧？所以有可能是神秘家族培养的暗杀者，亦或是一些半血统。”
<br/>　　“至于数量，很抱歉，这超出了我们的能力范围，对方就像会隐身一样，除非自己出现跟我们的人打起来，否则根本察觉不到。”
<br/>　　陆玟丞听完后目光深邃：“很可怕啊……”
<br/>　　“但是中将，我总觉得还有什么被遗漏了，却一时半会儿想不起来。”那人忽然皱眉，带着一丝谨慎。
<br/>　　陆玟丞连忙将所有情报都再次翻找一遍，而后豁然抬眼：“叶永阳！如果对方的人中有白家派来的，那他们的目标就不仅是老大，还有叶永阳！”
<br/>　　那名情报部成员也当下一惊：“我们可没有能力保护两个人。”
<br/>　　陆玟丞紧抿着嘴唇，道：“让叶永阳来我这里，如果老大那里被攻破了……我们这里也没存在的价值，叶永阳同样如此。”
<br/>　　“是。”那人冷静回复，扭头就出去接人。
<br/>　　这一次，对方要鱼死网破，狼牙便孤注一掷！
