夏夕绾不知道他怎么突然说起这个话题的，不过她又不傻，今天陆子羡揭开了她脸上的面纱，肯定是喜欢她的了。

<br/>        纤白的手指蜷了起来，夏夕绾摇头，“我不知道…”

<br/>
<br/>        夏夕绾扭头看着他，“你告诉我这些干什么，我又不喜欢他！”
<br/>
<br/>        陆寒霆看着前方，劳斯莱斯幻影在他的掌心里开的十分平稳，他没有再说话，今晚的他格外的沉默反常。

<br/>        ……

<br/>        “你累么？”陆寒霆突然打断了她。
<br/>
<br/>        “什么？”
<br/>

<br/>
<br/>        夏夕绾摇头，“我一点都不倦…”

<br/>        “哦，解释这两个字你没有说倦，但是我听腻了，夏夕绾，我们离婚吧。”

<br/>        夏夕绾，我们离婚吧！

<br/>        这句话在夏夕绾的耳畔炸开了，她把手松开，整个人僵在了原地。

<br/>        陆寒霆转过了身，一双深邃的狭眸落在了她苍白的小脸上，“这一次我没有开玩笑，我们离婚吧，我会让律师起草离婚协议书，财产方面不会亏待你的，你尽快签字，我们把婚离了。”

<br/>        夏夕绾怔怔的看着他的俊颜，花了几秒才明白他是认真的，他是真的要跟她离婚。

<br/>        她一点都不明白，他为什么要跟她离婚。

<br/>        陆寒霆将薄唇勾出了一道讥讽的弧线，“为什么离婚你心情不清楚吗，自从娶了你，每天到不同地方捉奸都快变成我的生活日常了，别告诉我你跟陆子羡什么都没有发生过，你们搂抱在一起，他还揭开了你的面纱，是不是要我亲眼看到你们脱了衣服滚在床上你才肯承认，今天是陆子羡，明天又是谁…”

<br/>        “够了别说了！”夏夕绾垂在身侧的两只小手已经攥成了拳，眼眶通红的瞪着他，“我没有，你不要污蔑我，如果你真的不相信我，那我给你检查一下身体，看我那道膜还在不在！”
