　　霓虹国，某航空自卫队机场。
<br/>　　两架F-15J载弹起飞，两个小时前，霓虹国防卫省接到来自美军冲绳军事基地的电话，被告知，两小时后，会有一架A340客机携带最新的变异P病毒，进入共和国的防空识别区。
<br/>　　美军太平洋舰队司令詹姆斯?沃尔特希望霓虹国航空自卫队能给予配合，在这架客机进入共和国防空识别区之前，将它拦截下来。
<br/>　　对于被蛰龙抢走的这两瓶最新的变异型P病毒，詹姆斯还是心有不甘，自己这边不能直接下手，那就只能假借他人之手，而霓虹国就是最好的选择。
<br/>　　A340客机客舱中，坐在窗户边罗昊探头朝窗外看去，薄薄的云层下，能看到蔚蓝色的海面。
<br/>　　很快就能进入防空识别区了，罗昊看了眼放在自己脚边的那个背包，原先心中的紧张，也是放松下来。
<br/>　　“别太担心了，只要进入防空识别区就会有战机起飞护航。”云熙睁开朦胧的睡眼，看了眼坐在旁边座椅上的罗昊，嘟囔着说道。
<br/>　　罗昊点头“嗯”了一声，替云熙拉好盖在她身上的毯子，说道：“再睡会儿，等到了我叫你。”
<br/>　　可就在这个时候，一架喷涂着霓虹国航空自卫队标识的F15J出现在A340客机翼尖位置，跟A340靠的很近，即便是在客舱里面，罗昊也能看到悬挂在F15J机翼和机腹下的导弹！
<br/>　　载弹飞行！
<br/>　　罗昊心头“咯噔”一跳，立刻察觉到了不妙，站起身，快步朝着飞机另一侧走去，果然在客机另一侧也发现了一架载弹飞行的F15J战斗机！
<br/>　　客舱中的其他人也是被惊动了，云熙掀开盖在自己身上的毯子，站起身走到罗昊身边，问道：“发生什么事情了？”
<br/>　　罗昊指了指飞机外，紧随着A340客机F15J战机，说道：“霓虹国防卫省的飞机，载弹飞行。”
<br/>　　“是冲着我们来的？”云熙问道。
<br/>　　“准确的说，应该是冲着我们手里的新型变异P病毒来的。”罗昊沉声说道：“回到座位上去，我去驾驶舱。”
<br/>　　罗昊按下密码，进入驾驶舱，对机长问道：“我们似乎被人盯上了。”
<br/>　　“这群兔崽子！”
<br/>　　这架飞机的机长以前在空军某部队服役，战斗机飞行员出身，退役后投身于民航业，有着丰富的驾驶经验。
<br/>　　虽然离开了部队有二十多年，但是作为军人心中的热血，却还没有冷却。这次更是在得知要运送一批在外行动，并且携带有危险物品的特种部队回国时，主动请缨要求执行此次飞行任务。
<br/>　　只是坐在副驾驶上的副机长此时却是显得有些慌乱，就连说话都是有些结巴，“机、机长，我、我们现在要怎么办？”
<br/>　　“照常飞行！”机长冷静的回答道：“很快就进入我们国家的防空识别区了，给他们十个胆子他们也不敢有动作！”
<br/>　　“可、可他们是战斗机，我们是民航客机，如果他们……”副机长还是有些不放心。
<br/>　　“他们不会发射导弹的！”
<br/>　　罗昊显得很笃定。
<br/>　　但副机长显然还是不放心，说道：“你、你怎么确定他们不会发射导弹？只要一颗导弹，我们这些人就全完蛋了。”
<br/>　　“因为他们的‘爸爸’不会同意。”罗昊眯着眼睛说道：“而且他们的目标不是我们的命和这架飞机，而是我们带上飞机的东西。”
<br/>　　“可是……”
<br/>　　“行了！”
<br/>　　副机长原本还要再说些什么，可是话还没说出口，就被机长冷声打断，命令道：“你现在离开驾驶舱去客舱！就你这个熊样，老子如果还在部队，别说是跟我一起飞行，你连给我做地勤都不配！回去以后，我会跟领导要求把你调离，看着心烦！”
<br/>　　副机长张了张嘴，不过最终还是什么话都没有说，解开安全带，起身离开了驾驶舱。而就在这个时候，一架F15J做了一个桶滚，从A340客机机首上方翻转而过。
<br/>　　桶滚，很考验战斗机飞行员的驾驶技术，在进行桶滚的时候，两架飞机之间的距离非常近，一般只有几十米的距离，这几十米的距离对于高速运动的战斗机而言距离撞击只有“咫尺”，稍有不慎就是两机相撞，机毁人亡的下场。
<br/>　　所以只有有着丰富飞行经验的飞行员才敢做这种危险动作。
<br/>　　“妈的！”机长是那种火药桶脾气，一点就着，被F15J做了一个滚桶，立刻就怒了，骂道：“老子如果再年轻二十年，不，十年，就算是驾驶着我们国家的歼-8，我都能追着你跑！”
<br/>　　“少校。”机长关闭了飞行系统的自动驾驶，改为手动驾驶，说道：“我知道你们学习过航空器的驾驶，我现在需要你来担任我的副机长，没问题吧？”
<br/>　　“副机长？”罗昊想了想，对机长说道：“虽然我学习过航空器驾驶，但是我有这边有一个人比我更加符合当副机长。”
<br/>　　“蜂鸟？”机长问道。
<br/>　　“您知道他！？”
<br/>　　罗昊很是诧异地看着机长，没想到他会知道林晓阳。
<br/>　　机长微微一笑，说道：“我虽然离开部队了，但是部队里一些事情我还是知道的，这小子是我们国家难得一见的天才，不到二十岁的年纪就能飞出普加乔夫眼镜蛇机动，别说是在国内，就是在世界范围内都是少见的。刚开始他上飞机的时候，我还不敢肯定，现在听你这么一说，我确信了。”
<br/>　　罗昊转身打开驾驶舱的门，招呼林晓阳进入驾驶舱。林晓阳二话不说，起身就往驾驶舱方向走。
<br/>　　坐在前空乘座椅上的副机长还在郁闷中，看到林晓阳也要进入驾驶舱立刻就站了起来，说道：“飞机驾驶舱不是什么人想进就能进的！”
<br/>　　飞机上，除了机长，就属副机长最大。
<br/>　　林晓阳扫了眼副机长，看到他制服肩膀上的三条黄线，问道：“你今年多大了？”
<br/>　　“什么？”副机长有些发懵的看着林晓阳，随后回答道：“二十九，怎么了？”
<br/>　　“二十九就是副机长了，也算事业有成了。”林晓阳拨开副机长，径直往驾驶舱走去，说道：“我19岁就能飞普加乔夫眼镜蛇机动，20岁就能做出zero-knot！”
<br/>(本章完)
