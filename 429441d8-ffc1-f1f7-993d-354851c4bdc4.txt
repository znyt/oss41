　　第188章
<br/>　　“XXX“
<br/>　　程大雷裂开大嘴，嘴里飙出一句三字经。
<br/>　　“你看，你看，大当家你快瞧哦，又来人了，好家伙，这回人多，至少有五千。”徐神机指着戎族大营高喊。
<br/>　　“好啦，好啦，我已经看见了。对面来人，你有必要这么兴奋么。”程大雷抿起嘴，盯着戎族大营：“为毛要这么和我过不去。”
<br/>　　从十天前开始，金问道的大营，陆陆续续有新的势力到来。每天旌旗招展，各种各样的大旗融入戎族大营中。程大雷心里估算，目前对方的人数应已超过十万。当然，这仅仅是程大雷的估计，插的眼睛（林少羽）被人拔掉后，程大雷已经摸不准对方的情形。
<br/>　　过去的时间，戎族又发动过几次攻势，每打一次仗，程大雷就能收割一波恐惧值，然后赶紧兑换成火药，再用兑换的火药投入到下一次战斗中。
<br/>　　如此以战养战，还真被程大雷撑了下来。可他心里也明白，以十万打两千，你除非给我来一卡车98K，否则守住这里的概率约等于零。
<br/>　　看来，是时候离开了。
<br/>　　可程大雷绞尽脑汁也想不明白，自己已经露出了牙齿和肌肉，照道理来说戎族就该知难而退，可为何他们不仅没退，反而不断增兵，直到兵力达到一个极其恐怖的程度。
<br/>　　要说青牛山也是战略要地，可戎族好像从不考虑战略方面的问题。难道是因为自己杀了他们的人，他们要与自己不死不休，以戎族的尿性，估计也不是很在乎这些。打仗么，大多是为了利益。
<br/>　　莫非，是因为自己太过英俊，他们嫉妒自己的美貌？
<br/>　　哎，这也是我的错么，苍天啊，为何英俊的人就要比普通人遭受更多的苦难。
<br/>　　……
<br/>　　戎族大帐。
<br/>　　大帐中央是一张帅案，左右手各有四张椅子，上面都坐着人。如今坐在帅案后的已不是金问道，以他北蛮部九王子的身份，也不过坐在最末尾。
<br/>　　如此，可见这次会议的规格之高。
<br/>　　今日有资格在大帐内占一把椅子的，在北蛮部的身份都极其尊贵，像部落头领这种级别的根本没有资格入内。大帐内一共九人，就连伺候的小卒都无有。
<br/>　　他们是北蛮部的九位王子。
<br/>　　北蛮部的王子排序倒也不是按照长幼，而是按照被封王的时间次序。所以，大帐内出现奇怪的一幕，三王子还没成年，四王子已经胡子一大把。
<br/>　　“我本来要去打京州的，再有一个月必然能攻进京州，可父王非要调我过来这边，哼，如果不是你们办事不利，我何必受这颠簸之苦。”
<br/>　　六王子雷怒，此人天生神力，使一柄三百斤的大锤。本来这次他是沿着并州一线向京州进攻，为了防御他的大军，帝国在京州边界布下重兵。可当大兵枕戈待旦，怀着忐忑的心情等待雷怒的进攻时，雷怒竟然退兵了。直到现在，帝国朝廷还没琢磨明白是为什么。
<br/>　　“我那里何尝不是，我的队伍已经连破五城，眼看第六城要投降了，父王却突然招我过来这边。”
<br/>　　八王子烈木合，此人擅长箭术，百步之外能穿悬在树上的麻绳。
<br/>　　“呵，连破五城算什么，我那里已经……”
<br/>　　“你们知道我杀了多少帝国大将么，现在他们的脑袋都在坛子里封着，要不要我取过来让你们看看。”
<br/>　　……
<br/>　　北蛮部的九位王子，个个都有出奇的本事，不然也不会被北蛮王封为王子。比如，某日北蛮王吃饱了遛弯，忽然看到一个人翻跟头不错，一问还是自己的儿子。
<br/>　　『你的母亲就是那个胸很大的西域女人么，好，你以后就是翻跟头王子』
<br/>　　这次戎族各部落结成联盟，大举入侵帝国，北蛮王派出这些人，令他们各率一支大军沿着帝国国境线入侵。一路长驱直入，戎族大军铺天盖地而来，在幽并两州发生了很多场战争，帝国……打得很苦。
<br/>　　可是，这些人竟突然收拢起来，同时聚到青牛山。如此，肯定是缓解了其他战区的压力，可蛤蟆寨的压力却飙升到一个天文数字。
<br/>　　在大帐中你一言我一语，火药味越来越浓，戎族崇尚武力，崇拜强者，每次开会，吵着吵着打起来的情况很经常，谁把谁打服了，等下一次开会的时候，对方就老实了。
<br/>　　眼看今日大帐内又将发生这一幕，大王子挥挥手道：“好啦，好啦，父王既然派我们前来，我们大军集结，打下蛤蟆寨，取走父王要的东西就是。诸位对打破蛤蟆寨，有什么主意？”
<br/>　　大王子声音落地，发现大帐内瞬间安静几分。他心中一喜：自己如今的威信这么高么。
<br/>　　呼尔勒死后，北蛮部群王无首，几位王子谁也看不起谁。真要说起来，这九个人都得感谢程大雷，不是程大雷杀了呼尔勒，他们根本没有领兵的机会。
<br/>　　大王子仔细一看，发现事情并不是如此。大帐内还吵吵嚷嚷，说些『三日必破城』『呵，我下午就办了』之类话的，都是刚到，还没和蛤蟆寨交过手的，而已经和蛤蟆寨打过交道的人，此时都保持着安静。
<br/>　　“怎么，不好打？”
<br/>　　金问道一直没抬起头，此刻道：“那蛤蟆寨的首领程大雷极其狡诈，诡计多端。”
<br/>　　“还会掌心雷。”也在程大雷手中吃过苦头的五王子道。
<br/>　　“那是火药。”金问道嘲笑他的没文化。
<br/>　　“十万人打一座山寨，应该不会太困难吧。”
<br/>　　大王子皱皱眉头，俯视着帅案上一份青牛山的草图。
<br/>　　“如果城门无法突破，那从两侧悬崖呢？”
<br/>　　“我已经派人看过，山谷两侧的悬崖都在百丈以上，上面并没有可攀登之物，而且我族士兵的确不擅长翻身越岭。莫说攀不过去，就算真有几百人过去，怕也是送上门给人杀。”
<br/>　　“如此看来，智取是不行了，只能强攻。”大王子皱皱眉头：“十万人打几千人，还需要用什么计谋么？”
<br/>　　他手拍在帅案上，定下了今日会议的结果：“打打看。”
