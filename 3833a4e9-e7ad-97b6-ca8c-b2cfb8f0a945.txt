　　正所谓近朱者赤近墨者黑，历史上的长乐以贤良淑德而著称，但现实中经过李承乾的言传身教，这丫头已经完全黑化，从某无良皇帝处取经归来之后，立刻加快速度，大干快干外加埋头苦干起来。
<br/>　　广告牌打造两千个，长安一百零八坊，一个坊市摆上四个，城外火车站，田间地头摆上百来个，四个马球场附近也摆上几百个。
<br/>　　长孙冲被叫进宫里，长乐使出撒泼打滚的手段，包下大唐旬报整整一个版面，目的只有一个：广告拉招租。
<br/>　　大唐长安杯马球大赛，将有无数人来参加，还有什么比这个时候打广告再有意义的？打广告是为了什以？还不是为了让更多人知道你的品牌，现在这么好的机会怎么可以错过！
<br/>　　不得不说，那些商人明知道这个广告位不便宜，但最后还是受不了诱惑，主动跑到了马球大赛筹备办公室，随身带着数不尽的财富。
<br/>　　面对这样的情况，长乐这丫头也够狠，直接派了杨雨馨那丫头出面负责广告招标这件事情。要知道杨雨馨可是跟着李承乾混了不知道多少年，从某人身上学了‘一身本事’。
<br/>　　所以商人们来到马球大赛筹备办公室之后，立刻意识到这次怕是要大出血了。
<br/>　　为什么？简单，因为就在那个大门之上，清清楚楚的写着两个大字“招标”！
<br/>　　招标是啥意思？或许以前有很多人不清楚，但是现如今，就算是路边的乞丐都能给你简单的介绍一下。
<br/>　　这东西其实说白了就三个字；比钱多。
<br/>　　广告牌每一个都没有定价，完全就是你觉得它值多少钱那就投多少钱，所有有意向的人聚在一起，大家机会均等，不过却只有一次，你投标的价格低了？不好意思，这广告牌就是别人的了。
<br/>　　那位说了，难道就不能大家通通气，大家先提前商是好了谁家要包几面广告牌，统一标个价格，这样不就可以省钱了么？
<br/>　　这个当然可以，但是官家也不是傻子，你价格定的太低他完全可以按照流标处理，就是我不卖了，你们哪凉快哪待着的意思。
<br/>　　另外就是人心隔肚皮，虽然表面上大家商量好了，可谁知道会不会有人暗中使坏？你提前商量好每个广告牌一百贯，投标的时候大家分着投，永兴坊的四块广告你投了四百贯，别人家投三百九十六贯，而到了崇仁坊，你投三百九十六，另一家投一百。
<br/>　　可是万一别人使坏呢？别人投了一百零一怎么办？你到时候找谁哭去？要知道，招标不是拍卖，还有第二次机会。
<br/>　　招标这种事情机会只有一只，那就是大家一起投标，选竟选各自想要的位置，错过了也就错过了，根本没有机会再来挽回。
<br/>　　所以在看到招标两个字的时候，所以前来洽谈的商人全都心中一沉。
<br/>　　而随着他们进入到所谓的筹备办公室之后，一身干练打扮的杨雨馨已经带着两个侍女已经在等着他们了。
<br/>　　“诸位，坐吧，今天咱们只是公布一些招标的细节，大家可以放松一些，将精力都留到三天之后的招标大会。”
<br/>　　“谢过杨仪人！”
<br/>　　仪人，正二品官身，掌教化，教礼仪，是为仪人。
<br/>　　不过自从李承乾这货接了皇位之后，以前的老规矩被他改了不少，现在掌宫人礼仪之事已经变的不是那么重要。
<br/>　　杨雨馨等那些商人一个个施礼之后坐下，这才慢条斯理的说道：“这次招你们前来，想必大家心中都有诸多问题，不过大家可以放心，你们的问题在今天都得到解答，不会让你们失望。”
<br/>　　“是！吾等明白。”
<br/>　　大唐必竟不是后世，自然不可能像后世记者招待会一样任由下面那些商人发问，尽管李承乾这十几年来一直致力于改善他们的地位，但士、农、工、商这种阶级观念早就已经深入到人们的骨子里面，并不是十几年时间就可以改变的。
<br/>　　所以当杨雨馨讲话的时候，下面没有任何人敢胡乱开口，就算是她讲完了，下面同样没有人敢开口发言。
<br/>　　杨雨馨对这样的场面也没觉得有什么不正常，沉默片刻缓缓说道：“这次马球大赛，根据官方估计，前来观看的人次应该会达到一千万人次左右，这些人都是诸位潜在的消费者，希望诸位不要大意。”
<br/>　　“另外，大赛筹备办公室为这次大赛准备了两千个广告位，足够大家竞争，但广告位置随多，可也有好坏之分，这一点其实不用我说大家也都清楚。”
<br/>　　“而接下来要说的，就与诸位息息相关了，诸位要听好。”
<br/>　　因为曾经跟在李承乾身边待过十多年，杨雨馨身上的气场并不比朝中那些三品、四品的官员要差，介绍总体情况下的时候，下面那些商人就算是不想听，但一个个也都竖起耳朵，假装着努力听讲。
<br/>　　在他们看来，杨雨馨接下来要说的，应该就是警告之类的言语，所以见她停了下来，一个个全都屏住呼吸，准备迎接暴风雨的来临。
<br/>　　不想，杨雨馨似乎不想按常理出牌，简单的介绍了一下之后竟然直入主题，沉声说道：“现在要说的是招标的流程，首先，这次招标分为四个阶段，第一阶段，三百个广告位；第二阶段，四百个广告位；第三阶段，五百个广告位；其它广告位置，将在最后一个阶段招标。”
<br/>　　“其次要说的是，这四个招标阶段，是根据广告位置而分的，最好的位置都在第四阶段，而第一个段阶自然不用说，都是一些位置不怎么样的。”
<br/>　　“最后，这次招标的投标形式为暗标，你们可以把自己心中的价位开出来，我们筹备办公室会从中选择价最优者。”
<br/>　　“还有，一会儿你们离开的时候会各自领地一份地图，地图上标明了各个广告牌的位置，你们可以亲自去看。”
<br/>　　“哦对了，还有一点，作为对这次中标者的奖励，等到华夏杯马球大赛的时候，这次的中标者将有机会参与一次商业洽谈会，主要与会者将是其他各国的贵族与大商人，这一点希望可以引起大家的重视！”
<br/>(本章完)
