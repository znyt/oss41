　　“你说它是东皇钟！！”叶少阳几乎叫起来。
<br/>　　在场的都是强者，耳朵都很好，再说叶少阳这一声说的很大声，所有人听到了。霎时间，所有人都怔住了。
<br/>　　东皇钟！
<br/>　　上古妖皇东皇太一的本命法器！居然就在眼前！
<br/>　　“怪不得……我明白了！”摇光仙子失声说道，“我们早就发现铁塔有问题，却一直没想到，东皇钟就在塔的中间！”
<br/>　　其实也不是没想到，之前也有人试过把铁塔打开，一探究竟，但是铁塔外层的金属，像是一种法术的绝缘体，任何法术作用上去，都是石沉大海，完全不会产生任何反应，又无比坚固，空界可不是人间，也没有炸药之类的东西，因此没有成功，后来也就放弃了。
<br/>　　谁能想到，开启东皇钟所需要的，是九尾天狐的血！
<br/>　　不过众人立刻也就想明白了：九尾天狐是天选之妖，说是天选，其实就是东皇太一选的。九尾天狐是万妖之王，名义上是东皇太一选的掌控整个妖界的人选，她能开启东皇钟，也算是有资格的。换句话说，如果连她都没有资格得到东皇太一的法器，世上也没人有这个资格了。
<br/>　　现在所有人关心的都是，这东皇钟作为东皇太一的本命法器，究竟有着怎样可怕的力量？
<br/>　　“太迟了，太迟了……”小九读完上面的咒文之后，喃喃自语。
<br/>　　“什么意思？”
<br/>　　“这的确是东皇钟，上面的咒文，可以解开它的封印，只有我能做到，但是……我要死了，要东皇钟又有什么用？”
<br/>　　叶少阳一怔，道：“肉身没有办法恢复了吗？”
<br/>　　“我肉身……在被刺破了心脏后，已经死了，我只是用魂魄暂时控制肉身而已。”小九递给叶少阳一个苦笑，并没有说出残忍的真相，心中也是打定了主意，将这机缘送给叶少阳，于是转过身，轻轻抱紧了叶少阳，嘴巴在叶少阳嘴上印了一下。
<br/>　　叶少阳没有抗拒，刚要主动一点，突然感到小九身体一颤，一道倩影，离开了她的身体，飞上了东皇钟的顶端。
<br/>　　是她的魂魄，盘踞在东皇钟的顶端，冲叶少阳微笑着。
<br/>　　“你干什么！”叶少阳又一丝不详的预感。
<br/>　　“东皇钟现世，需要用我的血，解开封印，需要的却是我的魂力……少阳，李浩然之前那一掌，已经震裂了我的元神，我用自己的魂力，一直勉强裹着元神，也只是因为舍不得你……我已经必死无疑……以我现在的魂力，如果打开封印，也会立刻魂飞魄散……”
<br/>　　小九端坐在东皇钟顶端，浑身上下发出一种幽绿色的光芒，叶少阳一看便知，这是在燃烧魂力，立刻飞身跳上东皇钟的顶端，站在小九面前，却什么都不敢做，生怕自己如果强行阻止她的话，她的魂魄会立刻碎裂，加速魂飞魄散的进程。
<br/>　　“你个白痴，你在干什么啊，这个时候了要什么东皇钟啊，你快点中止，我带你走啊！”叶少阳急的浑身哆嗦，嘶声吼叫起来。
<br/>　　“来不及了，少阳，就算我放弃，我的魂力也维持不了太久的，我已经失去了修为……一旦魂力耗尽，元神一样会散去……”
<br/>　　叶少阳泪流满面，喃喃道：“那你还祭炼这东皇钟干什么！”
<br/>　　“不是为我，而是为你……”
<br/>　　小九一边说着，一边燃烧着魂力，化作绿色的光华，渗入了东皇钟四周的符文上面，一点点开启着东皇钟的封印。
<br/>　　“少阳，能在死前为你做最后一件事，挺好的。”小九没有流泪，她现在所有魂力都用在了开启封印上面，根本没有多余的魂力来模拟哭泣，而且，她想留给叶少阳一个最美好的自己，于是尽力微笑着。
<br/>　　在她的意识之中，反复回放着与叶少阳在一起的点点滴滴，从最初**叶少阳不成，自己动了凡心……到叶少阳为了救自己，不惜跟通玄道人拼命，那一刻，她的心就给了这个不平凡的男人……
<br/>　　小九突然笑起来，对叶少阳说道：“还记得那次在幻象中吗，其实，那是我第一次施展魅惑之术，结果……没能迷住你，却让自己彻底陷入进去，真是挺丢人的……”
<br/>　　“小九，你——”
<br/>　　“少阳，我彻底已经没救了。”小九打断他的话说道，“这是你和我最后的时间，你就好好地陪陪我，好吗？”
<br/>　　叶少阳心中悲伤至极，却也知道这是无法挽回的结果，只好静静地坐在他的对面不动，听他静静往下说。
<br/>　　“在西域，那一天我陪着你喝醉，然后你搂着我睡着，当时那种感觉，是我一生中最开心的时刻，我当时就在想，哪怕将来我注定要死，有那么一段最美好的回忆，我也值了。”
<br/>　　“很可惜你给我买的那身衣服，我现在没有穿着……”她的魂力越来越弱，人影变得透明起来，声音也越来越小了。叶少阳静静地坐在她面前，一个劲地盯着她，想要把她的样子牢牢记在心底。
<br/>　　突然，几道人影从山谷两边的山野间射过来，极快地冲向叶少阳和小九。
<br/>　　“阻止她祭炼东皇钟，所有人一起，修正目标，夺回东皇钟！”
<br/>　　是女魃的声音。
<br/>　　这突然来袭的数十道人影，正是女魃和她手下数十个最为精英的飞僵，就算是僵尸，他们也十分觊觎东皇钟这种上古神器。
<br/>　　退一步说，就算他们是僵尸，无法驾驭这件宝贝，也一定要把它夺走，绝对不能让它留在空界，否则不管是谁得到并操控它，将来对于尸族来说，都是一个巨大的隐患。
<br/>　　“不能让尸族夺走东皇钟！！”
<br/>　　空界诸强者之中，爆发出了一声喊叫，大伙回过神来，一个个都朝着东皇钟这边飞奔而来。
<br/>　　李浩然手持长剑，凌空挥舞，组成了一道剑阵，迎上了女魃。
<br/>　　女魃也想要一举堪破李浩然的防守，结果被震了回去，定睛看着李浩然，道：“我一直以为你是传说中的人物，欺世盗名，没想到失去一只手，还这么厉害。”
