　　察觉到体内扩散趋势愈加剧烈的尸王，眼神开始微微着急起来，他已经无心继续这场战斗，但刘哔死死缠住他，根本没有一丝逃脱的机会。
<br/>　　“狂妄，真当我奈何不了你！”
<br/>　　紧咬着牙关，尸王强行用身体接下了刘哔的进攻，同时密密麻麻的触手纠缠到一起，拧成一柄长剑的模样。
<br/>　　完全用血肉铸成的长剑上，带着似乎无可匹敌的力量，迅疾的向刘哔斩去，而这一刻尸王本身的气势，也是在瞬间爆发出来。
<br/>　　面色稍稍有些凝重，刘哔站在原地未动，黑色的风衣遮挡住他的身体，一双猩红的瞳目，散发着彻骨的冰寒。
<br/>　　炙热的火焰，覆盖住他的双臂，手持着火焰长棍，刘哔的双手往上一撩。
<br/>　　“呼！”
<br/>　　霎时，棍体精准的击打在了触手长剑上面，面对尸王的顷力一击，他没有选择闪躲，而是依靠自身强大的力量硬撼了上去。
<br/>　　“锵！”
<br/>　　两炳武器碰撞到了一起，顿时发出一阵争鸣声，火花溅射间，一股强大的气浪向四周肆掠开来。
<br/>　　强大的冲击力，甚至让周围的空气都有些扭曲。
<br/>　　而身处波动中心的二人，似乎没受到任何影响，这一次交锋中，谁都没占到什么便宜。
<br/>　　紧接着，两人互相对攻了起来，短短不过数息的时间，他们已是互相交手数十回合，下手皆是狠辣，直指要害。
<br/>　　但在这种激烈交锋间，尸王的眼神，愈发的阴沉了下来，因为他发现，即便是激活机甲的全部战力，却依旧占不到任何的上风。
<br/>　　该死，这家伙的战斗力怎么会这么强大，这次进攻，花费了他太大的代价，甚至凭借这一手，他都能和比自己高一个等阶的存在战斗。
<br/>　　可偏偏，这样都奈何不了刘哔，尸王心底不禁泛起一抹恐惧。
<br/>　　“铛！”
<br/>　　双方的武器又是一次碰撞，刘哔眼中红芒一闪，那吞吐着火焰的棍棒尖端，突然有凌厉的棍芒凝聚而出，约莫半丈，吞吐之间，连空间似乎都要被撕裂。
<br/>　　这道棍芒，是汇聚了刘哔体内大半的能量凝聚而成。
<br/>　　因为一瞬间涌入了太多能量，滔天的烈火从棍身上汹涌而出，吞吐之间，似能灼尽世间一切物品。
<br/>　　刘哔棍身一压，那道半丈左右的棍芒，竟是脱离而出，化为一道无形的冲击波，快逾闪电般的向尸王激射而去。
<br/>　　随着冲击波暴射而至，尸王浑身的汗毛也是猛的倒竖了起来，同时，眼中泛起了一抹慌乱。
<br/>　　在这一刻，他感受到了面临死亡的恐惧。
<br/>　　面对这样凌厉的进攻，尸王不敢有丝毫的怠慢，他深知刘哔恐怖的攻击力，如果没接下这次攻击，就算是他，也必然十死无生。
<br/>　　“吾有一剑，可斩天下敌。”
<br/>　　低喃一声，尸王的整条右臂，都汇入了身前的触手长剑中。
<br/>　　突然，猛的舞动，一轮美幻绝伦的青色光圈，在其面前慢慢成型。
<br/>　　浓郁至极的煞气，从剑身上散发出来，尸王的气势开始攀升，在他身上，泛起了一种至高无上的威严，宛如，万鬼至尊一般。
<br/>　　“嗤！”
<br/>　　无形的棍芒搅进青色光圈之中，瞬间受到一股狂暴能量的冲击，但青色光圈依旧在扩散，气势攀升的却愈加剧烈。
<br/>　　挡下了这一击，但尸王的脸色却更加阴沉，在其前胸的部分，赫然出现了一道深可见骨的伤痕！
<br/>　　刚才的那一道攻击，即使大部分的威力都被格挡了下来，但那强大的冲击力，却依旧将他胸前击出了一个大洞。
<br/>　　虽然伤势并不算很重，但却让他心中涌上一股劫后余生的庆幸感。
<br/>　　如果威力稍稍再强一些，甚至光凭冲击力，都能将他给砸成肉泥。
<br/>　　但来不及多想，短短不到一瞬的功夫，尸王回过神来，青筋凸起的脸颊有些狰狞，原本阴柔的语气也变得极为森然：“真是让人遗憾啊，只差一点，你就能杀掉我了。”
<br/>　　说话的功夫，响尾蛇手中触手剑重重一挥，空间瞬间扭曲，一圈肉眼可见的冲击波爆发开来。
<br/>　　“不过，你也该适可而止了！”
<br/>　　喝声落下时，尸王抬起长剑，在其尖端，有淡淡的光芒浮现，隐隐间，似乎是有着一柄五彩缤纷的华丽剑影，缓缓从出手剑身上凝聚出来。
<br/>　　剑影在尸王用血肉铸成的剑身上微微颤抖，一道嘹亮的剑吟声，响彻而起。
<br/>　　“铮！”
<br/>　　清脆的剑吟声传出，初始微弱，下一瞬，竟是猛然间响彻天地，一股恐怖至极的凌厉剑气，此时竟是自尸王体内爆发出来。
<br/>　　刘哔的瞳孔不由一缩，原本踏出的步伐也是停顿了下来。
<br/>　　望着面前的尸王，他的眼神有些惊疑不定，刘哔没能想到，已经穷途末路的尸王，竟还能爆发出如此惊人的力量。
<br/>　　那种惊天的剑气，连他都是感觉到了深深的威胁。
<br/>　　看着脸色阴沉的刘哔，尸王嘴角轻轻一勾，紧紧的握着手中那五彩缤纷的剑影，下一刻，猛然出手，剑影对着前方，轻轻一斩。

<br/>　　五彩剑影落下，与那半圆形防护罩碰了一下，‘唰’的一声，竟是诡异的直接消失不见。
<br/>　　尸王的攻击，来得快，消失得更快，仅仅不过数息的时间，便是消散与天地之间。

<br/>　　但他知道，尸王的这一剑，并没有这么简单。
<br/>　　在这一刻，四周的空气似乎都变得诡异了起来。
<br/>　　一丝略微粘稠的触感，从刘哔脖颈处传来，有些泛黑的血液，滴答落下。
