人开始整装启程，朝着喜马拉雅山脉往珠峰方向移动。<br/>　　<br/>　　    于此同时，距离山脉大概三十公里左右方向，那架直升机降落在了一处空地上，旁边不远就是王静这条阿门乘坐后弃在这里的兰德酷路泽。<br/>　　<br/>　　    黄九郎裹着厚厚的军服从机舱中出来，一个挂着一杠三星的军官迎了过来“啪”的一声两腿并拢，敬礼道：“报告首长，第三十八连上尉鲁靖尧奉命在此等候！”<br/>　　<br/>　　    黄九郎摆了下手，说道：“不用客套了，那辆兰德酷路泽是什么状况？”<br/>　　<br/>　　    “人员全部消失，车中补给也全都带走了，从他们离去的迹象来看，应该是进入喜马拉雅山脉中了，目前方向和移动轨迹不明”<br/>　　<br/>　　    “你们来了多少人，携带了什么武器，装备”<br/>　　<br/>　　    “报告首长，三十八连全员120人全部出席，携带九五式，五四式还有热成像仪，卫星定位系统……”<br/>　　<br/>　　    “由你领队二十人左右，跟我进入喜马拉雅山脉，其余人延迟两个小时进山，随时做好待命状态”<br/>　　<br/>　　    “是，首长！”<br/>　　<br/>　　    随后，黄九郎和鲁靖尧还有二十名全副武装的士兵，开始朝着喜马拉雅山脉行进，剩余队伍则是原地等待，准备稍后在进入。<br/>　　<br/>　　    临近晌午左右，王惊蛰和孔良还有陈三岁一路追踪着藏獒的痕迹，最后在一处山窝里发现了一滩血迹，还有碎裂的骨头和内脏，两根羚羊角，血迹都已经干了。<br/>　　<br/>　　    这显示，那头獒王追着夜宵在这附近得逞后直接就给吃完，然后又再次离去了，之间至少已经过去半夜加一上午了。<br/>　　<br/>　　    孔良颓然的叹了口气：“时间太久了，这下子想再找到它可就有点难了，喜马拉雅山脉面积太广，想找一头藏獒太难了”<br/>　　<br/>　　    陈三岁轻声问道：“能算么？”<br/>　　<br/>　　    想和更多志同道合的人一起聊《天命赊刀人》，微信关注“热度网文或者rdww444”与更多书友一起聊喜欢的书
