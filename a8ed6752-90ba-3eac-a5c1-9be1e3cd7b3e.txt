　　第335章
<br/>　　“十”
<br/>　　“九”
<br/>　　“八”
<br/>　　……
<br/>　　程大雷一字一顿数着，节奏不快不慢，事实上，叶历投降与否，对他而言并不重要。甚至于说，直接杀掉这帮人，要比留着他们做俘虏节省更多麻烦。
<br/>　　自己刚到琴川关，假若需要杀人立威，眼前岂不正是一个最好的机会。
<br/>　　“七”
<br/>　　“六”
<br/>　　可程大雷如此做，却给叶历带来了巨大的压力。他读懂了程大雷眼神中的意思，那是无视和不屑。
<br/>　　“五”
<br/>　　“我……投降。”
<br/>　　叶历艰难道出这三个字，手中的武器砰地一声砸落在地上。
<br/>　　程大雷嘴角抿起，发出一声冷笑。
<br/>　　“秦蛮，接收他们。”
<br/>　　叶历的人陆陆续续从院内走出，放下兵器蹲在墙角。
<br/>　　此刻琴川关内的战斗已接近尾声，有些人见事不对，就带着人从城门逃了。有些人逃向关内，有些人逃向草原，而这些也是程大雷乐意看到的。真如果这些人坚守一地，负隅顽抗，想要彻底清除他们也是相当麻烦。
<br/>　　程大雷开始带着人打扫战场，生火的地方要想办法灭掉，俘虏们卸掉武装后，找几个院子关起来。也要给这些人做一些背景调查，总不能他们说投降就投降，我蛤蟆寨的录用也是有条件的好不好。
<br/>　　还有一些顽抗的小股敌人，这些人要么缴械投降，要么直接杀掉，总之算不上什么麻烦。
<br/>　　最主要的麻烦竟然是灭火，放火容易灭火难，诸多人一起忙碌，也费了很大力气。
<br/>　　这些事情自然不用程大雷插手，他把琴川关的地形图要来，一边走在城墙上，一边与手中的地图对照。
<br/>　　据说，莫鸣米没事就喜欢走在城墙上，现在程大雷可以体会他的感觉了。目之所及，都是自己的地盘，就是一天看三百六十五遍也不嫌多。
<br/>　　琴川关是一座真正的雄关，也是一座严格意义上的军事堡垒。城内建筑能容得下十万精兵，有兵营、将军府、马场、矿场……除了少数几个大人物有独立的院子外，几乎没什么生活建筑。
<br/>　　程大雷下了城头，出现在将军府，在大厅最大的椅子上坐下，程大雷长长出了口气。
<br/>　　这个位置早该是自己的，最后混了大半年自己才做到这里。在青牛山、在蛤蟆岛程大雷都没有住过这么好的房子，雕梁画栋，曲折回廊……事实上，以莫鸣米的官职住将军府是有些违制的，只不过这里天高皇帝远，莫鸣米就是这里的土皇帝，自然也不会有人管他。
<br/>　　程大雷第一件要做的事情，就是把莫鸣米留下的痕迹抹掉，目前这里已经物归原主。
<br/>　　“来人，把人给我押上来！”程大雷重重道了一声，惊堂木拍在帅案上。
<br/>　　五花大绑的严笛被押了上来，程大雷意味深长的冲他撇撇嘴，忙站起来身，向严笛迎去。
<br/>　　“谁做的事，谁做的事，不知道严军师是自己人么，快给军师松绑！”程大雷走到严笛面前，情真意切道：“同志，光明到来了。”
<br/>　　呸！
<br/>　　一口血沫喷在程大雷脸上，程大雷讨了个没趣，拂袖将脸上的血沫抹去。
<br/>　　“军师，这是何苦来哉，咱们是自己人呐！”程大雷道：“现在莫鸣米已死，正是你回归组织的时候，我不会忘记你为组织做过的一切，日后你就是我的右军师，和徐神机平起平坐，这样做如何？”
<br/>　　“狗贼！”严笛将牙咬得咯吱咯吱响：“你要杀就杀，不过一死而已，何必废话！”
<br/>　　“军师，息怒，息怒。”程大雷倒真不想杀了严笛，收为己用不好么。
<br/>　　这世间并没有永恒的敌人，也没有永恒的朋友，只有永恒的利益。
<br/>　　严笛恶狠狠的盯着程大雷，良久目光内的凶意收敛，轻轻叹了口气。
<br/>　　“程当家，兵不厌诈，无论你使怎样的诡计，我都不能怪你。这次是我对不起将军，你若仁慈，就赐我一个全尸，让我到九泉之下，再向将军赔罪。”
<br/>　　程大雷默然，慢慢坐回到帅案后。他并不想杀了严笛，如今正是用人之际，留下一个熟悉琴川关情况的严笛，会省很多麻烦。
<br/>　　可看着严笛的眼睛，程大雷忽然觉得，或许赐他一死，才算是成全他。
<br/>　　程大雷沉默良久，最后挥了挥手。
<br/>　　“带他下去，给他一个全尸，将他的坟和莫鸣米葬在一起。”
<br/>　　临别之前，严笛看了程大雷一眼，却没有说什么，慷慨从容的走出屋子。
<br/>　　程大雷有些低落，其他人倒不觉得如何。刘悲近前道：“大当家，如今还有一件事情，比较棘手，需要你亲自处理。”
<br/>　　“什么事啊，我不拿主意就办不好了？”程大雷。
<br/>　　“把人带上来吧。”刘悲拍拍手。
<br/>　　程大雷眯着眼坐在太师椅上，已经有些无精打采，待看到带上来的一个个人时，程大雷眼睛一下睁开，身体前倾。
<br/>　　带上来的都是一个个女人，二十岁左右，身材窈窕，约有十几个，杵在程大雷面前，形成一道靓丽的风景线。
<br/>　　“这些是什么人啊？军妓？”程大雷低声问。
<br/>　　“不。”刘悲摇摇头：“是莫鸣米的小妾。”
<br/>　　“一个，两个，三个……”程大雷默默说着，直到数到两位数：“我擦，莫鸣米这么奢靡啊。”
<br/>　　“这只是一部分，莫鸣米的夫人已经悬梁自尽了。”刘悲道：“这些人怎么处理？”
<br/>　　“呔！”程大雷轻喝一声：“你们打算怎么办，是愿意留在这里，还是想离开自谋生路。”
<br/>　　这些女人跪倒在程大雷面前，几乎同时道：“我们愿意服侍将军。”
<br/>　　“这……这……”程大雷正想说声那我就不客气了，忽见一旁的樊梨花美目翘起，隐隐有杀机流转，他立时板起脸，道：“呔，你把本寨主当成什么人了，是和莫鸣米一样的色中饿鬼么。你们如果愿意留下，从今天开始就是琴川关居民。”
<br/>　　“居民”
<br/>　　“嗯呐，你们可以选择生活在这里，也可以选择离开，总的说来，你们自由了。”程大雷清了清嗓子道，不知为何隐隐有些肉疼。
