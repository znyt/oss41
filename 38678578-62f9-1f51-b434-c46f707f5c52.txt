　　“国外那个TM战队已经为小艺报价两个亿了吧，这个身价已经赶上十王者了，他们也是真的下血本了。”秦霄一脸感叹的说道。
<br/>　　“清浅不也是被报价一个亿了么，两个孩子还不大就能赚这么多钱，真羡慕你们。”林萧一脸艳羡的说着。
<br/>　　“他们两个上场了三周，实力都是有目共睹的，无论是小艺还是清浅只要拉入他们的战队中，对于他们的实力都绝对是一个非常大的提升的，所以他们虽然报价2个亿，1个亿，但是与他们两个本身能够创造的价值来说，其实还不算多。”叶颖笑着分析着。
<br/>　　“不过他们两个不愿意去啊，那些人现在只能干瞪眼了，未来的几年随着老一辈的选手都退役，到时候咱们荣耀战队肯定会一枝独秀的。”蒋念稍稍有些兴奋的说着。
<br/>　　“接下来几周的比赛都不算太难，大家也趁着现在这段时间好好放松一下吧，等到打季后赛的时候，时间可就紧张起来了，到时候不仅战队的实力强，而且夏季赛结束不久马上就是世界赛了，中间休息的时间不多。”叶帝开口说道。
<br/>　　这几周他们遇到的都是国内中下水平的队伍，以他们的实力自然是不用担心，所以这个时候叶帝倒是打算让大家好好放松一下。
<br/>　　毕竟过了这个月，下个月就要开始季后赛了，等到夏季赛结束后马上又是世界赛，如今的世界赛仅仅只有32支队伍。
<br/>　　虽然队伍变的少了很多，但是同理这些队伍的实力却是要一个比一个强，到时候他们面对的肯定是更加激烈的对战的，甚至可能因为时间的紧张，导致赛程都安排到了一起，那个时候他们可就彻底没有休息时间了。
<br/>　　“嗯，这个月是大家最悠闲的一个月了，从下个月开始进入季后赛后，大家就要全力以赴了，平时的休息时间也没有了，到时候咱们可是还要为世界赛做准备呢。”李景禹跟着开口说道。
<br/>　　“那这个月一周大家就休息一天半吧，你们想要出去玩，那就出去走走，不想出去玩的话，也可以在家里睡懒觉。”叶帝当即下了决定。
<br/>　　听到休息，基地内顿时传出了一阵欢呼声，要知道职业选手其实也是很累的，尤其还是他们荣耀战队。
<br/>　　每天早上5点钟就起床锻炼，然后上午裸妆补兵，下午还有训练赛乱七八糟的，这样子其实跟上课没有什么区别，所以听到放假，大家自然都是非常期待和欣喜的。
<br/>　　当天夜里叶帝便是返回了京都，大家休息，他自然也要跟着休息一下了，而且已经有一段时间没见苏沐婉了，他也有些想她们了。
<br/>　　基地内的人倒是也没有都走，像是杨小艺、秦霄还有林雅楠、柳清月还有李景禹他们就都留在了基地里，并不是所有人都喜欢出去玩的，尤其是职业选手。
<br/>　　大部分的职业选手其实本质上都是一个宅男，空闲的时间他们宁愿打打游戏，也不愿出去溜达。
<br/>　　当回到家里后，看到苏沐婉那已经隆起来的肚子，叶帝心里对这个世界突然有了更多的归属感。
<br/>　　苏沐婉肚子里的孩子就是证明自己来过这个世界的痕迹。
<br/>　　轻轻的拥着苏沐婉，两个人什么都没说，就这么静静的拥抱着。
<br/>　　“是男孩还是女孩？”叶帝开口问道，到现在苏沐婉已经怀孕6个月了，还剩下4个月，如果有心的话肯定是能够看出来的。
<br/>　　“妈妈他们没查，说无论男女都一样养。”苏沐婉悄声说道，不想让另外一个屋子里的父母听到他们的对话。
<br/>　　夜里两人聊了很多，直到11点多的时候，两人才相拥而睡。
<br/>　　第二天在家里吃了顿午饭后，叶帝便是再次着急忙慌的赶回了上海，一天半的时间在叶帝还没有察觉的时候便是已经悄然而过。
<br/>　　再次离开京都这座城市的时候，叶帝的心里却是多了一份挂念，这一刻他真的很想什么都不管，就这么安静的陪在苏沐婉身边，等待着自己孩子的降世。
<br/>　　“不过说实话沐婉和刘诗诗长的还真是像啊，而且小名还叫诗诗，这真是太巧合了。”坐在飞机上向着苏沐婉和记忆中那个人的模样，叶帝还是感觉有些不可思议。
<br/>　　回到基地的时候，已经是下午5点钟了，除了他是最后一个回来的，蒋念他们几人这个时候也都是早就回来了，此刻正坐在大厅中闲侃呢。
<br/>　　“怎么样，沐婉还有几个月生小孩啊，男孩女孩？”叶帝刚刚进屋，蒋念便是迫不及待的开口问道。
<br/>　　“还早着呢，估计要等到咱们打领土战的时候才会生小孩，男孩女孩家里也没查，顺其自然。”叶帝笑着说道。
<br/>　　不过他也知道，其实父母之所以不查可能还是怕失望吧，毕竟他们叶家需要男丁来继承家业，如果苏沐婉生下一个女孩，想必他们一定会很失望。
<br/>　　不过这份情绪他们又不能在苏沐婉面前表现出来，所以他们只能克制住自己内心中的好奇，等待着孩子出世的那一天。
<br/>　　“我们可都还等着做干爹呢！”蒋念嬉笑着说道。
<br/>　　“怎么这个词语从蒋念的嘴里说出来，感觉这么污呢。”一旁的香蕉小声的嘀咕着。
<br/>　　虽然他的声音很小，但是周围就这么大的距离，众人还都是能够听的到的。
<br/>　　一瞬间众人看向蒋念的表情都变的有些诡异了起来。
<br/>　　“我说蒋念你不会是有那个爱好吧，我现在只能祈祷小叶子赶紧生儿子了。”秦霄一脸恶寒的说道。
<br/>　　其他几人也都是如此。
<br/>　　“卧槽，你们能不能不要这么污啊，不过我倒是想有个干女儿啊，我还是比较喜欢女孩的，小叶子你最好让沐婉生一个龙凤胎。”蒋念一脸无语的说道。
<br/>　　“草，我倒是想啊，不过那也不是我说的算的啊！”听着蒋念的话，叶帝也是非常的无语。
<br/>　　其实他内心里也是喜欢女儿的，所以如果苏沐婉能够生一个龙凤胎，那他可就真的是人生圆满了。
