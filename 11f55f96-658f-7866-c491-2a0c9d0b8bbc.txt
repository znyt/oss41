厉嫣然突然半路杀了出来，跟夏夕绾抢夺这条香槟色的长裙。

<br/>        蓝烟一听火冒三丈，“厉嫣然，先来后到懂不懂，这条裙子是我们先看上的，你凭什么跟我们抢？”
<br/>
<br/>        厉嫣然伸手，直接挽住了陆寒霆的健臂撒娇，“寒霆哥哥，我想要那条裙子，可不可以嘛~”
<br/>
<br/>        导购员看着陆寒霆迅速恭敬的叫了一声，“老板。”
<br/>
<br/>        蓝烟：靠！她都忘了帝都最大的这家商场是陆氏旗下的，陆寒霆可是这里的老板，大家都必须听他的。
<br/>
<br/>        厉嫣然得意的看向夏夕绾，“夏夕绾，你还不知道吧，寒霆哥哥是这里的老板，你身上这条香槟色的裙子寒霆哥哥说不卖给你就不卖！寒霆哥哥，我就想要那条裙子，你送给我吧。”
<br/>
<br/>        厉嫣然又贴在陆寒霆的身上软磨硬泡的撒娇。
<br/>
<br/>        夏夕绾澄亮的翦瞳落在了陆寒霆的俊脸上，坦诚大方的看着他，“陆总，做生意有做生意的规则，你应该不会为了宠妻而侵犯我们顾客的权益吧？”
<br/>
<br/>        宠妻…
<br/>
<br/>        陆寒霆知道自己已经订婚了，但是从她的小嘴里听到“宠妻”这两个字他还是觉得很刺耳，她果然一点都不在乎他，她也不会为了他而吃厉嫣然的醋！
<br/>
<br/>        他和厉嫣然订婚，对她一点影响都没有，而且，她还准备去相亲选夫什么的。
<br/>
<br/>        陆寒霆只要想到明天她会穿着这条香槟色的长裙出现在那些男人的面前，将那些男人给迷得神魂颠倒的，他就觉得胸膛那里闷闷的，像是被一块大石给压住了，喘不了气。
<br/>
<br/>        他不想让那些男人看到她纤裸的美背，她只能穿给他看。
<br/>
<br/>        陆寒霆抿了一下薄唇，深邃的狭眸看着夏夕绾，“我做生意不需要讲规则，我想怎么做就怎么做。”
<br/>
<br/>        “…”夏夕绾看着他，这万恶的资本主义！“所以，陆总是什么意思？”
<br/>
<br/>        陆寒霆伸手搂住了厉嫣然的香肩，“我未婚妻的意思就是我的意思。”
<br/>
<br/>        厉嫣然迅速笑了，笑的像一朵花，她仰着漂亮夺目的小脸看着陆寒霆，“寒霆哥哥，谢谢你。”
<br/>
<br/>        夏夕绾垂下纤长的羽捷，不去看那两个人秀恩爱的姿态，其实她已经猜到了他的答案，现在厉嫣然是他的未婚妻，他当然听自己未婚妻的。
<br/>
<br/>        她早就不该对他抱有什么希望了，但是每一次遇到他，她的心还是会因为他而跳动。
<br/>
<br/>        蓝烟早已经忍不住了，她直接跳了出来，“陆寒霆，你…”
<br/>
<br/>        “蓝妈，”夏夕绾一把拉住了蓝烟，“算了，一件衣服而已，我们再挑别的，我现在去换下来。”
<br/>
<br/>        夏夕绾都这么说了，蓝烟只能忍住了，不过她狠狠的瞪了陆寒霆一眼，想要用目光将陆寒霆给瞪出一个窟窿来。
<br/>
<br/>        夏夕绾很快就换下了那件香槟色的长裙，她将长裙递给厉嫣然，“裙子给你。”

<br/>        厉嫣然骄傲的抬起了下巴，伸手接过了长裙，然后递给导购员，“帮我把裙子包起来，夏夕绾，谢谢你哦，谢谢你忍痛割爱。”
