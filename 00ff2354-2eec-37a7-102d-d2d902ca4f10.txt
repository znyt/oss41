　　奥帕的介绍，顿时令得底下前来参加拍卖的人一阵哗然，眼下，最令人谈之色变的就是勒索病毒，具有自主传播能力，通过系统漏洞自动进行传播，除非断开网络，不然随时都有可能遭到这种病毒的攻击，造成损失。
<br/>　　所以全球各大网络安全公司都想着能研发出一款针对此次勒索病毒的防病毒软件，或者说是源代码，来解决此次病毒危机。
<br/>　　全球各大网络安全公司的人都不是傻子，谁都清楚，谁先掌握了能够解决这次勒索病毒的方法，谁就占据了优势，在第二年跟各大公司进行网络安全业务谈判时也是有了足够的底气，这是一项能够为公司带来长久利益的研究，所以各大网络公司都是不敢怠慢，可奈何经过了SHADOW之手的勒索病毒变得格外复杂，即便是各大网络安全公司日夜不停的进行研究，可是也没有能攻克这个难题。
<br/>　　出于对拍卖品持有者安全的考虑，在拍卖会上，只会公布拍卖品，但是对于这件拍卖品属于什么人是不会被公布的，所以舞台下，那些来自各大网络安全公司的代表，便开始纷纷猜测拿出这个针对勒索病毒防病毒代码的到底是何方神圣，毕竟目前这个代号“永恒之蓝”的勒索病毒是目前各大知名网络安全公司都无法解决的棘手难题，而他们看向舞台上展示柜中那个U盘的眼光也变得炽热起来。
<br/>　　奥帕对于舞台下那些炽热的目光非常满意，这才是他想要看到的画面。奥帕清了清喉咙，说道：“这件拍卖品的主人给出的起拍价格是三十万美金加一个跟勒索病毒同一级别的病毒源代码。”
<br/>　　奥帕将这个起拍价格公布以后，舞台下又是传来一阵哗然，三十万美金的起拍价格真的不算高，甚至可以说是非常低，真正令竞拍者感到棘手的，是三十万美金后面那个附加条件——跟勒索病毒同一级别的网络病毒源代码！
<br/>　　这个条件可以说是把在场绝大多数竞拍者挡在了竞拍起跑线上，在场所有的网络安全公司中，除了刚才拍下第十四件拍卖品的卡巴斯基公司以外，没有任何一家网络安全公司具有竞拍资格。
<br/>　　当然，在子午眼中，除了刚才拍下一组病毒源代码的“自由者”外，还有人具有竞拍资格，就是藏在歌剧院竞拍者中的SHADOW的手下。
<br/>　　原本子午只是猜测SHADOW会让人来参加这次源代码拍卖会，可是当刚才那组病毒源代码出现的时候，子午就确定，一定有SHADOW的人藏在竞拍者中，因为提供拍卖品进行拍卖的人必然是会在竞拍者中的，他们需要见证自己东西被拍卖的过程。
<br/>　　子午双手抱胸，嘴角轻轻勾起一抹微笑。
<br/>　　子午一开始编写这套防病毒源代码就是为了引出SHADOW手下的人，然后找到SHADOW的下落。而且子午相信，这个先前拍下病毒源代码的“自由者”应该不是简单的“自由者”，而是某个网络安全公司隐藏在红色卡片区中的人。
<br/>　　子午的猜测并没有错，这个以三百万美金的价格，拍下刚才那套网络病毒源代码的人，是卡巴斯基公司那边隐藏在红色卡片区中的人，这样做完全是为了掩人耳目，在关键时刻发力，打其他网络安全公司一个措手不及。
<br/>　　红色卡片区内，那个拍下网络病毒源代码的俄罗斯人“自由者”在征询了上面的意思后，摇了摇牙终于是举起了牌子。
<br/>　　卡巴斯基公司拍下那套跟勒索病毒同一级别的病毒源代码，是为了从中进行研究，研发出新一代的防病毒软件，可是现在为了拿到这套针对勒索病毒的防病毒源代码，卡巴斯基公司也只能是把刚刚竞拍到的病毒源代码拿了出来，再次进行竞拍。
<br/>　　卡巴斯基公司对于这次竞拍可以说是非常有信心，因为整个拍卖会现场，只有他们卡巴斯基公司能满足竞拍条件，所以在举起牌子的时候，这个俄罗斯人已经是露出了胜利的微笑。
<br/>　　“好的，我们看到有人开价了，是卡巴斯基公司！”奥帕拿着话筒大声喊叫着，说道：“三十万美金加一套跟勒索病毒同等级的病毒源代码一次，还有没有人竞价？！”
<br/>　　代表卡巴斯基公司前来参加这次源代码拍卖会的俄罗斯人嘴角露出嘲笑，现场除了他们卡巴斯基公司以外，还有谁的手里有跟勒索病毒同一级别的网络病毒源代码？在这个俄罗斯人看来，根本就不需要什么竞拍，直接就可以宣布结果了。
<br/>　　红色卡片区内，一个戴着鸭舌帽的男子双手在手机屏幕上飞快的敲打着进行请示，很快手机中受到了一条未读信息，在得到了答复后，男子收起手机举起了自己的牌子，说道：“五十万美金加上一套跟勒索病毒同级别的病毒源代码！”
<br/>　　“好的，59号出价了，五十万美金加上一套病毒源代码！”奥帕显然是对这个结果不感到意外，拿着话筒大声喊道：“五十万美金加上一套病毒源代码，第一次！”
<br/>　　“七十万美金！”俄罗斯人第二次举牌。
<br/>　　目前情况下，增加病毒源代码是不可能的，只有在价格上进行争夺。
<br/>　　“九十万美金！”男子也跟着第二次举牌。
<br/>　　“一百二十万美金！”俄罗斯人第三次举牌。
<br/>　　卡巴斯基公司拍下他们手中那套病毒源代码就花了三百万美金，现在又加上一百二十万美金，他们对舞台上展示柜中那套防病毒源代码的竞价已经是达到了四百二十万美金！这可以说是一个很惊人的高价了！
<br/>　　“放弃！”
<br/>　　鸭舌帽男子准备第三次举牌的时候，他的手机上传来了一条消息，让他放弃竞拍。鸭舌帽男子没有去询问询问原因，只是按照手机上的指示没有再举牌竞拍。
<br/>　　“一百二十万美金加上一套病毒源代码，第三次！”
<br/>　　啪！
<br/>　　奥帕砸下了手中的木锤，对着放下牌子的俄罗斯人说道：“自由者区域，85号，这个U盘是你的了！”
