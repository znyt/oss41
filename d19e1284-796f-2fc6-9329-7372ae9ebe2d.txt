　　一身黑色劲装，布鲁斯韦恩紧紧盯着苏重。背部弓起，好似一头潜藏草丛，紧盯猎物的狮子。
<br/>　　下一刻，他骤然发动。整个人猎豹般狂猛冲出，双手成爪状，猛然抓向苏重腋下。这一下若抓实，一下扣紧肋骨，能让人直接丧失战力！
<br/>　　布鲁斯韦恩身强体壮，足有两米高，在专业训练下，身体力量更是强的不行。整个人壮硕的像头牛。但这一刻，他的动作却灵活敏捷，好似灵猴跳跃，竟丝毫不见笨拙！
<br/>　　砰！
<br/>　　布鲁斯冲得快，回来的更快。好似被火车撞了似的，直接倒飞而回。躺在地上翻白眼，一张脸皱成一个苦瓜。但他却不挣扎，直接四脚朝天的躺在那里恢复。
<br/>　　“苏……你下手就不能轻点儿？”布鲁斯韦恩郁闷道。
<br/>　　“那你就不能不来找我？”苏重也无奈了。
<br/>　　自己安静练枪，这未来蝙蝠老爷，也不知道搭错哪根筋。没事就来找揍。挨一次揍不够，还每天都来找揍！搞的苏重都怀疑，这家伙是不是有什么特殊癖好！
<br/>　　“你是助教，我要训练格斗，不找你找谁？”布鲁斯·韦恩努力把气喘匀。又揉了揉疼痛胸口，慢慢坐起来，脸色还是不太好看。
<br/>　　不过这几天连续被苏重打，竟已逐渐习惯，抗击打能力加强，总算不像开始那样，一倒下就得躺半天。现在只要稍微缓一会儿，竟然能够坐起来。
<br/>　　为了能多在苏重这里多撑一会儿，他给自己不断加练。可也只不过能让他早恢复一会儿，始终都是一招就倒的命！
<br/>　　“你不是要挑战杜卡吗，去找他呀。”苏重也无奈，这家伙怎么一根筋？
<br/>　　“您还真是两耳不闻窗外事，最近可都是杜卡老师在亲身指导我。”布鲁斯·韦恩摇头失笑。他早就已经和杜卡交上手了。也就只有苏重沉迷练枪，才不知道基地动态。
<br/>　　不过想到苏重枪法，他就觉得好笑。就那渣枪法，基地里谁不知道。就这，还是整天抱着枪打个不停的结果。他已经听好多人抱怨，嫌弃苏重占据靶场，浪费资源。
<br/>　　“那你还来找我？”苏重皱眉瞪着布鲁斯，有杜卡教导，还到自己这里来找虐？神经病吧？
<br/>　　“当然是想学习更厉害的格斗术。”布鲁斯·韦恩从来都不是一个迂腐之辈。
<br/>　　几次交手他就确定，师从杜卡的苏重，在格斗术上已然胜过杜卡。而不是其他人所认为的那样，在基地所有人之上，却唯独在杜卡之下。
<br/>　　毕竟，所有人都知道，苏重是跟着杜卡学的格斗。青出于蓝而胜于蓝这种事，可不多见。而且，苏重的学习时间，还那么短。加在一起也还不到半年而已。
<br/>　　这也是他不停来找虐的原因，因为苏重的格斗术确实强大。
<br/>　　苏重的身形并不太强壮，身上有些流线型肌肉。但并不是那种身高力壮的人。
<br/>　　对付这种类型对手，直接力量碾压最是轻松。布鲁斯却知道，如果谁这么对付苏重，下场绝对凄惨。他自己就受过惨痛教育。
<br/>　　自己引以为傲的力量，在苏重凌厉迅捷的攻击之下，根本就毫无作用。
<br/>　　这让他不得不重视技巧的作用。苏重之所以能那么轻易击败他，依靠的不就是超绝的技巧吗？这正是格斗术的魅力所在。
<br/>　　依靠固有身体力量，发挥出数倍威力。甚至以弱胜强。
<br/>　　正因为在苏重这里，他看到了进步的可能，布鲁斯才会不停来找打。他又不是有病，不然怎么会天天过来找揍。
<br/>　　可看苏重满脸上的不耐，布鲁斯也气笑了：“你每天一招就把我打到。前后总共花不到五分钟。五分钟你能干什么？难道能把你那渣枪法提高到百发百中！指导影武者训练，可是你的本职工作。”
<br/>　　苏重撇撇嘴，直接走人。
<br/>　　“真是？”布鲁斯哭笑不得：“喂，你不用嫌麻烦了，明天我要进行考核。咱们的训练可以停一天。你能多练会儿枪啦，预祝你枪法大进。哈哈……”
<br/>　　布鲁斯幸灾乐祸的扬声喊道。所有人都知道苏重的枪法，那不是一般的烂。虽然每天挨打，可只要想到，苏重拿枪械一点儿没办法，他立马就感觉好多了。
<br/>　　苏重脚步一顿，旋即走开。
<br/>　　明天考核？这是到了基地毁灭的时候了。
<br/>　　砰砰砰……
<br/>　　激烈的射击声响彻靶场。靶子拉近，纸质靶子上，只有一个圆形孔洞。周边亮着红色光芒，缕缕青烟冒起，已被高温点燃！
<br/>　　刚才十多发子弹打出去，竟全都打在一个地方！子弹不断通过一个位置，这才让高温点燃靶纸！
<br/>　　心念一动，技能面板出现。
<br/>　　枪械射击：圆满；……
<br/>　　“不容易啊，这么多天练习，终于把枪械射击给点满啦。”苏重长出一口气。
<br/>　　有这个圆满枪械射击技能，苏重立马就成了枪械达人。只要是不超出苏重认知枪械，一上手，他立刻就能圆满掌握。
<br/>　　任何枪械，在其射程之内，苏重都能做到指哪打哪！
<br/>　　“明天布鲁斯要进行考核，杜卡这是要检验洗脑成果？”苏重听了布鲁斯的话，就知道这一天终于到来。
<br/>　　杜卡费劲心思的训练布鲁斯，可不是单纯的爱才心切。他想要毁灭哥谭这座城市，如果有布鲁斯·韦恩，这个哥谭土豪做内应，行动将会轻而易举。
<br/>　　这么多天，他不断训练布鲁斯。并不时向布鲁斯灌输以暴制暴的理论，掀起布鲁斯心中愤怒，引导他去憎恨哥谭。并把影武者联盟的立意无限拔高，成为一个消灭罪恶的组织。
<br/>　　可惜，杜卡注定要失败。认清影武者真面目的布鲁斯，不会把联盟看成正义组织，只会把它当做一个残暴的私刑者。
<br/>　　“明天必会发生冲突，混乱一起，就是我离开的时候。”苏重在心里盘算：“枪械技能正好练到圆满，可以直接离开。不过还是要做些准备。”
<br/>　　原著中，布鲁斯为了逃离基地，引爆了储藏室内的黑火药。爆炸毁灭了基地，很多影武者被炸死。他这才趁着混乱，逃离基地。
<br/>　　苏重不知道，自己的到来会不会引起变化。他需要提前准备，不管明天结果如何，他都必须离开。
<br/>　　布鲁斯肯定不会屈服杜卡。没了布鲁斯，杜卡对苏重的忍耐将会到达极限，稳定风波后，第一时间就会找他麻烦。苏重可不想被动挨打。
<br/>　　同时，他必须做好被爆炸波及的准备。而且，不管明天到底如何，他都要借助这次机会，离开基地！
<br/>　　……
<br/>　　清早，整个基地变得静悄悄。这种景象似曾相识。当初他接受杜卡考验时，就是这种场景。
<br/>　　同时，苏重敏锐发现，墙边阴影处，竟隐藏着两个影武者。
<br/>　　苏重眼睛眯起，这还是杜卡第一次对他进行监控！
<br/>　　这代表，杜卡对他的容忍已到达极限！不管布鲁斯是不是被驯服，今天杜卡必定会来找他麻烦！
<br/>　　是时候做个了断！苏重一身黑色劲装，腰间挂着武士刀。像一个普通的影武者一样，漫步走到训练场。
<br/>　　找了一个隐蔽角落，斜靠着柱子，苏重看着布鲁斯·韦恩接受考验。
<br/>　　他没刻意隐藏，杜卡又眼观六路耳听八方，苏重一出现他就注意到。手里拿着一个陶碗，里面放着几块殷红木炭，青烟袅袅。
<br/>　　然后，杜卡拿出一个紫蓝色花朵，揉碎了扔进碗里。抬起头，他就看到了苏重玩味的眼神。
<br/>　　当初苏重的测试，杜卡可是直接抓了一大把！根本就是往死里搞！
<br/>　　杜卡神色不变，情绪丝毫没有波动。他已经决定今天就和苏重摊牌。
<br/>　　要么臣服，要么去死。丝毫不在乎苏重玩味眼神。
<br/>　　看着吸入致幻烟雾，身形踉跄不断摇头的布鲁斯。苏重身形一闪，消失在大厅角落。
<br/>　　“助教，你不能走。”两个黑影，一前一后挡住苏重。
<br/>　　“你觉得，你们能挡住我？”苏重面无表情。
<br/>　　“所以我们不是一个人。”不知何时，又有四个人从隐蔽地方闪出。
<br/>　　“两个随身跟踪个，四个潜伏在远处。对付布鲁斯可能行，可对付我？杜卡是不是太小瞧我了？”
<br/>　　六个人眼神毫无波动，他们都是严格训练的冷酷杀手。一旦动手，轻易不会动摇。
<br/>　　没有语言沟通，六个人同时出手。武士刀高高举起，凶狠砍向苏重。一上来，就下杀手！
<br/>　　显然，杜卡有过吩咐，只要他有逃离迹象，直接格杀！
<br/>　　苏重了然，早就知道不会那么容易离开。抽出腰间武士刀，矮身前冲，一刀劈出。
<br/>　　噗嗤！
<br/>　　血花飞溅，两个因武者捂着脖子倒地，已然被苏重一刀毙命。
<br/>　　一招得手，苏重更不留情。他不退反进，一头扎进剩下四人中间。单手握着武士刀，就好似拿着一根稻草，轻易舞动穿刺。
<br/>　　噗噗噗噗！
<br/>　　刀刀入肉，四个人根本就没反应过来，心脏咽喉等要害处便已中剑。
<br/>　　只是两个来回，苏重武士刀挥舞，轻易解决六个影武者！
<br/>　　布鲁斯知道苏重格斗厉害，却不知道苏重武道境界到底有多高。
<br/>　　强悍的灵魂，让苏重快速掌控身体。高超的境界，让他能攻击身周任何方位！一旦厮杀起来，对常人来说宛若杀神的影武者，根本就不是苏重一合之敌！
<br/>　　手腕震动，甩掉刀上血液。苏重推开基地大门，快速消失在冰天雪地之中。
