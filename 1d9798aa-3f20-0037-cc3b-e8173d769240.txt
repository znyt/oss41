　　叶尘前脚刚走，一道道强横的气息，陡然出现在客栈上方的千米高空之上，
<br/>　　“哈哈！看来诸位，都是被方才的天地异象，给吸引过来的喽？”
<br/>　　“天地变，异象生！再加上这四周丹香四溢，这里不久之前，必定有宝丹问世！”
<br/>　　“奇怪！怎么一追到这里，就突然断了线索？而且周围也没有察觉到气息特别强大的存在？”
<br/>　　“看来这位高人，不但丹术超群，隐匿功夫，也是一流啊！”
<br/>　　“可惜！没能亲眼目睹宝丹问世的场景，实在太可惜了！”
<br/>　　……
<br/>　　就在一众强者纷纷惋惜之时，叶尘此时已经来到一处金碧辉煌的古朴楼阁前。
<br/>　　门匾之上，是金光闪闪的三个大字“天藏阁”！
<br/>　　这里便是整个云丹城最大的拍卖场了。
<br/>　　叶尘举步迈入其中，一名打扮妖艳的青年女子，笑着迎了上来，
<br/>　　“先生是要买东西，还是卖东西？我们天藏阁，乃是云丹城第一大拍卖场，丹药、法宝、功法秘籍，以及各种天材地宝应有尽有！同时也收购各种宝贝……”
<br/>　　没等那妖艳女子说完，叶尘直接将之前炼制的三枚化婴丹取了出来，
<br/>　　“我来卖丹药！”
<br/>　　那妖艳女子顿时一愣，脸色的热情顿时消减了许多，甚至还隐隐露出一抹轻蔑之色。
<br/>　　要知道，在云丹城最不缺的就是丹药，因而相比周围的其他星域，丹药的价格，至少也低上好几倍都不止。
<br/>　　毕竟，云丹城里的炼丹师实在太多了，早已经供大于求，除非高级灵丹，又或者是万中无一的完美品阶灵丹，否则的话，在这拍卖场中拍卖，根本就无人问津。
<br/>　　而且，
<br/>　　她见叶尘年纪轻轻，也不像是丹道大师，又能拿出什么高级丹药出来？
<br/>　　“小先生，若是你想卖丹药的话，我天藏阁可是有规矩的，至少也得是三品以上的灵丹，而且还要由我们专业的鉴定师傅，来做出评判，能够进入拍卖资格，然后我们才能为您拍卖！”
<br/>　　叶尘自然也看出了对方眼中的轻视，不过他并没有放在心上。
<br/>　　四品化婴丹，在云丹星绝对算的上是高级灵丹，更何况是完美品阶的化婴丹呢？
<br/>　　“好，那就先鉴定吧！”
<br/>　　叶尘淡淡回应了一句。
<br/>　　那妖艳女子见叶尘竟然还不死心，顿时眉头闪过一抹不耐之色，不过即便对方是乞丐，来到这里也是客人，她自然不敢过分怠慢，只是明知道浪费时间，心里不免有些不爽。
<br/>　　很快，
<br/>　　在那名妖艳女子的引领下，叶尘来到旁边的一处偏殿之中。
<br/>　　足足等了十几分钟，妖艳女子再次出现在叶尘眼前，旁边还多了一名身穿青袍的灰发老者，一脸的傲然之色，
<br/>　　“就是你要拍卖丹药？”
<br/>　　青袍老者瞥了叶尘一眼，脸色的不耐之色更浓，显然同妖艳女子一样，觉得简直就是浪费时间。
<br/>　　叶尘点了点头，将手中两个盛装丹药的玉瓶递了过去。
<br/>　　三枚化婴丹，是分开来装的，两枚精品品阶的丹药放在了一个瓶子，那枚完美品阶的丹药，单独放在一个瓶子。
<br/>　　青袍老者不耐烦的从叶尘手中接过玉瓶，随手打开了那瓶盛装精品丹药的玉瓶。
<br/>　　呼~
<br/>　　玉瓶一打开，顿时一股浓郁的丹香，瞬间传遍整个大殿。
<br/>　　之前还一脸不耐烦的青袍老者，一下子脸色就变了，当即将那两枚丹药取出，一翻仔细观察之后，不由得两眼发亮，满脸激动之色，道：
<br/>　　“四品化婴丹！而且这品质，应该是精品品阶吧？”
<br/>　　此言一出，不但旁边的那名妖艳女子，同样脸色大变，大殿里的其他人，也纷纷将目光投了过来。
<br/>　　“精品品阶的化婴丹！即便是寻常的四品炼丹师，也很难炼制出来吧？”
<br/>　　“据说这精品品阶的化婴丹，可以让金丹境巅峰的修士，有六成的概率，晋升为元婴境呢！”
<br/>　　“不错！对于那些活了几百岁的金丹巅峰境修士来说，化婴丹可是比任何丹药都珍贵！记得一年前，天藏阁就出现过一颗精品品阶的化婴丹，最终足足拍到一百万灵石的高价！”
<br/>　　……
<br/>　　听到众人的议论声，就连叶尘也微微有些惊愕。
<br/>　　才只是精品品阶的化婴丹，居然就可以拍到百万灵石了？
<br/>　　看来我还是低估了这丹药的价值啊！
<br/>　　至于一旁的青袍老者，还有那名妖艳女子，早就已经激动不行，当即命令下人上茶，对叶尘的态度，也发生了一百八十度的大转弯。
<br/>　　毕竟，这可是百万级的大生意啊！
<br/>　　“叶先生对吧？不知道这两颗化婴丹，是出自那位大师之手呢？”
<br/>　　青袍老者和叶尘热情寒暄了几句之后，小心翼翼的开口问道。
<br/>　　周围众人，听到这个，也纷纷竖起了耳朵。
<br/>　　毕竟，放眼整个云丹城，可也炼制出精品品阶化婴丹，也没有多说，而每一位，都必定是大师级的人物！
<br/>　　叶尘不由得眉头微微一皱，他只想用丹药换取灵石，并不想再节外生枝，
<br/>　　“这个你就不必问了！只管告诉我，我的丹药，够资格在你们这儿拍卖吗？”
<br/>　　那青袍老者见叶尘有些不喜，也不敢再继续追问，连忙道：
<br/>　　“够够够！当然够资格！”
<br/>　　说到这里，青袍老者微微一顿，接着又道：
<br/>　　“叶先生，是这样的，按照我们天藏阁的规矩，您可以有两种选择！”
<br/>　　“第一，您可以将丹药直接卖给我们天藏阁，我们会按照市场价来购买！”
<br/>　　“第二，您也可以选择在我们的拍卖会上拍卖，不过一旦拍卖成功，我们天藏阁，要收取百分之二的佣金！”
<br/>　　“不知道您更倾向于哪一种方式呢？”
<br/>　　叶尘略微想了想，
<br/>　　“我选第二种！”
<br/>　　根据周围众人的反应，他手中的丹药，应该挺受欢迎的，若是按照市场价卖给天藏阁，极有可能就亏大了！
<br/>　　青袍老者似乎早就料到，叶尘会选择第二种，微微一笑，道：
<br/>　　“既然如此，我这就让人去安排！刚巧今天晚上，我们这儿要举办一场拍卖会，叶先生可以先移步到我们的贵宾包厢……”
<br/>　　还没等青袍老者说完，叶尘摆了摆手，
<br/>　　“不急！这里还有一枚丹药没有鉴定呢！”
<br/>　　叶尘指了指青袍老者手中，另外一只玉瓶，缓缓开口道。
<br/>(本章完)
