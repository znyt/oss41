<br/>　　“对啊老大，至少要送你出去！”瓜瓜等人也跟着说道。
<br/>　　“不用，一起走。”
<br/>　　“一起走就死啦！”小马拦腰抱起叶少阳，朝前方冲去。用一身修为硬扛着来自前方的攻击，“少阳，我欠你一条命，今天我还给你了。”

<br/>　　“活下来啊，以后还能再还！”
<br/>　　“别放屁了，我们根本杀不出去！”小马将一身修为释放出来，浑身上下好像包裹着一团火，直接对着人群撞过去，邪物横七竖八地倒了一大片，不过他心里清楚，这也是强弩之末，等修为耗尽，就会被敌人重重围住，到时候，就只能自暴元神了。
<br/>　　其余人的情况，也都好不了多少。
<br/>　　突然，前方掀起一股巨浪，一条巨大的水蛭挡住了去路。
<br/>　　“叶少阳，你去哪里啊！”血池罗刹欧诺个
<br/>　　小马暗暗叫苦，把心一横，就要自暴元神，突然脑袋上挨了叶少阳一下，回头看时，叶少阳脸上带着略装逼的微笑，抬起一只手，做了一个用力劈下去的手势。
<br/>　　“啥时候了，你干啥啊！”
<br/>　　小马话音刚落，突然一个庞然大物从头顶飞过，抬头一看是穷奇，心中一片惨然，本能地拽过叶少阳拉到自己身下，用自己肥胖的身躯挡住他。
<br/>　　“轰！”
<br/>　　穷奇砸在水面上，掀起了一大股巨浪。
<br/>　　额，没砸在自己这边？
<br/>　　小马本能地以为，一定是砸在哪个同伴身上，转头看去，穷奇居然越过他们，砸在前方人群里了，砸死了好些邪物。
<br/>　　失误了？
<br/>　　果然，穷奇猛然转身，双手朝他们挥舞起来。
<br/>　　想跑都来不及了。大伙赶紧游到一起，打算一起承受这可怕的一击，只有叶少阳依然笑着，昂首挺胸的。之后发生的一幕，让所有人都如堕梦中：
<br/>　　穷奇居然没有攻击他们，他挥舞双手，是帮他们扫除附近的敌人，两只手一挥，就掀起了一大片邪物，硬生生从千军万马中腾出了一片空间。
<br/>　　这，是真的？
<br/>　　所有人呆呆地望着穷奇，这个狰狞的邪物在为他们扫清了障碍之后，居然掉转过身，朝前方游动，双掌在两边滑动，清理出大片空间，然后……居然还回头看了他们一眼。
<br/>　　“愣着干什么，钻到他身下去，跟着往前走啊！”
<br/>　　叶少阳催促了好几遍，大家才回过神来，跟着他先后钻到穷奇身下四肢的缝隙里，紧张的是他们自己，穷奇根本没有对他们动手的打算，反而保护他们，只要鬼兵靠近，就用双手和翅膀将他们弹开。
<br/>　　“教主！”
<br/>　　雪魔看到这一幕，比叶少阳他们还要吃惊，回过神来，冲穷奇大叫起来。
<br/>　　穷奇眨着独眼，怔了那么一瞬间，突然张嘴，喷出了一大口瘴气，雪魔无奈只有退开，抓住同样目瞪口呆的血池罗刹，问她怎么回事。
<br/>　　“不知道，八成是被叶少阳施展了什么邪术！”血池罗刹一咬牙，“不管了，先攻击教主！”
<br/>　　雪魔怔怔地看她。
<br/>　　“绝不能让他们跑了，就算是牺牲教主，也没得办法了！”
<br/>　　于是一声令下，那些同样懵逼的邪物开始对穷奇发动了潮水般的袭击。
<br/>　　从远处看去，穷奇巨大的身躯与那些鬼兵形成了强烈对比，就像哥斯拉与人类。
<br/>　　叶少阳一干人在穷奇双翅的庇护下，倒是很安全。

<br/>　　叶少阳长出了一口气，颓然坐在了地上，之前一直苦苦支撑，现在上了岛，他一口气总算吐出去，彻底松懈下来，坐在地上就起不来了。
<br/>　　大伙都泄了气，瘫坐在地上，一个个几乎虚脱。
<br/>　　“叶少阳，今天……今天算你们走运，咱们来日方长，你的捉鬼联盟，我一定要杀得一个都不剩！”
<br/>　　血池罗刹站在水上，恨恨地望着叶少阳，眼神是那样的不甘和愤怒，但最后也只能命令大伙撤兵离去。
