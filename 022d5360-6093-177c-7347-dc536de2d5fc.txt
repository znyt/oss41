　　从肇事狙击手之前跟自己对话时的语气，罗昊就知道这是一个骄傲，甚至骄傲到有些自负的人，可是现在他看向自己的眼睛中哪里还有一丝骄傲？那种空洞，那种无神，就像是一条离开水太长时间，已经奄奄一息的鱼，罗昊丝毫都不怀疑这个时候如果他旁边的调查小组成员把手松开的话，他就会立刻软倒在地上，并且没有力气再爬起来。
<br/>　　“我以为那一枪是可以打中他的。”
<br/>　　肇事狙击手蠕动着嘴唇，最终有气无力的留下了这样一句话。
<br/>　　“可也正是你的‘以为’，让这个城市蒙上了悲伤。今夜，不知有多少家庭为了你那连狗屁都不如的自傲和自信，用鲜血来买单，而你就是谋杀了他们的杀人凶手！”
<br/>　　罗昊的话很不客气，可谓是字字诛心，但是现场没有人上前规劝罗昊，或许他们都认为这个狙击手有现在的下场完全是他自找的！罗昊那双如鹰一般的眼睛中透出一种叫做“锐利”的东西，狠狠地钉在肇事狙击手身上，把他的灵魂都要刺穿，冷冷的说道：“我见过很多跟你一样自傲的人，但是他们的自傲最终不仅杀死了他们自己，还有他们身边的人。如果你能免于安乐死的话，你以后的每一个晚上只要你闭上眼睛，就会看到满眼的鲜血，听到那些亡灵的哭喊和充满怨气的呐喊。如果我是你，我会在法庭上请求法官给自己死刑，这样你至少不必在无人的夜晚饱受折磨，最终变成一个……疯子！”
<br/>　　肇事狙击手被调查小组带走了，一起被调查小组带走的还有现场监视小组的组长，以及当时担任狙击手观察员的快速反应部队队员。调查所需的时间不会很长，但是也不会断，K国政府需要在这段时间做好安抚民众情绪的工作，并且还要找到一个合理的解释，而最合理的解释就是“恐怖袭击”。
<br/>　　自“911”事件后，这是最常听到的一个词，也是各国对发生的暴乱事件最优的解释选择。
<br/>　　快速反应部队的司令官盯着罗昊看了好一会儿，如果以军衔而论，他不知道要比罗昊高多少，可即便是如此，他站在罗昊面前却依然是一句话都说不出来，最后只能抬起手对着罗昊敬了一个礼，转身离开。
<br/>　　李毅峰抱着狙击步枪走到罗昊身边，看着远去的汽车，说道：“如果那个狙击手心志不够坚定的话，你刚才那番话，足以逼得他去自杀结束自己的生命。”
<br/>　　“蠢货，总是要为自己的愚蠢行为付出代价。”罗昊淡淡的说道：“而往往这个代价会很重，重到他必须用自己的命去支付。”
<br/>　　李毅峰清楚明白带着那样一种愧疚生活，是怎么样的一种感觉，用一个词形容，那就是“行尸走肉”！当年的他只是误杀了一个人质，而那个狙击手却是间接谋杀了几十上百人！
<br/>　　“谢谢！”
<br/>　　李毅峰突然对着罗昊道了声“谢谢”。
<br/>　　“怎么了，突然这么矫情？”罗昊扭头看着李毅峰说道，他自然是明白李毅峰这句感谢是因为，有些事情并不是一定要说穿的。
<br/>　　“没什么，只是突然矫情了一把，想要对你感谢而已。”李毅峰抱着狙击步枪，看着眼前破了一块玻璃墙的写字楼说道。
<br/>　　罗昊收起望远镜，说道：“走吧，今天晚上这里没搞头了。看到那块碎了的玻璃墙，老子心里就堵得慌！”
<br/>　　几个人钻进GLK，掉头离开。
<br/>　　返回指挥中心的途中，罗昊突然对坐在副驾驶上的李毅峰说道：“狐狸，其实你要真感谢我，你可以考虑把你的宝贝女儿给我儿子当媳妇。”
<br/>　　“可以啊。”李毅峰直接就回答道：“我想林芸也会同意的，姐弟恋什么的最有爱了。”
<br/>　　“喂喂喂，你们两个就这么把事情订下了？”
<br/>　　慕容飞坐在后排直嚷嚷，显然是对罗昊抢走了李毅峰女儿做自己儿媳妇感到不满，“小雨萱就这么许配了，是不是太草率了，我儿子怎么办？”
<br/>　　“你儿子？”李毅峰说道：“你家那个现在还是个胚胎呢，能看得出是男是女？这要是出来是个女的，你想让我女儿当百合啊？再说昊子家不是还有一个吗？小悠儿，继承了公主所有的优点多好。”
<br/>　　慕容飞顿时眼睛一亮，一拍手说道：“对啊，我怎么把小悠儿给忘了！”
<br/>　　“别……”罗昊连连摇头说道：“他的儿子一定跟他一样是个游戏宅男，我家悠儿以后的择偶要求可是很高的，他家儿子不行。”
<br/>　　“昊子，这兄弟没法当了啊！”
<br/>　　慕容飞急了。罗昊和李毅峰却是大笑起来。这一打岔，也是把刚才那种无比压抑的气氛给驱散了不少。
<br/>　　第二天，K国政府就对昨天晚上发生在地铁站的爆炸案做出了解释，是K国国防部长亲自召开的记者招待会，K国方面给出的解释跟罗昊预料的差不多。
<br/>　　K国国防部长在记者招待会上，一脸悲切的看着在场记者，挤出了几滴眼泪后，说道：“对于昨天晚上所发生的地铁站爆炸案，目前已经可以肯定是一起人为操纵的恐怖袭击事件，我向所有在爆炸案中死难同胞的家属道歉，是因为我们工作方面的疏忽导致了此次事件的发生，后续我们会展开相关的理赔工作，以尽快让那些失去至亲的家属从悲伤中走出来。”
<br/>　　“部长先生。”一名记者举手站起身来，发问道：“但是昨天晚上，有人在爆炸发生前听到了枪声，请问这又是怎么回事？”
<br/>　　不得不说，记者的嗅觉是灵敏至极的，昨天晚上的事情虽然被列入保密范围，但还是被抓到了蛛丝马迹。
<br/>　　而K国方面显然是对此有了应对方案，国防部长掏出手帕擦了擦眼睛，说道：“事情是这样的，前几天有一伙恐怖分子进入我国境内，我们立刻对他们进行了控制，昨天晚上我们本来是打算对他们采取行动，但是很可惜，我们的行动失败了，使得他们心生报复，导致爆炸案的发生。”
