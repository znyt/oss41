　　时光飞逝，转眼间一周的时间已经过去。
<br/>　　在这一周内，迦南学院已经恢复以往的秩序，除了消失的白山与古薰儿，包裹萧炎在内，总共四十六名学员进入内院。
<br/>　　而内院选拔赛上，比赛当天所发生的事情，犹如南柯一梦。
<br/>　　仅仅第二日清晨，周围的学员就将一幕幕忘却，仿佛根本就没有发生过一样，就连白山的名字也不再有学员提起。
<br/>　　历史是由胜利者修改的，虽然迦南学院做的只是平息慕清霄的怒火。
<br/>　　他们成功了，事后就由斗尊出手，抹除，并且篡改了学员们在内院选拔赛上的记忆，轻而易举。
<br/>　　抛开迦南学院的事情，慕清霄与小医仙则一直待在只属于两人的阁楼中，仿佛度蜜月一般。
<br/>　　至于迦南学院的高层，仿佛都无视了两人的存在，况且有慕清霄在四周布下的结界，他们也无法干扰。
<br/>　　获得心法后，彩鳞就一直在闭关中。
<br/>　　慕清霄与小医仙则每晚都夜夜笙歌，两耳不闻窗外事。
<br/>　　……
<br/>　　……
<br/>　　慕清霄站在阁楼的阳台上，负手而立，站在他身后的是一名白袍老者。
<br/>　　“当初，我们有过约定，我帮你恢复灵魂，你则保护仙儿五至六年，现在何去何从由你自己选择。”
<br/>　　天火尊者陷入沉思，都说大树底下好乘凉，面前就站着一尊斗圣，他似乎没理由离开。
<br/>　　而且，他也没有想去的地方，若是跟在慕清霄身后，未必不能更上一层楼。
<br/>　　捋了捋白花花的胡子，天火尊者微微一笑，道：“老夫此时也无处可去，只好厚着脸皮请小友收留。”
<br/>　　微微点头，慕清霄并不意外，正好他也缺一个打手，毕竟，不可能碰到点小打小闹他都要出手。
<br/>　　斗尊级的打手，除了远古八族，整个大陆估计也就只此一家了吧。
<br/>　　若是天火尊者知晓，在慕清霄心中，他只是一个打手，不知道会做何感想。
<br/>　　“以你的资质，本不可能突破斗圣，想要复活恐怕也不会随随便便，往后我会给你找一具斗圣骨骸。”
<br/>　　闻言，天火尊者双眼大亮，心中更是激动无比。
<br/>　　活了两辈子，他怎么可能听不出慕清霄的言下之意。
<br/>　　慕清霄说的不错，以他的资质，斗尊已经是巅峰，他想过复活，只是缺少各类灵药而已。
<br/>　　但是，复活后的他依旧只是六星斗尊，运气不错估计也能在提升一点，但是斗圣对他来说，遥不可及。
<br/>　　可是，如果有一具斗圣骨骸，那事情可就不一样了。
<br/>　　斗圣骨骸生前境界越高，他复活后，便能恢复骨骸生前的境界，甚至可能会更强。
<br/>　　并且，生前能成为斗圣，资质可见一斑，他若是以斗圣骨骸复活，在未来的路上便能走的更远。
<br/>　　“老夫虽是魂体，但一身实力也留下了七层，对付寻常斗尊是毫无问题的。”
<br/>　　慕清霄轻轻点头，道：“既然如此，前往中州的路上，若是遇到某些不长眼的，就劳烦前辈了。”
<br/>　　“自然。”
<br/>　　说罢，天火尊者就钻入雪白纳戒中，纳戒就飘入慕清霄的手心里。
<br/>　　决定了天火尊者的去留，迦南学院的事情也已经解决，再也没有留下的必要，慕清霄打算先回加玛帝国看一看。
<br/>　　午时，慕清霄躺在椅子上，望着院落中挥洒灵气的小医仙，眼中闪过精芒。
<br/>　　经过一周的修炼，小医仙已经完全掌控体内浩瀚的灵气，最重要的是，她转化的至寒灵气中，居然含有一丝毒性。
<br/>　　也就是说，她在将斗气转化成灵气的时候，同时也继承了厄难毒体的毒性。
<br/>　　拥有顶级心法，小医仙在战斗中本就占据优势。
<br/>　　现在，灵气中蕴含着毒性，对于敌人来说，简直防不胜防，若是加以利用，绝对是敌人的噩梦。
<br/>　　对此，慕清霄感到有些意外。
<br/>　　他的女人全都修炼的《玄阴素女经》，小医仙修炼后，毒气并未消散，反而融入了她的灵气中。
<br/>　　不知道等彩鳞将斗气全部转化，会不会产生某些变化。
<br/>　　对此，慕清霄并不担心，因为这种变化对她们有利无害。
<br/>　　此时，他并没有将各种剑诀招式交给小医仙，后者现在不过元婴阶段，并且斗气刚刚转化，心境也需要磨练。
<br/>　　虽然已经专修仙路，但斗气概念已经根深蒂固，需要一段时间改变过来。
<br/>　　“仙儿，我们该走了。”
<br/>　　“是去中州吗？”
<br/>　　小医仙蹦蹦跳跳的跑了过来，一把抱住慕清霄，小脸蛋在他胸膛蹭了蹭，犹如撒娇的小猫咪。
<br/>　　慕清霄抚摸着她的银丝，道：“暂时先回一趟加玛帝国，去找一个人。”
<br/>　　“是女人吧？”
<br/>　　慕清霄微微一愣，道：“什么时候变得如此聪明了？”
<br/>　　粉拳在他胸膛锤了几下，小医仙道：“你说找人的时候，眼神很温柔，与看我和彩鳞姐姐的时候是一样的。”
<br/>　　“在没有遇到你之前，在魔兽山脉中遇到的，当初我曾隐藏境界，她并不知道我的实力……”
<br/>　　说着，慕清霄微微一笑，道：“我们曾经有过约定，若是有一日我实力足够便去找她。”
<br/>　　闻言，小医仙皱了皱可爱的小鼻子，不敢置信道：“也就是说，你成功的把她给忽悠住了，然后莫名其妙的就答应你了。”
<br/>　　“似乎就是这样……”
<br/>　　慕清霄不置可否，成熟与高贵，云韵当时的气质与容貌确实将他吸引住了。
<br/>　　小医仙美眸瞪了眼慕清霄，忍不住揉了揉眉心。
<br/>　　她当然知道自己的丈夫很优秀，也曾经想过会有很多姐妹，只是没想到来的如此快，也不知道，还有没有别的女人。
<br/>　　“哼，等彩鳞姐姐出关，在找你算账。”
<br/>　　望着她吃醋的模样，慕清霄不禁有些想笑，忍不住捏了捏她的小鼻子。
<br/>　　简单的吃过午餐，将平日里生活用的东西收拾了一下，慕清霄与小医仙就转身离开迦南学院。
<br/>　　对于两人的离开，学院高层惊喜过望，能将这么一尊大神送走，他们真的比谁都高兴。
